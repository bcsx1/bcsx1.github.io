Obsidian 精髓——反向链接

使用两个中括号[[]]，可以链接到本地文件，也可以输入先【#】可以链接到标题，也可以先输入【^】链接文本块，也可以先输入【|】指定的文本

反向链接到文件示例：[Obsidian反向链接文件](Obsidian反向链接文件.md)

备注，支持以下文件的嵌入：

  • Markdown 文件：md;
  • 图像文件：png, jpg, jpeg, gif, bmp, svg; 其中外引图片不会直接显示图片；
  • 音频文件：mp3, webm, wav, m4a, ogg, 3gp, flac;
  • 视频文件：mp4, webm, ogv;
  • PDF 文件：pdf.

上标、下标

  1. 上标符号，符合^，如： x 4 x ^ 4 x4，显示为： x 4 x^4 x4
  2. 下标符号，符号_，如： C O 2 CO _ 2 CO2​，显示为： C O 2 CO_2 CO2​

基础运算

  1. 加法运算：符号：+，如： x + y = z x + y = z x+y=z，显示为 x + y = z x + y = z x+y=z
  2. 减法运算：符号：-，如： x − y = z x - y = z x−y=z，显示为： x − y = z x - y = z x−y=z
  3. 加减运算：符号：\pm，如： x ± y = z x \pm y = z x±y=z，显示为： x ± y = z x \pm y = z x±y=z
  4. 减加运算：符号：\mp，如： x ∓ y = z x \mp y = z x∓y=z，显示为： x ∓ y = z x \mp y = z x∓y=z
  5. 点乘运算：符号：\cdot，如： x ⋅ y = z x \cdot y = z x⋅y=z，显示为： x ⋅ y = z x \cdot y = z x⋅y=z
  6. 星乘运算：符号：\ast，如： x ∗ y = z x \ast y = z x∗y=z，显示为： x ∗ y = z x \ast y = z x∗y=z
  7. 除法运算：符号：/，如： x / y = z x / y = z x/y=z，显示为： x / y = z x / y = z x/y=z
  8. 分式表示1：符号：\frac{分子}{分母}，如： x + y x − y \frac{x+y}{x-y} x−yx+y​，显示为： x + y x − y \frac{x+y}{x-y} x−yx+y​
  9. 分式表示2：符号：{分子}\over{分母}，如： x + y x − y {x+y}\over{x-y} x−yx+y​，显示为： x + y x − y {x+y}\over{x-y} x−yx+y​
  10. 绝对值：符号：||，如 ∣ x ∣ |x| ∣x∣，显示为： ∣ x ∣ |x| ∣x∣
