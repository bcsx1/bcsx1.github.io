原标题：易备：驱动器镜像(热)备份 – 磁盘克隆、增量备份、备份验证

易备备份软件提供全方位的驱动器镜像备份功能。用户可以为整个磁盘创建镜像, 并在几分钟内恢复计算机或服务器, 不需要重新安装操作系统或程序。这种备份方式支持Vista之后的所有操作系统(Windows 7, Windows 8, Windows 10, Server 2008, Server 2012)。利用易备，用户甚至还可以进行异机恢复操作。

主要功能

*磁盘克隆

*驱动器镜像功能与Windows备份功能兼容

*主动防勒索病毒的保护机制

*易备磁盘镜像：硬盘克隆、P2V、瞬时恢复

*已打开文件的备份(VSS)

*根据日程计划，自动执行备份任务

*邮件实时通知备份结果

*创建恢复磁盘(易备恢复环境®)

*自动上载到FTP、S3以及云端

*单个文件的恢复

*异机恢复(HIR)

*裸机恢复

*可以保存几份磁盘镜像

*Windows镜像备份：增量备份(块级别备份)

*驱动器镜像可直接放在网络上, 甚至是在网络存储设备上

*快速有效的驱动器镜像

*完整备份以及个人计算机和服务器的裸机还原

*磁盘克隆和物理虚拟转换

*通过在Hyper-V中启动克隆系统，进行瞬时恢复

*易备恢复环境®：利用易备恢复磁盘启动机器

*标准格式的备份文件 (VHD/VHDX)

*完整的BitLocker支持(备份和恢复)

*高级备份验证

*无需其他驱动程序

*无需重启系统

*不干扰服务

*自动将磁盘镜像复制到磁带

*与RAID阵列兼容

*驱动器镜像与Windows服务器备份功能兼容，从而可以对应用程序或虚拟机(如：Exchange、Hyper-V、或活动目录)进行选择性的恢复

*Windows 10,7,8 - Server 2008, Server 2012, Server 2016, Server 2019

关键特性

磁盘克隆以及P2V

易备软件可以在不同机器中克隆硬盘驱动器和复制操作系统。事实上，利用易备驱动器镜像功能，用户可以创建精准复制磁盘包括分区逻辑结构的VHD及VHDX文件。这样可以对Hyper-V虚拟机系统进行即时恢复，以及将物理机转换为虚拟机。

恢复磁盘的创建

利用易备软件，用户可以创建恢复磁盘、USB驱动器或烧录到光盘中的ISO文件。通过使用易备恢复装置启动机器，用户即可获得易备恢复环境®，从而可以选择易备创建的驱动器镜像备份，并将其恢复到新的硬盘中。即便是在异机环境下，易备仍是一款完美简单的裸机还原应用程序。

低成本的完整备份功能，同样适用于服务器

易备软件具有很高的性价比。用户不仅可以制作系统镜像，还可以在任何大容量存储设备(USB驱动器、RDX等)或网络设备(服务器、NAS等)中制作基于文件的增量备份，同时还可以进行zip压缩、同步和加密。

磁盘镜像和裸机恢复

易备备份软件使用Windows标准格式制作驱动器镜像，生成VHD或VHDX格式的文件。用户可以打开这些文件，恢复单独的文件；将这些文件挂载为普通的硬盘驱动器；或在系统崩溃后恢复整个系统。恢复系统时，用户仅需使用Windows安装盘启动机器，然后从另外一个磁盘或网络设备中，选择需要恢复的磁盘镜像。

Windows镜像的增量备份(块级)

易备制作驱动器镜像(Windows镜像备份)速度极快，并且为增量备份模式。程序创建整个系统磁盘(或任何其他磁盘)的镜像，并以VHD或VHDX的文件格式，保存到本地或网络目标位置。第一个镜像制作完成之后，后续的备份速度极快，因为仅需将上次备份之后所做的更改复制到已有的镜像文件中。

Email通知、日程安排、详细报告

用户可以简便地对驱动器镜像备份任务进行日程安排，使易备软件在定义的日期和时间制作镜像。每次备份操作之后，易备都会发送电子邮件，用户可以对备份结果保持关注。

异机恢复

易备驱动器镜像备份功能的重要的特性之一是它可以将操作系统恢复到与制作备份的计算机不同的另外一台机器中。利用该特性，用户可以完整地恢复系统，即使该系统当时安装所在的旧机器已不复存在(独立于硬件的恢复功能)。

不干扰服务，无需重启机器

使用易备软件，用户可以制作服务器的“热备份”，无需重启机器，并且不会对任何服务造成干扰。同时还可以复制被锁定或使用中的文件、邮箱数据库、虚拟机、以及所有安装的程序。返回搜狐，查看更多
