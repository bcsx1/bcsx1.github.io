最满意答案

传递给jQuery.each回调的第一个参数是数组中值的索引; 第二个参数是实际值。

尝试使用：

$.each(data, function(i, entry) {

// your code here

});

The first argument passed to the jQuery.each callback is the index of the value in the array; the second argument is the actual value.

Try using:

$.each(data, function(i, entry) {

// your code here

});
