Docker拉取Nginx

1 搜索nginx镜像

docker search nginx

2 选择第一个热度最高，拉取nginx镜像

docker pull nginx

3 使用镜像启动一个nginx容器

docker run -di --name=mynginx -p 80:80 nginx


4 访问Nginx，浏览器输入[ip]:80

总结

  • 使用docker拉取镜像启动容器绝对比本地安装快捷方便
