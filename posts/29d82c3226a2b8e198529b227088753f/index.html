Matlab的示例代码:

以下是一个简单的MATLAB代码示例，用于计算蔬菜各品类及单品销售量的分布和关联关系：

% 示例数据

categoryA_sales = [20, 30, 40, 50, 60];

categoryB_sales = [10, 20, 30, 40, 50];

categoryC_sales = [5, 10, 15, 20, 25];

product1_sales = [12, 18, 25, 30, 40];

product2_sales = [8, 12, 16, 20, 24];

% 计算品类销售量

categoryA_total_sales = sum(categoryA_sales);

categoryA_avg_sales = mean(categoryA_sales);

categoryB_total_sales = sum(categoryB_sales);

categoryB_avg_sales = mean(categoryB_sales);

categoryC_total_sales = sum(categoryC_sales);

categoryC_avg_sales = mean(categoryC_sales);

% 计算单品销售量

product1_total_sales = sum(product1_sales);

product2_total_sales = sum(product2_sales);

% 计算单品相关系数

correlation_coefficient = corrcoef(product1_sales, product2_sales);

product_correlation = correlation_coefficient(1, 2);

% 输出结果

fprintf('品类A销售总量: %.2f\n', categoryA_total_sales);

fprintf('品类A平均销售量: %.2f\n', categoryA_avg_sales);

fprintf('品类B销售总量: %.2f\n', categoryB_total_sales);

fprintf('品类B平均销售量: %.2f\n', categoryB_avg_sales);

fprintf('品类C销售总量: %.2f\n', categoryC_total_sales);

fprintf('品类C平均销售量: %.2f\n', categoryC_avg_sales);

fprintf('单品1销售总量: %.2f\n', product1_total_sales);

fprintf('单品2销售总量: %.2f\n', product2_total_sales);

fprintf('单品1和单品2的相关系数: %.2f\n', product_correlation);
