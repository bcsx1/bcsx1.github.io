n与nodejs（MAC）

一、什么是n

n是 Mac 下的 node 管理工具,可以在同一台电脑上安装多个Node.js版本灵活切换。

二、nvm 与 n 的区别

n 命令是作为一个 node 的模块而存在，而 nvm 是一个独立于 node/npm 的外部 shell 脚本，因此 n 命令相比 nvm 更加局限。
由于 npm 安装的模块路径均为 /usr/local/lib/node_modules，当使用 n 切换不同的 node 版本时，实际上会共用全局的 node/npm 目录。 因此不能很好的满足『按不同 node 版本使用不同全局 node 模块』的需求。

三、安装

# 1. 清除node缓存:
sudo npm cache clean -f 
# 2. 安装 n 执行:
npm install  n -g 
# 3. 查看n是否安装成功：
n -V

注意：如果报错如:code EACCES errno -13,表示你没有权限安装，使用管理员身份安装：sudo npm i n -g

四、使用

使用n管理node

  1. 查看node可以使用的列表n ls
  2. 安装制定版本node
# 1. 查看官方node版本 
npm view node versions 
# 2. 安装指定的node版本,如14.17.3
n 14.17.3

注意：如果最终显示的还是原来node的版本号则需要在管理员权限下运行: sudo n node版本号
3. 切换node版本
首先执行n，通过上下键进行切换选择，最后使用enter键选中版本。
如果选中后还是原来版本，则使用 sudo n，后再通过上下键进行切换选择，最后使用enter键选中版本。
4. 删除node版本

n rm node 14.17.3
# 或
sudo n rm 14.17.3
  1. 卸载 n
npm uninstall n -g
# 或
sudo npm uninstall n -g
  1. node版本展示
n ls-remote # 列出最新的node版本 
n ls-remote 11 # 列出11X的版本
  1. 安装版本
sudo n latest # 安装最新正式发布版本 
sudo n lts # 安装最新的长期支持正式发布版本 
sudo n 14.14.0 # 安装指定版本
