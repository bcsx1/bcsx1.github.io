Anaconda 安装与创建环境

Anaconda安装与创建环境

一、下载Anaconda

进入官网，下载

https://www.anaconda.com/

二、执行安装程序

1、点击下载的exe文件

2、一步步配置





3、等待安装完成


4、安装结束

三、创建环境

1、先打开Anaconda Promt

2、输入命令创建环境

conda create -n name python==3.7.4

注意：“name”是你要创建的环境的名字，可以自定义；python版本可以指定也可以不指定

如

conda create -n Qi

3、激活环境

activate name

注意：“name”应该是你自定义的环境名称

如

activate Qi


4、在新创建的环境安装Spyder和Jupyter

打开 Anaconda Navigator


切换到新创建的环境

找到Spyder或Jupyter安装

四、Spyder

打开Spyder

输入代码，然后点击运行代码

print("Hello World!")

可以在控制台查看打印结果

五、Jupyter

5.1 修改默认工作路径

1、打开 Anaconda Prompt，进入新创建的环境，输入命令

jupyter notebook --generate-config

2、根据框框中的路径，打开框框中的文件。虽然是py文件，但也可以用记事本打开

3、找到

# c.NotebookApp.notebook_dir = ''


删除前面的 “#” 号，在单引号内添加自定义路径。如图我设置的路径为“D:\JupyterHome”。保存文件，退出

4、找到新创建环境下的Jupyter的快捷方式。

打开属性

删除选中的内容

 "%USERPROFILE%/"


5.2 Jupyter使用

点击Jupyter，会在浏览器打开一个界面。编程操作在此界面进行

1、创建代码文件

2、打印“Hello World!”

六、使用Anaconda Prompt安装包

1、命令

可以通过

conda install 包名

或

pip install 包名

安装包。安装时可以指定包的版本

2、演示

如图，在刚刚安装的环境中没有安装“matplotlib”包

打开Anaconda Prompt，进入新创建的环境，输入命令

conda install matplotlib

遇到，输入“y”，回车即可

安装完成后，再次执行代码。没有再报错，安装成功！
