实现效果

​​
想图省事的可以直接用我做好的镜像，往下看

Docker镜像地址

v1.8.0有个小bug，不建议用，推荐使用v1.8.0-1

$ docker pull itkevindoc/yapi:v1.8.0-1

通过Helm安装到Kubernetes

添加chart源

$ helm repo add ygqygq2 https://ygqygq2.github.io/charts/

将chart下载到本地，并解压，同时更新helm依赖

$ helm fetch ygqygq2/yapi --version 2.0.0 --untar
$ cd yapi && helm dependency update
$ cd charts && tar xf common-1.17.1.tgz && tar xf mongodb-10.31.5.tgz
$ rm -rf common-1.17.1.tgz mongodb-10.31.5.tgz

修改values.yaml，以下是我的配置

# 容器image及tag
image:
  #（1）改此处镜像地址
  repository: itkevindoc/yapi
  tag: v1.8.0-1
  pullPolicy: IfNotPresent # IfNotPresent: 有则不拉（减少流量和操作步骤），Always: 不管tag总拉（适合tag不变时更新）
  imagePullSecrets: []

#（2）修改此处的config.json
secret:
  enabled: true
  mountPath: /yapi/config.json
  subPath: config.json
  readOnly: true
  data: 
    config.json: |-
      {
        "port": "3000",
        # 关闭注册
        "closeRegister": true,
        # 版本通知
        "versionNotify": true,
        # 管理员账号（实测，配了也无法自动创建，需要手动添加到mongodb里的user表里）
        "adminAccount": "admin@admin.cn",
        "db": {
          # 配置mongodb的svc地址
          "servername": "yapi-mongodb.yapi",
          "DATABASE": "yapi",
          "port": 27017,
          "user": "yapi",
          "pass": "yapi",
          "authSource": ""
        },
        # 配置发送邮箱
        "mail": {
          "enable": false,
          "host": "smtp.exmail.qq.com",
          "port": 465,
          "from": "xxxx@xxxx.cn",
          "auth": {
            "user": "xxxx@xxxx.cn",
            "pass": "xxxxxxx"
          }
        }
      }
      
#（3）修改此处，打开mongodb，并配置好mongo的连接地址和用户名密码   
mongodb:
  # if enabled set "false", fill the connection informations in "external" section
  enabled: true
  external:
    mongodbHost: "这里填mongodb连接地址"
    mongodbDatabase: "yapi"
    mongodbPort: "27017"
    mongodbUsername: "yapi"
    mongodbPassword: "yapi"

  auth:
    enabled: true
    rootUser: root
    rootPassword: "root@123"
    usernames: ["yapi"]
    passwords: ["yapi"]
    databases: ["yapi"]
  persistence:
    enabled: true  # 注意 false 时不保留数据
    medium: ""
    existingClaim: ""
    storageClass: "csi-disk"
    accessModes:
      - ReadWriteOnce
    size: 10Gi 
    
#（4）修改此处，属于bug，缺少[]
env: []

安装

$ kubectl create ns yapi
$ helm install yapi . -n yapi -f ./values.yaml

更新

$ helm upgrade yapi . -n yapi -f ./values.yaml
