kotlin学习-魔术类小技巧

1.字符串内嵌表达式

不需要像Java那样去拼接字符串，而是可以直接将表达式写在字符串里面

fun main(){
    
    val name ="Diana"
    val age=12
    println("name=$name,age=$age")
    
   
}
输出：
name=Diana,age=12

2.函数的参数默认值

我们可以在定义函数的时候给任意参数设定一个默认值，这样当调用此函数时就不会强制要求调用方为此参数传值，在没有传值的情况下会自动使用参数的默认值

例如：

fun main(){
    printParams(123)
    
   
}
fun printParams(num:Int,str:String ="hello"){
    println("num id $num,str is $str")
}
输出：
num id 123,str is hello

当然这个情况比较理想化，默认值在最后，只需要传前面的值就行

所以换一下

会报错，因为

首先num类型不匹配，然后是str没值

所以我们可以这样写

fun main(){
    printParams(str="hello",num=123)
    
   
}
fun printParams(num:Int = 123,str:String ){
    println("num id $num,str is $str")
}

kotlin提供键值对的方式来传参，从而不必像传统写法那样按照参数定义的顺序来传参

还可以进一步省略num

fun main(){
    printParams(str="hello")
    
   
}
fun printParams(num:Int = 123,str:String ){
    println("num id $num,str is $str")
}
输出：
num id 123,str is hello
