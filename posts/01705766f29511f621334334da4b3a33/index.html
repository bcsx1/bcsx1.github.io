1.树

1.1树的概念

树是一种非线性的数据结构

  • 顶层的第一个数据被称为根节点
  • 除了根节点外，其余结构与树类似的集合是子树
  • 树是递归定义的
  • 子树与子树时间不能有交集
    ———————————————————————————————
    节点的度：一个节点含有的子树个数
    叶节点：度为0的节点
    分支节点：度不为0的节点
    父节点：若一个节点有子节点，则称这个节点为子节点的父节点
    子节点：一个节点包含其子树的跟节点称为该节点的子节点
    兄弟节点：有相同父节点的节点
    树的度：一整棵树中，最大的节点的度
    节点的层次：从跟开始，跟为第一层，依次向下
    树的深度/高度：树中节点的最大层次
    节点的祖先：从根到该节点所经分支上的所有节点
    子孙：以某节点为跟到该节点所经分支上的所有节点
    森林：多颗互不相交的树称为森林

1.3二叉树

  • 二叉树不存在度大于2的节点
  • 二叉树子树有左右之分，不能颠倒，二叉树是有序树
  • 根节点的层数看成1（方便计算）

特殊二叉树：
1.满二叉树
如果一个二叉树的层数为K，节点总数是2^K-1，他就是满二叉树
（ 20+21+…+2(k-1)=2k-1 )
2.完全二叉树
前k-1层是满的，最后一层可以不满，但必须从左到右连续

堆是特殊的完全二叉树；
堆分为大堆和小堆；
任何一个数组都可以看作完全二叉树，但不一定是堆；
大堆：树中的所有父亲都大于等于孩子
小堆：树中的所有父亲都小于等于孩子

2.堆的实现

2.1.向下调整

向下调整算法有一个前提：根的左右子树必须是堆，才能调整

向下调整：

void AdjustDown(HPDataType* a, int n, int parent)   //a[]数组、n=size数组数据个数、parent父亲
{
        int Bigchild = parent * 2 + 1;
        while (Bigchild < n)            
        {
               if (a[Bigchild] < a[Bigchild + 1] && Bigchild + 1 < n)   //找大孩子
               {
                       Bigchild++;
               }
               if (a[Bigchild] > a[parent])   //孩子比父亲大，就互换
               {
                       Swap(&a[Bigchild], &a[parent]);
                       parent = Bigchild;
                       Bigchild = parent * 2 + 1;
               }
               else
               {
                       return;
               }
        }
}

相比递归比较好（能不用递归就不用递归）

2.2堆的创建——向下调整实现

给出一个数组，可以看作完全二叉树，但是不是堆。
从倒数第二行（倒数第一个非子叶节点）开始向下调整，一直调整到根节点，即可将这个数组构建成堆。

2.3建堆的时间复杂度

向上调整建堆O(N*logN)
向下调整建堆O(N)

2.4堆的插入——向上调整实现

[以大堆为例]
C语言实现堆时，要完成插入数据HPPush，注意插入的值可能会影响它的祖先，（插入后依旧要符合堆的性质：大堆/小堆）如果插入的值比它的父亲小/相等，则无影响；如果插入的值比它的父亲大，则有影响。
大堆：
若插入的值比它的父亲大——向上调整（Adjustup）：
小于父亲时，直接插入扩容插入，时间复杂度为0(1)；
大于父亲时，扩容插入后向上调整，最多上调树的高度K次，时间复杂度为O(log(n))，n为节点个数；（满二叉树/完全二叉树最多：K = log（n+1）、完全二叉树最少：K = log（n）+1）
小堆：
若插入的值比它的父亲小——向上调整（Adjustup）；
向上调整：

void Adjustup(HPDataType* a,int child)
{
    int parent = (child - 1)/2;
    while(child > 0)
    {
    if（a[child] < a[parent])//这里是大堆，小堆换成大于号即可
        {
            Swap(&a[child],&a[parent]);
            child = parent;
            parent = (child -1)/2;
        }
        else
        {
            return;//break也可以
        }
    }
}

2.5堆的删除——向下调整实现

堆的删除是删除堆的根（堆顶的数据），首先根与数组的最后一个数据交换，然后删除最后一个数据，再对交换到堆顶的数据进行向下调整。

void AdjustDown(HPDataType* a, int n, int parent)//n为数据个数，第一个parent=0
{
        assert(a);
        int Bigchild = parent * 2 + 1;
        while (Bigchild < n)
        {
               //确认Bigchild指向大的那个孩子
               if (a[Bigchild] < a[Bigchild + 1] && Bigchild + 1 < n)
               {
                       Bigchild++;
               }
               if (a[Bigchild] > a[parent])
               {
                       Swap(&a[Bigchild], &a[parent]);
                       parent = Bigchild;
                       Bigchild = parent * 2 + 1;
               }
               else
               {
                       return;
               }
        }
}

删除：

void HeapPop(Heap* hp)
{
        assert(hp);
        assert(hp->size > 0);
        Swap(&hp->a[0], &hp->a[hp->size - 1]);
        hp->size--;
        AdjustDown(hp->a, hp->size, 0);
}

2.7二叉树的顺序结构

void HeapInit(Heap* php)
{
	assert(php);

	php->a = NULL;
	php->size = 0;
	php->capacity = 0;
}

void HeapDestory(Heap* hp)
{
	assert(hp);

	free(hp->a);
	hp->a = NULL;
	hp->size = hp->capacity = 0;
}

void HeapCreate(Heap* hp, HPDataType* a, int n)
{
	assert(hp);
	hp->a = (HPDataType*)malloc(sizeof(HPDataType) * n);
	if (hp->a == NULL)
	{
		perror("malloc fail");
		exit(-1);
	}
	memcpy(hp->a, a, sizeof(HPDataType) * n);
	hp->size = hp->capacity = n;

	//建堆算法————通过为后一个父节点开始依次向前排序，因为向下调整只能用于左右子树均为堆的情况
	for (int parent = (n - 1 - 1) / 2; parent >= 0; parent--)
	{
		AdjustDown(hp->a, n, parent);
	}
}

void HeapPush(Heap* hp, HPDataType x)
{
	assert(hp);

	if (hp->size == hp->capacity)
	{
		int Newcapacity = hp->capacity == 0 ? 4 : hp->capacity * 2;
		HPDataType* tmp = (HPDataType*)realloc(hp->a, sizeof(HPDataType) * Newcapacity);
		if (tmp == NULL)
		{
			perror("realloc fail");
			exit(-1);
		}
		hp->a = tmp;
		hp->capacity = Newcapacity;
	}
	hp->a[hp->size] = x;
	hp->size++;
	AdjustUp(hp->a, hp->size);
}

void HeapPop(Heap* hp)
{
	assert(hp);
	assert(hp->size > 0);

	Swap(&hp->a[0], &hp->a[hp->size - 1]);
	hp->size--;
	AdjustDown(hp->a, hp->size, 0);
}

HPDataType HeapTop(Heap* hp)
{
	assert(hp);
	assert(hp->size > 0);

	return hp->a[0];

}

int HeapSize(Heap* hp)
{
	assert(hp);

	return hp->size;
}

bool HeapEmpty(Heap* hp)
{
	assert(hp);

	return hp->size == 0;
}

void Swap(HPDataType* p1, HPDataType* p2)
{
	HPDataType tmp = *p1;
	*p1 = *p2;
	*p2 = tmp;
}

void AdjustUp(HPDataType* a, int child)//child = size
{
	assert(a);
	int parent = (child - 1) / 2;
	//大堆，大的在上面；小堆，小的在上面；这里是大堆
	while (child > 0)
	{
		if (a[parent] < a[child])
		{
			Swap(&a[child], &a[parent]);
			child = parent;
			parent = (child - 1) / 2;
		}
		else
		{
			return;
		}
	}
}

void AdjustDown(HPDataType* a, int n, int parent)//n为数据个数，第一个parent=0
{
	assert(a);
	int Bigchild = parent * 2 + 1;
	while (Bigchild < n)
	{
		//确认Bigchild指向大的那个孩子
		if (a[Bigchild] < a[Bigchild + 1] && Bigchild + 1 < n)
		{
			Bigchild++;
		}
		if (a[Bigchild] > a[parent])
		{
			Swap(&a[Bigchild], &a[parent]);
			parent = Bigchild;
			Bigchild = parent * 2 + 1;
		}
		else
		{
			return;
		}
	}
}
