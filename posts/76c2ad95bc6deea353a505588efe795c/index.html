 二.ipython

ipython介绍

IPython是一个交互式计算系统。主要包含三个组件：增加的交互式 “Python shell”，解耦的双过程通信模型，交互式并行计算的架构。支持变量自动补全，需要在pip安装

ipython安装步骤

win+R打开运行框输入cmd，进入命令行，输入命令，pip install ipython即可安装

ipython使用

如果需要使用，在命令行输入ipython就即可使用

 三.pycharm介绍

PyCharm是一种Python IDE（Integrated Development Environment，集成开发环境），带有一整套可以帮助用户在使用Python语言开发时提高其效率的工具，比如调试、语法高亮、项目管理、代码跳转、智能提示、自动完成、单元测试、版本控制。此外，该IDE提供了一些高级功能，以用于支持Django框架下的专业Web开发。

 四.pycharm安装

可以在Download PyCharm: Python IDE for Professional Developers by JetBrains官网下载

步骤1
进入官网以后选择右边的Download社区版本安装（不收费）点击即可安装。

 步骤2

下载完毕点击运行即可，安装没有过多的教程，一直点击下一步就可以，

 步骤3

等待进度条加载完毕

五.pycharm汉化教程

步骤1

打开PyCharm软件，默认显示英文界面。

 步骤2

点击“file”选项，然后点击“setting”。

 步骤3

打开设置菜单后，点击“plugins”选项。

 步骤4

输入“Chinese”找到中文安装包，点击“Install”。

 步骤5

安装完成后，点击“restart IDE”重启程序。

 步骤6

重启后，软件已经成功变成中文。

 六.为pycharm安装解释器

为什么要解释器创建一个文件他要让你去自己配置一个解释器，所以说我们要安装一个解释器。

步骤1

点击左上角文件找到设置点击

 步骤二

点击项目

步骤3

项目下有一个python解释器双击打开，右上角有一个添加解释器双击打开

 步骤4

找到系统解释器双击打开，点击右上角三个点

 步骤5

找到自己安装python的文件夹，选择Tools文件找到python文件点击即可，确定退出完毕
