31.求整数a与b的最小公倍数

# 法1：
a = int(input('请输入第一个整数：'))
b = int(input('请输入第二个整数：'))

32.求整数x和y的大公约数以及最小公倍数

y = int(input('请输入第二个整数：'))
l = x * y  # l是最小公倍数
while x % y != 0:  # 辗转相除法
    t = x % y
    x = y
    y = t
g = y  # g是最大公约数
l = l // g
print('最大公约数：', g, '最小公倍数', l)
# 请输入第一个整数：12
