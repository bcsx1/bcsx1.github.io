PyAutoGUI程序自动化控制

PyAutoGUI是一个款Python的GUI自动化工具，其目的是利用脚本控制鼠标和键盘操作。PyAutoGUI 适用于 Windows、macOS 和 Linux，并在 Python 2 和 3 上运行。

在已安装Python环境的基础上通过pip安装

PyAutoGUI库：

pip install -i https://pypi.douban.com/simple pyautogui

源码地址：https://github.com/asweigart/pyautogui

PyAutoGUI 有几个功能：

移动鼠标，单击或键入其他应用程序的窗口。

向应用程序发送击键（例如，填写表单）。

截屏，并给出一个图像（例如，按钮或复选框），在屏幕上找到它。

找到应用程序的窗口，并移动、调整大小、最大化、最小化或关闭它（当前仅使用 Windows）

在 GUI 自动化脚本运行时显示用户交互的消息框。

一、鼠标控制功能

鼠标坐标定义：

屏幕上的位置由 X 和 Y 笛卡尔坐标引用。X 坐标从左侧的 0 开始，然后向右侧增大，Y 坐标从顶部的 0 开始，然后向下增大。

左上角的像素位于坐标 0，0 处。如果屏幕的分辨率为 1920 x 1080，则右下角的像素将为 1919、1079（因为坐标从 0 开始，而不是从 1 开始）。

import pyautogui

screen_width, screen_height = pyautogui.size() # 获取屏幕分辨率

current_mouse = pyautogui.position() # 获取当前鼠标位置

mouse_on_screen = pyautogui.onScreen(1, 1) # 鼠标位置是否在当前屏幕上

鼠标绝对坐标位移

mouse_move_to = pyautogui.moveTo(100, 100) # 鼠标移动到指定坐标位置

mouse_move_to_dalay = pyautogui.moveTo(1000, 1000,duration=1) # 鼠标连续1s移动到指定坐标位置

pyautogui.moveTo(screen_width / 2, screen_height / 2) # 鼠标移动到屏幕中间位置

鼠标相对坐标位移

mouse_move = pyautogui.move(-10,100) # 鼠标相对当前鼠标位置移动

mouse_move_delay = pyautogui.move(1000,100,duration=2) # 鼠标连续2s相对当前鼠标位置移动

鼠标绝对坐标拖动

mouse_left_drag_to = pyautogui.dragTo(1000,1000,button=‘left’,duration=1) # 鼠标连续1s左键拖动

mouse_middle_drag_to = pyautogui.dragTo(1000,1000,button=‘middle’,duration=1) # 鼠标连续1s中间键拖动

mouse_right_drag_to = pyautogui.dragTo(1000,1000,button=‘right’,duration=1) # 鼠标连续1s右键拖动

鼠标相对坐标拖动

mouse_left_drag = pyautogui.drag(1000,1000,button=‘left’,duration=1) # 鼠标相对当前鼠标位置左键拖动

mouse_middle_drag = pyautogui.drag(1000,1000,button=‘middle’,duration=1) # 鼠标相对当前鼠标位置中间键拖动

mouse_right_drag = pyautogui.drag(1000,1000,button=‘right’,duration=1) # 鼠标相对当前鼠标位置右键键拖动

鼠标移动过程中的动作：moveTo() move() dragTo() drag()

pyautogui.moveTo(100, 100, 2, pyautogui.easeInQuad) # 开始慢，结束快

pyautogui.moveTo(100, 100, 2, pyautogui.easeOutQuad) # 开始快，结束慢

pyautogui.moveTo(100, 100, 2, pyautogui.easeInOutQuad) # 开始和结束快，中间慢

pyautogui.moveTo(100, 100, 2, pyautogui.easeInBounce) # 末端反弹

pyautogui.moveTo(100, 100, 2, pyautogui.easeInElastic) # 末端橡皮筋

鼠标点击

pyautogui.click() # 鼠标点击，点击之气需要配合moveTo()一起使用

pyautogui.click(10,10) # 鼠标移动到指定坐标，并且点击

pyautogui.click(button=‘right’) # 鼠标不同按钮单击，‘left’:指鼠标左键单击,‘middle’:指鼠标中键单击,‘right’:指鼠标右键单击

pyautogui.click(clicks=2) # 鼠标单击2次

pyautogui.click(clicks=2,interval=1) # 鼠标单击2次，并且单击之间间隔1s

pyautogui.click(clicks=2,duration=1) # 连续1s，鼠标单击2次

pyautogui.click(clicks=2,interval=0.2,button=‘middle’) # 鼠标中键单击2次，并且单击之间间隔0.2s

鼠标双击

pyautogui.doubleClick() # 鼠标左键双击

pyautogui.doubleClick(1000,1000) # 鼠标在坐标（1000，1000）位置，左键双击

pyautogui.doubleClick(button=‘right’) # 鼠标右键双击，‘left’:指鼠标左键单击,‘middle’:指鼠标中键单击,‘right’:指鼠标右键单击

pyautogui.doubleClick(interval=1) # 鼠标双击，并且双击之间间隔1s

pyautogui.doubleClick(duration=1) # 连续1s，鼠标双击

pyautogui.doubleClick(interval=0.2,button=‘middle’) # 鼠标中键双击，并且双击之间间隔0.2s

鼠标滚动

pyautogui.scroll(100) # 鼠标向上滚动100次

pyautogui.scroll(-100) # 鼠标向下滚动100次

pyautogui.scroll(-100,10,10) # 鼠标在（10，10）位置，向下滚动100次

鼠标按下松开

pyautogui.mouseDown(100,100) # 鼠标移动到（100，100）位置按下

pyautogui.mouseUp(100,100) # 鼠标移动到（100，100）位置松开，配合mouseDown一起使用

pyautogui.write(‘test’,interval=1) # 每个字符串延迟1s输入（不支持中文输入）

pyautogui.press(‘f1’) # 按F1键

模拟键盘按下之后释放

pyautogui.keyDown(‘left’)

pyautogui.keyUp(‘left’)

控制按压次数

pyautogui.press([‘left’,‘right’,‘left’])

pyautogui.press(‘left’,presses=3)

组合快捷键

pyautogui.hotkey(‘alt’,‘f4’)

pyautogui.hotkey(‘ctrl’,‘c’)

三、消息框功能

import pyautogui

pyautogui.alert(text=‘test’,title=‘标题’,button=‘yes’) # 显示包含文本和单个按钮的简单消息框。返回单击的按钮的文本。

pyautogui.confirm(text=‘test’,title=‘标题’,buttons=[‘yes’,‘no’,‘cancle’]) # 显示一个包含多个按钮的消息框。可以自定义按钮的数量和文本。返回单击的按钮的文本。

pyautogui.prompt(text=‘test’,title=‘title’,default=‘’) # 显示包含文本输入的消息框，以及"确定"和"取消"按钮。返回输入的文本，如果单击"取消"，则返回"无"。

pyautogui.password(text=‘test’,title=‘password’,mask='') # 显示包含文本输入的消息框，以及"确定"和"取消"按钮。键入的字符显示为返回输入的文本，如果单击"取消"，则返回"无"。

四、屏幕截图功能

import pyautogui

截图

pyautogui.screenshot(‘test.png’) # 截取整个屏幕图片并保存

pyautogui.screenshot(‘test.png’,region=(640,225,300,500)) # 截取从坐标（640，223）开始，宽度为300，高度为500的图片并保存

图像匹配

coods = pyautogui.locateOnScreen(‘test.png’) # 在屏幕中查找图片，返回图片的（左，上，宽，高）位置

pyautogui.center(coods) # 获取定位图片的中心坐标

pyautogui.locateCenterOnScreen(‘test.png’) # 在屏幕中查找图片，返回图片的中心点坐标（x,y）

pyautogui.locateCenterOnScreen(‘test.png’) # 在屏幕中查找图片，返回图片的中心点坐标（x,y）

pyautogui.locateCenterOnScreen(‘test.png’,region=(100,50,30,60)) # 缩小范围，加快查找速度。在屏幕中(100,50)坐标开始，宽度为30，高度为60的图片范围内查找指定图片，返回图片坐标（x,y）

pyautogui.locateOnScreen(‘test.png’, grayscale=True) # 灰度匹配：这会使图像和屏幕截图中的颜色去饱和，从而加快定位速度，但可能导致误报匹配

import pyautogui

新的改变

我们对Markdown编辑器进行了一些功能拓展与语法支持，除了标准的Markdown编辑器功能，我们增加了如下几点新功能，帮助你用它写博客：

  1. 全新的界面设计 ，将会带来全新的写作体验；
  2. 在创作中心设置你喜爱的代码高亮样式，Markdown 将代码片显示选择的高亮样式 进行展示；
  3. 增加了 图片拖拽 功能，你可以将本地的图片直接拖拽到编辑区域直接展示；
  4. 全新的 KaTeX数学公式 语法；
  5. 增加了支持甘特图的mermaid语法1 功能；
  6. 增加了 多屏幕编辑 Markdown文章功能；
  7. 增加了 焦点写作模式、预览模式、简洁写作模式、左右区域同步滚轮设置 等功能，功能按钮位于编辑区域与预览区域中间；
  8. 增加了 检查列表 功能。
