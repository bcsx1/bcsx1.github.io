为什么要写博客？写博客的好处？

做任何事都要怀揣初心，矢志不渝。

一个在线共享的笔记本

我不知道计算机专业的大家有没有这种感觉，尤其是在大一大二，感觉学校教的内容都是杂而不精，很多软件语言，刚刚配置好软件环境，浅尝辄止，踩了无数坑，一学期结课后再也没有碰过。但是后来参加科研竞赛项目，又要用到原来学过的php了，但是突然发现自己连apache是什么都不知道了·······

所以写一个博客吧，未来的你会感谢现在的自己，记录自己安装配置软件的过程和遇到的坑，防止遗忘。

所以写一个博客吧，未来的你会感谢现在的自己，把知识点记录下来用于期末复习，避免下次一定，修改错误。

一段三省吾身的学习历程

记得一位程序员说过(当然是开玩笑的)：
如果我们没能力修好它，我们就会告诉你它根本没坏。

程序优化第一法则：不要优化。程序优化第二法则：还是不要优化。

写博客是督促自己整理代码反省自身的过程，你在博客中复现代码的过程就是反省自身的一个过程。再次审视一边自己的代码（因为马上发出来，总不能浑身都是破绽，贻笑大方吧？）

一种严密有序的思维方式

记得一位程序员说过（狗头保证，不是无中生猿）
Talk is cheap，show me the code (废话少说，放码过来)

我最讨厌有人写代码不写注释；我最讨厌写代码的时候别人让我写注释（在线双标）

现在很多程序员的通病是只是自身有过硬的技术，但是和人沟通交流，团队协作的能力亟待提高。

写博客肯定不能是直接复制黏贴代码草草了事，因为这样很快你就会发现，你自己回头过来看自己的博客的时候也很难受，建议写博客要对自己的项目有一个大体结构的思路框架，列好目录分级标题的结构，养成严密有序的思维方式。

很多优秀的学长都有自己的博客，在考研求职面试中也是比较有分量的，能够把自己明白的东西用博客清晰的讲出来，必然是真正地掌握了知识，而不是浅尝辄止，而且博客写的好，与人打交道的能力肯定差不了，无论是导师还是上司，肯定对你加上不少好感度。

一种因吹思厅的作业提交途径

  • 如图，我上个学期的期末大作业，有部分的内容我整理之后放在博客上，在和老师沟通后，我的大作业里引用了我的博客。（没错，不许人间见白头，我抄我自己）
  • 在博客在线观看的体验必然是比word上好的。（前提是不能博客写的乱，要有目录结构）

我与CSDN的故事

与CSDN相知----相熟----成为一名博主的过程。

与CSDN的初次相遇

在我13岁的时候加入了CSDN，那个时候CSDN还没有今天，我还只是个小学生（确实是计算机要从娃娃抓起，显示码龄9年），记得当时是为了询问一个博主关于VB的技术问题，注册了CSDN帐号，博主热心地解答了我的问题。

每日上CSDN逛逛

偶尔在百度查不到的技术内容，CSDN上有，因为学业，直到上了大学，我才重拾CSDN，在这个平台学习一些自己感兴趣的内容。别人水群逛贴吧，我每天上CSDN签到看热榜。

我可以这么说，计算机专业学生学习实践的过程中，肯定会遇到bug，报错，百度的结果很多都会指向CSDN的博主博客，如果计算机专业的大学生不知道CSDN，肯定是没有好好学习（如果是天才型选手，从不遇到问题另说。）

因为高考失利，没有考上好大学，所处学校以文科为主，计算机系实力不太行，很多东西老师让我们自己研究学习，如果没有CSDN的博主们为我答疑解惑，对于我这种尴尬处境的自学学生无疑是困难重重。

成为一名博主

  • 有个比较喜欢的专业课老师在上课的时候给我们展示过师兄学长的CSDN博客，很好的解答了学习中遇到的坑和解决方法。老师建议我们写博客整理学习历程。
  • 之前在CSDN白嫖各位大牛博主分享的学习心得，在接触过GitHub等开源项目之后，站在一个程序员的角度，我更加深刻的认识到，计算机知识必须要有源头活水，把知识开源分享，才能一直保持清如许，有时候我个人的思路可能只是一个特例，还有很多内容我没有意识到，在和大家交流分享中才能找出自己的错误，在大家的建议下更新新的思路方法，保持进步。

为什么我选择了CSDN？

  • 刚刚开始写博客的时候，我也尝试过知乎等网站，但是我后来发现在计算机专业领域内容新手写博客比较多的是博客园和CSDN，其他网站（掘金等暂时不提）都太杂了。
  • 博客园在申请后可以得到比较大的个人js权限，页面的模板框架要自己设计（也可以用大佬写好的，好看确实是好看），但这样看不同人的博客就有不同的博客页面风格，作为程序员，我比较喜欢统一化的观看体验，真正找学习笔记的时候没心情管这些花里胡哨的个性化设置。CSDN出于网站风格统一和安全的考虑，没给新手博主这种权限，两者算是各有千秋吧。
    使用CSDN插件助手后的效果，风格简约统一，每一篇文章大体都是这样
  • 我一开始同时在CSDN和博客园上发布文章，博客园的访问量人数远远少于CSDN。
  • CSDN的在线网页编辑器让人舒服很多，博客园的感觉很简陋，看着很不舒服，我都是在自己本地上写好Markdown在导入到博客园，而CSDN我一般直接在网页上写，相比较而言CSDN便于我随时随地写博客。
    博客园对于我来说感觉太过于简约了，有种梦回上世纪论坛的感觉

    CSDN的编辑器的投票、模板等功能都很不错，干净清爽
