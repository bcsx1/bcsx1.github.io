基于Alitum21

天府大懒猫

单击软件右上角的设置按钮打开设置菜单。

General-通用菜单

提供与编辑基于原理图的文档相关的许多通用控件。

Units-单位

选择单位为mil或mm。

Options-选项

  • Break Wires At Autojunctions - 勾选此项，走线节点处会自动断线，如T型连接、十字连接、两线相交。

  • Optimize Wires & Buses - 勾选此项，自动优化走线，会自动删除多余的走线。
  • Components Cut Wires - 勾选此项，当放置新器件到走线上时，会自动在电气结点处切割走线，必须勾选“Optimize Wires & Buses”才会生效。
  • Enable In-Place Editing - 勾选此项后，文本可以在选中后通过再次单击文本或按快捷键“F2”直接修改，否则你必须打开属性框才能进行修改。
  • Convert Cross-Junctions - 勾选此项时，当进行一个十字连接时将自动转换两个三端连接。

  • Display Cross-Overs - 勾选此项时，交叉线位置变为弧形。

  • Pin Direction -勾选此项，将会在原理图中显示元器件管脚信号的方向，需要本身原理图符号的管脚配置了信号方向才可以。

  • Sheet Entry Direction - 勾选选项，图纸入口将根据信号方向进行方向显示。

  • Port Direction - 勾选此项，则端口(Port)样式由连接的I/O类型属性决定，如I/O类型为输入(Input)则端口也为输入。

  • Unconnected Left To Right - 勾选此项，则未连接的端口样式为从左到右的样式。

  • Drag Orthogonal - 勾选此项，拖动元器件时，元器件的连接线保持正交，否则任意角度。
  • Drag Step - 下拉菜单选择拖动的步进。

Include with Clipboard-剪贴板内容

  • No ERC Markers -勾选此项可在剪贴板中包含ERC标记。
  • Parameter Sets - 勾选此项可在剪贴板中包含参数集。
  • Notes -勾选此项可在剪贴板中包含注释。

Alpha Numeric Suffix

原理图符号包含多个部分时，设置命名后缀，可选方式为：字母后缀、以点分割的数字，以冒号分割的数字。

Pin Margin-管脚余量

  • Name - 设置管脚名称与元器件轮廓之间的间距，最小值1mil。
  • Number -设置管脚编号与元器件轮廓之间的间距，最小值1mil。

Auto-Increment During Placement-自增模式

  • Primary - 设置自增模式的初始值，当字符串为数字或者最后一位为数字时，自增模式才会生效。
  • Secondary - 设置自增模式的增量值，如放置I/O管脚，则管脚编号将会按照增量进行自增或自减，如初始值设置为1，增量设置为1，则按1，2，3依次递增。
  • Remove Leading Zeroes - 勾选此项将移除前导零，如输入0045将变为45。

Port Cross References-端口交叉参考（Report菜单下打开）

  • Sheet Style - 选择端口交叉参考时引用的图纸属性，根据自己的需要选择最适合的方式
  • None - 选此选择则不添加图纸属性。
  • Name - 选择此选项则会在交叉参考中增加图纸名称。
  • Number - 选择此选项则会在交叉参考中增加图纸编号。

  • Location Style - 选择端口交叉参考位置信息的显示方式
  • None -不设置位置信息的显示方式
  • Zone - 位置信息以区域显示
  • Location X,Y - 位置信息以坐标显示

Default Blank Sheet Size-空白图纸尺寸

  • Sheet Size -选择创建新图纸的图纸默认尺寸
  • Drawing Area - 图纸编辑区域，不可修改

Graphical Editing-图形编辑菜单

Options-选项

  • Clipboard Reference – 勾选此项，在工作区内进行复制或粘贴时则需要选择参考点，如果使能了电气热点( Object's Electrical Hot Spot)，则以最近的电气热点为准 。
  • Add Template to Clipboard – 勾选此项，进行原理图的剪切和复制将会将当前图纸的模板也复制到剪贴板。
  • Display Names of Special Strings that have No Value Defined - 勾选此项，当字符串没有定义值时就显示字符串默认名称。
  • Center of Object – 勾选此项，对象进行移动时将以参考点为准进行移动，无参考点的则按对象中心点移动。
  • Object's Electrical Hot Spot – 勾选此项，移动对象将按对象最近的电气热点为参考进行移动。
  • Auto Zoom –勾选此项，当选择元器件，原理图将自动缩放。
  • Single '\' Negation – 勾选此项，需要输入电平标识符的管脚名、端口、网络名、图纸接口时，只需要在第一个字符前添加单反斜杆“\”就可以实现，否则需要在每个字符后添加单反斜杠“\”。

  • Confirm Selection Memory Clear – 勾选此项，清除选择存储器的内容需要确认，选择存储器可用于存储一组对象的选择状态，按Ctrl+Q可以打开选择存储器。
  • Mark Manual Parameters – 勾选此项，带点显示的参数表示自动定位已关闭，并且参数随其父对象（例如组件）移动或旋转。要隐藏点，请禁用此选项
  • Always Drag – 勾选此项，拖动元器件时，电气连接将保持连接，按空格（Space）旋转器件，按（Ctrl+Space)可以调整连接线角度。
  • Shift Click To Select – 勾选此项，在原理图工作区将通过Shift+Click选择对象。
  • Click Clears Selection – 勾选此项，可以在原理图其他任意位置单击取消选中的对象，否则必须在选中的对象单击才能取消。
  • Place Sheet Entries automatically – 勾选此项放置图纸接口时将自动匹配图纸内的有效网络名称，否则将会放置系统产生网络名。
  • Protect Locked Objects – 勾选此项如果移除的对象中有锁定的对象将不会被移除，如果不勾选，在移除锁定对象时将弹出警示框。
  • Display Strings As Rotated – 勾选此项将按字符串放置的角度自动调整显示方式，方便阅读，否则将始终按照默认的阅读方式显示。
  • Reset Parts Designators On Paste – 勾选此项，当复杂粘贴一个器件时，将会重置该器件的标识符。
  • Sheet Entries and Ports use Harness Color - 勾选此项图纸接口和端口将自动适配信号线束的颜色，否则图纸接口和端口按照各自的默认颜色显示。
  • Net Color Override -勾选此项使能网络颜色覆盖，如果禁能设置网络颜色时将会弹出确认对话框，对话框如下：

  • Double Click Runs Interactive Properties - 勾选此项双击放置的对象时将打开属性面板，否则打开模态对话框。
  • Show Pin Designators -勾选将显示引脚标识符。

Auto Pan Options-自动平移选项

  • Enable Auto Pan - 勾选此项使能自动平移
  • Style – 当十字准线动作光标处于活动状态并且您将光标移动到视图区域的边缘时，自动平移生效。 如果自动平移打开，工作表将自动向该方向平移。 设置此字段以在自动平移期间控制光标移动。 选项有 Auto Pan Fixed Jump（按固定步长平移图纸，该步骤在步长字段中设置 - 光标保持在视图区域的边缘）和 Auto Pan ReCenter（将图纸平移 固定步长，在步长字段中设置 - 平移后光标在视图区域中重新居中）。
  • Speed – 拖动此滑条以设置自动平移速度。越往左走，自动平移运动的速度越慢或越精细。
  • Step Size – 输入一个值以设置每个自动平移步骤的大小。 步长决定了启用自动平移时文档平移的速度。 值越小，自动平移运动越慢或越精细。
  • Shift Step Size – 设置按下Shift按键自动平移的步进。

Color Options-颜色选项

  • Selections – 设置对象被选中时显示的颜色

Special Strings with No Value - 设置未定义值得字符串显示颜色。

Cursor-光标

  • Cursor Type – 设置编辑光标的显示样式，有四种样式可选：大型十字光标，小型十字光标，大型X光标，小型X光标

Compiler-编译菜单

设置原理图编译相关配置选项和控件

Errors & Warnings-错误与警告

Errors & Warnings - 配置指示原理图内对象出现错误或告警显示的波兰线的颜色

Auto-Junctions- 自动配置节点

  • Display On Wires - 勾选此项将会在导线上显示节点.

  • Size -选择节点的尺寸.
  • Color - 选择节点的颜色.
  • Drag Color -设置导线拖动时显示的颜色
  • Display When Dragging - 勾选此项拖动时显示导线
  • Display On Buses - 勾选次选项在总线上显示节点.

  • Size - 设置在总线上节点尺寸。
  • Color - 设置在总线上节点颜色。
  • Drag Color - 设置总线移动时显示的颜色。

Compiled Names Expansion-编译名称扩展

Display the expanded compiled names of the following objects - 选择需要显示编译名称扩展的对象。

  • Designators - 编译设计项目时，所有逻辑表都将扩展为物理表，因此，还将扩展某些网络以反映扩展后的物理表。勾选此项允许将逻辑工作表扩展为物理工作表时，物理工作表上的组件标识符获取扩展的网络信息，下拉菜单控制在编译项目后如何显示扩展的已编译指示符名称。
  • Display superscript if necessary - 勾选此项当逻辑指示符名称和已编译的指示符名称不同时，将显示上标。
  • Always display superscript - 指示符始终显示上标。
  • Never display superscript - 指示符从不显示上标。
  • Net Labels - 编译设计项目时，所有逻辑表都将扩展为物理表，因此，还将扩展一些网络以反映扩展后的物理表。启用此选项可允许将逻辑表扩展为物理表时，物理表上的网络标签获取扩展的网络信息。下拉菜单控制在项目编译后如何显示网络标签的扩展编译名称。
  • Never display superscript - 从不显示网络标签的上标。
  • Always display superscript - 始终显示网络标签的上标。
  • Display superscript if necessary - 当逻辑网络标签名称和编译的网络标签名称不同时，将显示上标。
  • Ports - 勾选此项可允许将逻辑表扩展为物理表时，物理表上的端口获取扩展的网络信息。
  • Sheet Number - 编译设计项目时，所有逻辑图纸都会扩展为物理图纸，因此，还会扩展一些网络以反映扩展后的物理图纸。勾选此项可允许将逻辑表扩展为物理表时，物理表上的表号参数获取扩展的网络信息。下拉菜单控制在项目编译后如何显示图纸编号参数的扩展编译名称。
  • Never display superscript - 从不显示图纸编号的上标。
  • Always display superscript - 显示图纸编号的上标文本。
  • Display superscript if necessary - 当逻辑表号和已编译表号不同时，将显示上标。
  • Document Number - 编译设计项目时，所有逻辑表都将扩展为物理表，因此，还将扩展一些网络以反映扩展后的物理表。启用此选项可允许将逻辑表扩展为物理表时，物理表上的文档编号参数获取扩展信息。下拉菜单控制在项目编译后如何显示扩展的文档编号参数的已编译名称。
  • Never display superscript -从不显示文档编号的上标。
  • Always display superscript - 始终显示文档编号的上标。
  • Display superscript if necessary - 当逻辑文档号和已编译文档号不同时，将显示上标。

AutoFocus-自动对焦菜单

Dim Unconnected Objects-变暗未连接对象

  • On Place - 勾选此项在放置一个对象时，会将该对象连接网络上的所有对象变暗
  • On Edit Graphically - 勾选此项在重新调整连接对象的大小时，使原理图图纸上所有未连接的对象变暗。.
  • On Move -勾选此项当移动连接到原理图图纸上已连接对象网络的对象时，可以使所有未连接的对象变暗。
  • On Edit In Place - 勾选此项当编辑连接的对象时，会调暗原理图所有未连接的对象E
  • All On - 打开上边的所有选项
  • All Off - 关闭上边的所有选项

Thicken Connected Objects-加粗已连接对象

配置与（Dim Unconnected Objects）相同

Zoom Connected Objects-放大连接对象

配置与（Dim Unconnected Objects）相同

Library AutoZoom-库元件缩放菜单

原理图库切换器件后自动缩放

Grids-栅格菜单

设置栅格的样式，颜色，以及大小

Break Wire-断线菜单

Cutting Length-剪切长度

从下列选项中选中一项决定剪切长度

  • Snap to Segment - 剪切整个导线
  • Snap Grid Size Multiple - 按栅格尺寸的倍数进行剪切
  • Fixed Length - 剪切一个固定长度

Show Cutter Box

在断线模式下，选择以下选项之一来控制切割器框（虚线矩形框）的显示：

  • Never - 不显示切割器框.
  • Always - 始终显示切割器框
  • On Wire - 当光标经过导线时才显示切割器框

Defaults-默认设置菜单

设置原理图下所有对象的默认参数
