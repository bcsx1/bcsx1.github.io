PTA 链表去重

给定一个带整数键值的链表 L，你需要把其中绝对值重复的键值结点删掉。即对每个键值 K，只有第一个绝对值等于 K 的结点被保留。同时，所有被删除的结点须被保存在另一个链表上。例如给定 L 为 21→-15→-15→-7→15，你需要输出去重后的链表 21→-15→-7，还有被删除的链表 -15→15。

输入格式：

输入在第一行给出 L 的第一个结点的地址和一个正整数 N（≤105，为结点总数）。一个结点的地址是非负的 5 位整数，空地址 NULL 用 -1 来表示。

随后 N 行，每行按以下格式描述一个结点：

地址 键值 下一个结点

其中地址是该结点的地址，键值是绝对值不超过104的整数，下一个结点是下个结点的地址。

输出格式：

首先输出去重后的链表，然后输出被删除的链表。每个结点占一行，按输入的格式输出。

输入样例：

00100 5
99999 -7 87654
23854 -15 00000
87654 15 -1
00000 -15 99999
00100 21 23854

输出样例：

00100 21 23854
23854 -15 99999
99999 -7 -1
00000 -15 87654
87654 15 -1

思路：

使用不去重的map记录字符串对应的下一个字符串及其键值

struct node{
	string ne ; int w;
};

unordered_map<string,node>mp;

for(int i = 1 ; i <= T ; i ++){
		string a , b;int c;
		cin >> a >> c >> b;
		mp[a] = {b,c};
}

从第一个往后找，如果键值有重复或者出现过就将后一个放入vector中，输出这次的结果继续进行dfs；

void dfs(string s){
	if(s == "-1")return;
	string t = mp[s].ne;
	while(t != "-1" && (abs(mp[t].w) == abs(mp[s].w)||
		vis[abs(mp[t].w)])){
	 	v.push_back(t);
	 	t = mp[t].ne;
	}
	vis[abs(mp[s].w)] = 1;
	cout << s << " " << mp[s].w << " " << t << endl;
	dfs(t);
}

总代码：

#include<bits/stdc++.h>
using namespace std;
struct node{
	string ne ; int w;
};
unordered_map<string,node>mp;
unordered_map<int,int>vis;
vector<string>v;
void dfs(string s){
	if(s == "-1")return;
	string t = mp[s].ne;
	while(t != "-1" && (abs(mp[t].w) == abs(mp[s].w)||
		vis[abs(mp[t].w)])){
	 	v.push_back(t);
	 	t = mp[t].ne;
	}
	vis[abs(mp[s].w)] = 1;
	cout << s << " " << mp[s].w << " " << t << endl;
	dfs(t);
}
int main(){
	ios::sync_with_stdio(false);
	string s; int T;
	cin >> s >> T;
	for(int i = 1 ; i <= T ; i ++){
		string a , b;int c;
		cin >> a >> c >> b;
		mp[a] = {b,c};
	}
	dfs(s);
	for(int i = 0 ; i < v.size() ; i ++){
		cout << v[i] << " " << mp[v[i]].w << " " ;
		if(i != v.size() - 1)cout << v[i+1] << endl;
		else cout << "-1" << endl;
	}
}
