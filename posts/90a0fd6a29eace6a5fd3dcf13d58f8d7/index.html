VS2019安装和使用（C语言）

VS2019视频、VS2022视频，博主看视频做的笔记，视频讲解比较详细更容易上手

一、下载

  • VS2019官网下载
  • 分为三个版本，个人用户下载Community 2019足够了，登录微软账号免费
  • 点击下载后跳转新页面可能要等一会才能弹出下载窗口，可能需要翻墙

二、安装

  1. 安装Visual Studio Installer强制安装在c盘

  2. 运行Visual Studio Installe
    学习c语言选择安装这两个即可

  3. 安装位置推荐选择D盘，虽然选择的D盘但一些库函数还是会默认安装在C盘

  4. 选完后点击安装，等待下载和安装，重启电脑安装完成，没有桌面快捷方式，在菜单栏打开运行

  5. Visual Studio Installer不要卸载，修改工作负荷和卸载都要用到

  6. D盘可能会多出一个Windows Kits目录，不要随便移动位置否则项目会找不到库报错

  7. 会多安装一个Blend for Visual Studio 2019，不用管blend主要设计界面vs主要写代码

  8. 配置环境，常规是所有语言都兼容的环境

三、创建项目

  1. 这里选中C++筛选一下，推荐使用window桌面向导新建项目，其余的都是桌面向导的一部分，因为比较常用放在外面相当于快捷方式

  2. 配置项目名，选中项目的存放位置，解决方案和项目名默认相同

四、使用技巧

  • 调整字体
    工具栏→选项→环境→字体和颜色，红框位置可以调整不同窗口的字体，字体推荐Consolas

  • 视图：所有的小窗口都可以在视图中找到，也可以设置工具栏的快捷工具

  • 出现错误要点否，根据输出和错误列表解决问题
