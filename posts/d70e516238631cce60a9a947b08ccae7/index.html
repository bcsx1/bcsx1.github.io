二、第三方库的pip安装方法（最主要的方式，需联网使用）

pip工具可在 windows、Linux、Mac OS下运行，以常用的windows系统为例说明。

2.1 pip命令帮助信息

打开cmd窗口，键入“pip -h”，单击回车键，即可查看pip的帮助信息。

2.2 常用pip命令

2.2.1 pip install <第三方库名>

安装一个新的第三方库。

2.2.2 pip install -U <第三方库名>

指定一个已安装的第三方库，将其更新到最新版本。

2.2.3 pip uninstall <第三方库名>

卸载一个已安装的第三方库。

2.2.4 pip download <第三方库名>

下载指定的第三方库，但不安装。

2.2.5 pip show <第三方库名>

列出一个第三方库的详细信息（介绍、使用方法等）。

2.2.6 pip search <关键词>

根据关键词在名称和介绍中搜索第三方库。

2.2.7 pip list

列出当前系统已经安装的第三方库。
