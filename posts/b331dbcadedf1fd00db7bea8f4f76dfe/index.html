Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

基本指令

  • v-html 可以进行渲染，会识别html标签

    <h2 v-html="str">++++++ ------</h2>

  • v-text 可以进行渲染，只会识别成普通文本

    <h3 v-text="str">++++++ ------</h3>

  • v-if v-else-if v-else 控制元素的显示隐藏

    注意：如果这三个都用或其中两个使用的话，中间不要添加其他内容

    <p v-if="num == 1">看看我在不在1 if</p>
<p v-else-if="num == 2">看看我在不在2</p>
<p v-else>看看我在不在3</p>

  • v-show 控制元素的显示隐藏，通过display:none

    <p v-show="flag">看看我在不在 show</p>

  • v-for

    • v-for="(item ,index ) in arr" item => 数组中每一项 index => 每一项所对应的下标
    • v-for="(value, key, index) in obj" value => 键值 key=>键名 index => 下标
    • v-for=“i in 10” 1-10

  • v-bind:属性名=“属性值” => :属性名=“属性值”

    注意：属性值不加引号的都是变量

  • v-on:事件类型=“事件处理函数” => @事件类型=“事件处理函数”

    <button v-on:click="fun"></button>

笔试题

  1. {{}} 和 v-html 和 v-text的共同点和区别

    答：共同点是都可以将一个变量渲染到元素内，区别是，{{}} 相当于是一个占位，并不影响元素内其他内容的输出，v-html和v-text会完全替换元素原有内容,v-html会识别html字符串并进行解析，而v-text只会渲染成普通文本

  2. v-if 和 v-show的区别，该如何选择

    都可以做到显示和隐藏，v-if是通过挂载和卸载节点来做到的，而v-show是通过css的display：none来隐藏的。如果该节点需要频繁的切换显示和隐藏，则选用v-show
