爬楼梯的算法面试题

一个小孩爬楼梯，每次可以爬1个、2个或3个台阶，编程求出这个小孩爬完10个台阶的楼梯一共有多少种走法。

这个问题可以用递归的方法来解决。我们可以定义一个函数f (n)，表示爬n个台阶的楼梯有多少种走法。那么，f (n) = f (n-1) + f (n-2) + f (n-3)，因为每次可以爬1个、2个或3个台阶。当n为0、1、2、3时，f (n)分别为1、1、2、4。用python编程，可以写成如下的代码：

def f (n):
  if n == 0 or n == 1 or n == 2:
    return 1
  elif n == 3:
    return 4
  else:
    return f (n-1) + f (n-2) + f (n-3)

print (f (10)) # 输出结果为274

这是一种可能的解答，你也可以尝试其他的方法，比如动态规划或者数学公式。

一个小孩爬楼梯，每次可以爬1个、2个或3个台阶，编程求出这个小孩爬完100个台阶的楼梯一共有多少种走法。 

 这个问题可以用不同的方法来解决，比如递归、动态规划、矩阵快速幂或者斐波那契数列公式。我可以用Python编程给你展示一些可能的解法。

方法一：递归

def f (n):
  if n == 0 or n == 1 or n == 2:
    return 1
  elif n == 3:
    return 4
  else:
    return f (n-1) + f (n-2) + f (n-3)

print (f (100)) # 输出结果为180396380815100901214157639

这种方法的时间复杂度是O(3^n)，空间复杂度是O(n)，效率很低，因为会重复计算很多子问题。

方法二：动态规划

 

def f (n):
  dp = [0] * (n + 1) # 创建一个长度为n+1的数组，存放每个台阶的走法数
  dp[0] = dp[1] = dp[2] = 1 # 初始化前三个台阶的走法数
  dp[3] = 4
  for i in range (4, n + 1): # 从第四个台阶开始，递推计算每个台阶的走法数
    dp[i] = dp[i-1] + dp[i-2] + dp[i-3] # 每个台阶的走法数等于前三个台阶的走法数之和
  return dp[n] # 返回第n个台阶的走法数

print (f (100)) # 输出结果为180396380815100901214157639

这种方法的时间复杂度是O(n)，空间复杂度也是O(n)，效率比递归高，因为避免了重复计算子问题。

方法三：矩阵快速幂

def matrix_mul (a, b): # 定义一个矩阵乘法的函数
  c = [[0, 0], [0, 0]] # 创建一个2x2的零矩阵，存放乘法结果
  for i in range (2):
    for j in range (2):
      for k in range (2):
        c[i][j] += a[i][k] * b[k][j] # 按照矩阵乘法的规则计算
  return c # 返回乘法结果

def matrix_pow (a, n): # 定义一个矩阵快速幂的函数
  res = [[1, 0], [0, 1]] # 创建一个2x2的单位矩阵，作为幂的初始值
  while n > 0: # 当n大于0时，循环计算
    if n & 1: # 如果n是奇数，就乘以矩阵a
      res = matrix_mul (res, a)
    n >>= 1 # 将n右移一位，相当于除以2
    a = matrix_mul (a, a) # 将矩阵a平方
  return res # 返回幂的结果

def f (n):
  if n == 0 or n == 1 or n == 2: # 如果n小于等于2，直接返回1
    return 1
  elif n == 3: # 如果n等于3，直接返回4
    return 4
  else: # 否则，利用矩阵快速幂的方法计算
    a = [[1, 1], [1, 0]] # 创建一个2x2的矩阵a，其特征值和特征向量与斐波那契数列有关
    res = matrix_pow (a, n - 2) # 计算矩阵a的n-2次方
    return 4 * res[0][0] + res[0][1] # 返回第n个台阶的走法数，根据矩阵运算的结果推导得到

print (f (100)) # 输出结果为180396380815100901214157639

这种方法的时间复杂度是O(log n)，空间复杂度是O(1)，效率比动态规划高，因为利用了矩阵的性质，减少了计算次数。

方法四：斐波那契数列公式

import math # 导入数学模块，用于计算平方根和幂

def f (n):
  if n == 0 or n == 1 or n == 2: # 如果n小于等于2，直接返回1
    return 1
  elif n == 3: # 如果n等于3，直接返回4
    return 4
  else: # 否则，利用斐波那契数列公式的方法计算
    sqrt5 = math.sqrt (5) # 计算根号5
    result = (math.pow ((1 + sqrt5) / 2, n + 1) - math.pow ((1 - sqrt5) / 2, n + 1)) / sqrt5 # 计算斐波那契数列的通项公式
    return int (result) # 返回第n个台阶的走法数，转换为整数

print (f (100)) # 输出结果为180396380815100901214157639

这种方法的时间复杂度是O(1)，空间复杂度也是O(1)，效率最高，因为直接利用了数学公式，没有循环或递归。
