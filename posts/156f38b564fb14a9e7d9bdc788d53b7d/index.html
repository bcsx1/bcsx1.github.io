DishController 完善代码

菜品删除&批量删除：

因为前端返回url请求格式一样，区别只有List<Long>和Long的区别。所以这里统一按照数组形式操作。

    /**
     * 删除菜品
     * @param id
     * @return
     */
    @DeleteMapping
    public R<String> remove(@RequestParam("ids") List<Long> id){
        log.info("根据id删除菜品");
        dishService.removeByIds(id);
        return R.success("删除菜品成功");
    }

菜品状态更新&批量更新：

原理同上

    @PostMapping("/status/{status}")
    public R<String> statusUpdate(@PathVariable int status,
                                  @RequestParam("ids") List<Long> ids){
        log.info("更改菜品状态");
        for (Long id:ids){
            Dish dish=new Dish();
            dish.setId(id);
            dish.setStatus(status);
            dishService.updateById(dish);
        }
        return R.success("状态更改成功");
    }
