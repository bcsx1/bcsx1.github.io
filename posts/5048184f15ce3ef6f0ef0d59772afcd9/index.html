MySQL笔记1

   MySQL是一种关系数据库管理系统，所使用的 SQL 语言是用于访问数据库的最常用的标准化语言，其特点为体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，在 Web应用方面 MySQL 是最好的 RDBMS (Relational Database Management System：关系数据库管理系统) 应用软件之一。

2. 安装

note：下述操作均为免安装版；
操作步骤归纳：
• MySQL压缩包 解压缩到D盘（选择D盘是为了不影响C盘空间），文件夹更名为mysql（简单的文件夹名为了方便记忆）;

• 设置环境变量（path变量）

• 编写MySQL配置文件（）
在解压目录下新建 my.ini 文件
注意：没有my.ini文件，也可以安装，但是偶尔会出现安装失败或无法运行等一些问题，所以建议配置my.ini文件。

将下面文本拷贝进my,ini文件中

[mysqld]
# 设置3306端口
port=3306
# 设置mysql的安装目录   ----------是你的文件路径-------------
basedir=D:\mysql
# 设置mysql数据库的数据的存放目录  -------是你的文件路径data文件夹 自行创建，可以是MySQL文件放在一起，也可以单独存放比如datadir=E:\mysql\mysql\data
#datadir=D:\mysql\data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。
max_connect_errors=10
# 服务端使用的字符集默认为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306
default-character-set=utf8mb4

• 以管理员身份运行cmd（必须， 一定要是管理员身份，否则由于后续部分命令需要权限，出现错误！）

• 提示符界面使用语句安装MySQL
①下转到mysql的bin目录下：(环境变量设置好后，不需要跳转到bin目录下)
②安装mysql的服务： mysqld --install
③初始化mysql： mysqld --initialize --console
    初始化会产生一个随机密码, 记住这个密码，后面必须修改为自己的密码才能使用MySQL；
④开启mysql的服务: net start mysql
以上操作如下图：

注意：执行到 net start mysql 时出现了一个错误；这个错误可能是之前操作不当造成的（比如，不是以管理员身份启动的cmd，或者因为安装服务路径不对，没有配置my.ini文件）。如下图

解决方法如下：

发生系统错误2 解决方法1：
移除并重新安装MySQL，注意要在mysql目录下安装服务。
先执行 mysqld --remove，移除MySQL；如下图

再删除data目录里的全部内容，重启计算机。
重新开启cmd（管理员身份）重新安装MySQL，执行如下操作，MySQL服务启动成功；


发生系统错误2 解决方法2：
网上下载的Windows操作系统尝尝用方法1无法解决；这是因为该操作系统的注册表中残留了MySQL的注册信息需要删除；

命令行输入指令“regedit”，打开注册表；
搜索注册表含有mysql字样的信息；
发现残留信息，将其删除即可。

发生系统错误2 解决方法3：
有些电脑上，尝试过解决方法1和方法2后，故障依旧。那么，就可能是MySQL根目录下my.ini文件内的参数影响了启动MySQL服务。
删除my.ini文件后，重启命令提示行输入net start mysql，MySQL服务启动正常了。

注意：我遇到过因为MySQL根目录没有my.ini文件不能安装的情况（创建一个my.ini，拷贝上述参数，问题解决），也遇到过因为有my.ini文件而不能启动MySQL服务的情况（可能是配置写错了，但是我没看出来）；这两种对立的情况，为什么会发生，我并不明白，但是我能肯定“发生系统错误2”用这3中方法肯定能解决！！

更改密码：
方法1：（无需进入MySQL）

// 无需登录MySQL
// 格式
mysqladmin -u用户名 -p旧密码 password 新密码
//例子：
mysqladmin -uroot -p123456 password 123

方法2：（需要登录MySQL）

// 需要登录MySQL
// 格式
 ALTER USER 'root'@'localhost' IDENTIFIED BY '密码';
//例子：
 ALTER USER 'root'@'localhost' IDENTIFIED BY '111111';

MySQL安装完毕！！

mysqld --initialize --console 语句的解释

cd 到mysql bin文件夹中运行mysqld --initialize --console

2018-04-14T03:15:52.711719Z 1 [Note] A temporary password is generated for root@localhost: k9CRxtbKgt?j

记住这里的密码：root@localhost: k9CRxtbKgt?j
注释：
root为账户
@localhost 表示服务器地址 localhost本地服务器
k9CRxtbKgt?j 是初始密码
