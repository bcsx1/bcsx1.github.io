Ubuntu 系统下如何远程访问 Windows 桌面 ?

你一定听说过 Windows 应用程序远程桌面连接。该应用程序系统自带不用安装，并允许您远程访问另一台 PC 或服务器。它使用远程桌面协议建立远程桌面连接会话。

一些 Linux 发行版可能会提供 RDP 客户端来连接到 Windows 系统。但是，对于某些 linux 发行版，您可能需要安装 RDP 客户端来建立远程桌面连接。

作为一个 Linux 用户，有一些可用的 rdp 工具，您可以安装并用于 windows 远程连接。在这篇文章中，我们将解释如何在 Ubuntu linux 上安装 RDP 客户端，并使用它们访问(或连接)远程 windows 桌面。

Reminna

Reminna 是一个免费的、开源的、功能强大的远程桌面客户端，用于远程桌面共享。由于它提供了许多有用的特性，大多数 Linux 和 UNIX 用户都采用 remna 客户端来连接远程桌面。

Ubuntu 系统上，使用如下命令安装

$ sudo apt update
$ sudo apt install -y remmina remmina-plugin-vnc

安装完 remmina，就可以访问 gui 进行远程桌面连接。

输入您的 windows 系统 IP 地址并按 Enter。系统将提示您输入用户名和密码的详细信息。提交所有详细信息并控制远程桌面。

Vinagre

Vinagre 是 GNOME 桌面环境的 SSH，VNC 和 RDP 客户端。它具有高级功能，例如同时连接多个服务器并使用选项卡之间切换。Vinagre 还支持客户端和服务器之间的复制 / 粘贴。

Ubuntu 上安装 Vinagre，使用以下命令

$ sudo apt update
$ sudo apt install -y vinagre

安装完成后，转到应用程序列表并搜索远程桌面。

单击如下图所示的应用程序

单击 Connect，从下拉菜单中选择 RDP。输入远程桌面凭据并单击连接。

KRDC

KRDC 是为 KDE 桌面环境设计的远程桌面工具。KRDC 的安装带有 VNC 和 RDP 两种协议，使您可以轻松地访问您的远程桌面。

要在系统中安装和配置 KRDC，请执行以下命令

$ sudo apt update
$ sudo apt install -y krdc

安装完成后，您就可以使用 KRDC 客户机了。您可以在打开新的 gui 工具的命令行中输入 krdc。您还可以在应用程序列表中搜索 krdc 并启动

单击 KRDC 应用程序，您将在屏幕上看到以下图形用户界面。

从下拉菜单中选择 RDP 协议，输入您的远程桌面 IP 地址，然后单击 Enter。

您可以自定义配置，完成自定义之后，单击 ok。在下一个提示中，您将被要求输入用户名和密码

按照要求输入用户名和密码，然后就可以使用远程桌面了。

FreeRDP

FreeRDP 是一个免费的、开源的远程桌面协议客户端。FreeRDP 是在 Apache 许可下发布的一个可移植 rdp 协议。

使用以下命令在 Ubuntu 中安装和配置 freeRDP。

$ sudo apt update
$ sudo apt install -y freerdp2-x11

安装完成后，可以使用以下命令访问远程桌面

$ sudo xfreerdp /u:remote_user /p:remote_password /v:remote_host_ip

运行此命令后，可能会收到接受证书的警告，输入 yes，就可以访问远程桌面
