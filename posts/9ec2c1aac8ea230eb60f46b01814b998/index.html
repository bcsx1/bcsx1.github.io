一、什么是哈希表   

         百度百科：哈希表是种数据结构，它可以提供快速的插入操作和查找操作。

        哈希表是一种元素+索引的数据结构，哈希表的生成方式有拉链法和探测法。

        哈希表一般用于查找表中是否有指定的元素，最简单的哈希表结构就是数组，然后也有set和map。

二、哈希表的使用

1、有效的字母异位词（数组）

1.1、问题描述

        给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

        若 s 和 t 中每个字符出现的次数都相同，则称 s 和 t 互为字母异位词。

1.2、解题思路

        因为本题中只包含小写字母，所以申请一个大小为26的数组，遍历一遍s，将出现的次数存在数组中（数组的下标就为s[i]-'a'），然后再进行遍历t，每次在数组上减去当前字母出现的次数，到最后如果全为0就说明是互为字母异位词，否则不是。

1.3、代码实现

        1、循环遍历s，将每个字母出现的次数++。

        2、循环遍历t，将每个字母出现的次数--。

        3、判断最后的数组是否全是0。

bool isAnagram(string s, string t) {
        int sLen = s.size();
        int tLen = t.size();
        int temp[26] = {0};
        //循环s，将出现的次数存入数组中
        for(int i=0;i<sLen;i++)
        {
            temp[s[i]-'a']++;
        }
        //循环t，减去字母出现的次数
        for(int i=0;i<tLen;i++)
        {
            temp[t[i]-'a']--;
        }
        //判断是否全为0
        for(int i=0;i<26;i++)
        {
            if(temp[i] != 0)
            {
                return false;
            }
        }

        return true;
    }

2、两个数组的交集（set）

2.1、问题描述

        给定两个数组，编写一个函数来计算它们的交集。

输出结果中的每个元素一定是唯一的。 我们可以不考虑输出结果的顺序。

2.2、解题思路

        因为结果是没用重复的，所以这里可以使用set进行解题。

将其中一个数组转换为集合set，遍历另一个数组，在前一个set中寻找是否有其相同的元素，有的话就说明是一个交集，存入结果集中。

2.3、代码实现

        1、将nums1转化为set集合（可以方便另一个数组进行查找）。

        2、遍历nums2数组，判断该元素是否存在于set中，是的话就存入结果集中。

vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        //定义要返回的结果集，先用set（可以去重），之后再转为数组
        unordered_set<int> result_res;
        //将nums1数组转化为set
        unordered_set<int> num_set(nums1.begin(),nums1.end());
        //遍历nums2数组
        for(int num:nums2)
        {
            //判断该元素在set中是否存在
            if(num_set.find(num) != num_set.end())
            {
              //存在就插入结果集中
                result_res.insert(num);
            }
        }
        //将结果集转化为数组
        return vector<int>(result_res.begin(),result_res.end());
    }

3、两数之和（map）

3.1、问题描述

        给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

        你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍

3.2、解题思路

        本题使用map进行存储，将数组的值存入key中（查找更快），将下标存入value中（便于结果返回），先遍历数组，用target减去当前值就是下一个元素的值，然后在map中查找是否有此元素的存在，如果不存在的话就存入map中以便于后序的元素进行查找。（也可以先把所有元素都存入map然后遍历数组在map中寻找，但是要注意重复的情况，还有多遍历了一边数组增加了时间复杂度）

3.3、代码实现

        1、遍历数组，在map中进行查找，如果没用对应的元素就存入map中用于后续元素的查找，如果找到了对应的元素就直接返回当前数组的下标和map中存放的下标值。

vector<int> twoSum(vector<int>& nums, int target) {
        int len = nums.size();
        //map1存放遍历过的元素
        std::unordered_map<int,int> map1;
        for(int i=0;i<len;i++)
        {
            //查找map1中是否有对应的元素
            auto iter = map1.find(target-nums[i]);
            if(iter != map1.end())
            {
              //找到了就返回map1中的下标值以及当前数组的下标值
                return {iter->second,i};
            }
            //将遍历过的元素插入map1中
            map1.insert(pair<int,int>(nums[i],i));
        }
        return {};
    }

4、四数之和（map）

4.1、问题描述

        给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

        若两个四元组元素一一对应，则认为两个四元组重复。a，b，c，d互不相同。

4.2、解题思路

        先将数组进行从小到大的排序，然后两层for循环遍历nums1和nums2，在两层for循环中将第三个数作为left，第四个数为right，将四个数相加与target进行判断，如果target大就移动left，如果target小就移动right，直到right<left循环结束。

4.3、代码实现

        1、先将数组进行排序。

        2、先来两层for循环，作为第一个数和第二个数（进行剪枝和去重判断）。

        3、然后选取第三个数（left）和第四个数（right），根据与target值的大小进行left或right的移动，target值大就移动left，target值小就移动right，找到正确结果之后也要对left和right去重然后继续遍历中间是否有符合条件的情况存在，直到left超出了right的范围。

vector<vector<int>> fourSum(vector<int>& nums, int target) {
        int len = nums.size();
        vector<vector<int>> res;
        //将数组排序
        sort(nums.begin(),nums.end());
        int i,j;
        int left;
        int right;
        //第一层循环从0-len
        for(i=0;i<len;i++)
        {
            //最小的数为正数且比target大，直接返回没结果
            if(nums[i]>target && nums[i]>=0) break;     //剪枝
            //nums的元素不能相同，如果当前的i等于上一个就进入下次循环
            if(i>0 && nums[i]==nums[i-1]) continue;     //去重
            //第二层循环从i+1-len
            for(j=i+1;j<len;j++)
            {
              //同样的进行剪枝（最小的两个数大于target）
                if((nums[i]+nums[j])>=0 && (nums[i]+nums[j])>target) break;//剪枝             
                //同样的去重复
                if(j>i+1 && nums[j]==nums[j-1]) continue;     //去重
                //left为第三个数
                left = j+1;
                //right为第四个数
                right =nums.size()-1;
                //循环遍历的条件
                while(left<right)
                {
                  //如果target值小了，right就左移
                    if(((long)nums[i]+nums[j]+nums[left]+nums[right]) > target) right--;
                    //如果target的值大了，left就右移
                    else if(((long)nums[i]+nums[j]+nums[left]+nums[right]) < target) left++;
                    //相等就存入结果集中
                    else 
                    {
                        res.push_back(vector<int>{nums[i],nums[j],nums[left],nums[right]});
                        //将left和right进行去重
                        while(nums[right]==nums[right-1] && right>left) right--;
                        while(nums[left]==nums[left+1] && right>left) left++;
                        //继续寻找中间是否还有符合条件的值
                        left++;
                        right--;
                    }
                }
            }
        }
        return res;

    }
