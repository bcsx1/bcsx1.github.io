SIMD

CPU采用SIMD技术大大提高计算速度，特别是在矩阵运算上。Intel和AMD都在不同程度上实现了这样的技术，Intel拥有从最早的MMX，到SSE,SSE2；AMD是3D NOW!和3D NOW PRO，作为程序员学习下也挺好，说不定哪天优化代码的时候就用上了。 SIMD全称是Single Instruction Multiple Data,就是单个指令能够处理多个数据包。我的理解是，在SIMD之前，寄存器在对数组中各元素做加减的时候，必须按次序取出，放入相应寄存器，然后再相加，举个例子： 有两个数组 a[2] = {2,3}, b[2] = {3,4},然后 MOV ESI , a MOV ESI , a+1 MOV EDI , b MOV EDI , b+1 MOV EAX , [ESI] MOV EAX , [ESI] MOV EBX , [EDI] MOV EBX , [EDI] ADD EAX ,EBX ;完成了数组中a[0]和b[0]的相加运算 ADD EBX , EAX; 完成数组中a[1]和b[1]的相加运算 一共进行了两组运算，假如运用SIMD进行运算的话 MOV ESI a MOV EDI b MOVAPS MMX0 [ESI] MOVAPS MMX1 [EDI] PADDD MMX0 MMX1 ;一组指令就行了。 MMX 允许一次向MMX寄存器中存储最大64bits的数据，比如一个vector,数据值必须是整数，而且占用的是FPU运算单元。 3DNOW 比MMX好的一点是可以运用浮点数运算。 SSE 从PentiumIII开始新加入了8个寄存器，可以存储最大到128bits的一组数据，能够迅速进行vector4运算 3DNOW PRO基本等于SSE SSE2从名字上看比SSE更强大了，具体怎么个强大没去想，先放着了。 怎样测试自己的CPU支持哪种SIMD功能呢？这需要用到CPUID这条指令。还是先定义一个数组准备接受测试的结果 typedef struct CPUINFO_TYP { bool bSSE; bool bSSE2; bool b3DNOW; bool bMMX; char name[48]; //cpu name bool bEXT; //扩展功能 bool MMXEX; //AMD的扩展 bool 3DNOW; //Intel的扩展 char vendor[12]; //vendor name }CPUINFO; 检测时候，只需要喂给EAX一个值，然后调用CPUID指令，相应结果就会返回。比如，我向EAX送去0，CPUID指令就会返回不超过12个字的制造商名称，分别向EBX,EDX,ECX返回4个字；如果向EAX送去1，CPUID会返回CPU功能的一个列表，并且，如果CPU属于Intel，则返回brandID到EBX，如果是AMD，则返回brandID到EAX。如果送入的值是Ox80,那么即将检查的是CPU有无扩展功能，送入0x80到EAX后，CPUID会返回一个值回到EAX，如果该值大于0x80,那就再送出0x80000001到EAX再执行CPUID，我们就可以得到这些扩展功能的描述。如果是IntelCPU，就没必要找扩展功能了，如果是AMD,可以注意下有没有extended MMX和enhanced 3D NOW.

IPC，全名Inter Process Communication即进程间通讯，在同一台机器上的两个进程就用IPC，不能跨物理机器。IPC包括共享内存、队列、信号量等几种方式，由于IPC通讯效率之高，所以大量的Unix下软件都用IPC通讯，如oracle。 TCP/IP，全名Transmission Control Protocol/Internet Protocol即传输控制协议/网间网协议，TCP/IP可在同一台机子或两台物理机或不同操作平台之间的两个进程进行通讯。标准IPC/IP通讯过程：在主机1上，应用层将一串应用数据流传送给传输层；传输层将应用层的数据流截成分组，并加上TCP报头形成TCP段，送交网络层；在网络层给TCP段加上包括源、目的主机2 IP地址的IP报头，生成一个IP数据包，并将IP数据包送交链路层；链路层在其MAC帧的数据部分装上IP数据包，再加上源、目的主机2的MAC地址和帧头，并根据其目的MAC地址，将MAC帧发往目的主机2或IP路由器。在目的主机2，链路层将MAC帧的帧头去掉，并将IP数据包送交网络层；网络层检查IP报头，如果报头中校验和与计算结果不一致，则丢弃该IP数据包；若校验和与计算结果一致，则去掉IP报头，将TCP段送交传输层；传输层检查顺序号，判断是否是正确的TCP分组，然后检查TCP报头数据。若正确，则向源主机1发确认信息；若不正确或丢包，则向源主机1要求重发信息；在目的主机2，传输层去掉TCP报头，将排好顺序的分组组成应用数据流送给应用程序。这样目的主机2接收到的来自源主机1的字节流，就像是直接接收来自源主机1的字节流一样。 如果两个进程在同一台机子且在同一个操作平台，可选择IPC或TCI/IP两种通讯方式都可以，但IPC效率高于TCP/IP。采用IPC通讯，进程1直接把通讯包发给进程2，采用TCP/IP通讯，进程1将要先把通讯包发给“LO”即本地环路接口，通过"LO"再把通讯包发给进程2。 如果两个进程在不同的物理机上或在不同的操作平台，则不能用IPC，这时用TCP/IP通讯，进程1把通讯包发给本机的物理网卡1，物理网卡1通过网线把通讯包发给进程2所在的机器的物理网卡2，网卡2再把通讯包发给进程2。

C语言中extern修饰符的用法

今天休假，大概总结一下extern在C/C++语言中的常见用法吧。不是为了说明茴香豆有多种写法，而是发现很多同事，包括我自己，使用的时候还是经常犯晕。 在C语言中，修饰符extern用在变量或者函数的声明前，用来说明“此变量/函数是在别处定义的，要在此处引用”。 0. extern修饰变量的声明。举例来说，如果文件a.c需要引用b.c中变量int v，就可以在a.c中声明extern int v，然后就可以引用变量v。这里需要注意的是，被引用的变量v的链接属性必须是外链接（external）的，也就是说a.c要引用到v，不只是取决于在a.c中声明extern int v，还取决于变量v本身是能够被引用到的。这涉及到c语言的另外一个话题－－变量的作用域。能够被其他模块以extern修饰符引用到的变量通常是全局变量。还有很重要的一点是，extern int v可以放在a.c中的任何地方，比如你可以在a.c中的函数fun定义的开头处声明extern int v，然后就可以引用到变量v了，只不过这样只能在函数fun作用域中引用v罢了，这还是变量作用域的问题。对于这一点来说，很多人使用的时候都心存顾虑。好像extern声明只能用于文件作用域似的。 1. extern修饰函数声明。从本质上来讲，变量和函数没有区别。函数名是指向函数二进制块开头处的指针。如果文件a.c需要引用b.c中的函数，比如在b.c中原型是int fun(int mu)，那么就可以在a.c中声明extern int fun（int mu），然后就能使用fun来做任何事情。就像变量的声明一样，extern int fun（int mu）可以放在a.c中任何地方，而不一定非要放在a.c的文件作用域的范围中。对其他模块中函数的引用，最常用的方法是包含这些函数声明的头文件。使用extern和包含头文件来引用函数有什么区别呢？extern的引用方式比包含头文件要简洁得多！extern的使用方法是直接了当的，想引用哪个函数就用extern声明哪个函数。这大概是KISS原则的一种体现吧！这样做的一个明显的好处是，会加速程序的编译（确切的说是预处理）的过程，节省时间。在大型C程序编译过程中，这种差异是非常明显的。 2. 此外，extern修饰符可用于指示C或者C＋＋函数的调用规范。比如在C＋＋中调用C库函数，就需要在C＋＋程序中用extern “C”声明要引用的函数。这是给链接器用的，告诉链接器在链接的时候用C函数规范来链接。主要原因是C＋＋和C程序编译完成后在目标代码中命名规则不同。

100首经典的华语经典歌曲

一首歌曲能够流行3个月。我们将它称之为流行歌曲。 一首歌曲流行了20年，就成了经典歌曲 1、《恋曲1990》罗大佑 这首歌的经典程度无法用文字形容，罗大佑的情歌的确是极品中的极品。 2、《天花乱坠》达明一派 达明一派绝对是上世纪香港乐坛商业化下的异数，但他们的地位却是非同寻常的重要。 3、《马路天使》达明一派 这首歌很容易让人想起经典的同名电影，达明一派却带给我们新鲜的演绎方式。 4、《顺流逆流》徐小凤 徐小凤的经典代表作，现在很多人之所以不知，只怪她的成名时间比歌神许冠杰还要早。 5、《牵手》苏芮 执子之手，与子偕老，苏芮的激昂声音让很多人难忘，爱情的道理其实就是这么简单。 6、《千千阕歌》陈慧娴 当年的绝对金曲，要是那时有天后，陈慧娴早就当选了。 7、《你的眼神》蔡琴 蔡琴的中低音水准公认为一流，她唱情歌的境界也是最高的， 也许是太婉转了，你细细听才会感觉到一丝忧伤。 8、《一生何求》陈百强 百强的情歌让人回味无穷，但这首富有哲理的歌可谓他的完美之作，可惜他到离开这个世界的时候也未必知道什么是一生何求。 9、《爱在深秋》谭咏麟 谭校长的好歌也是不计其数，选择一首歌实在很难，但这首歌绝对能展现他的超群实力。 10、《改变1995》黄舒俊 黄舒俊当年要向罗大佑挑战，他当然有过人之处，这首歌好像现代的司马迁在书写一段历史。 11、《大海》张雨生 这首歌意味着什么，我不想累述，但它的确是雨生生前最值得铭记的单曲之一。 12、《当年情》张国荣 电影《英雄本色》最近被评为百部港片之首，哥哥的金曲和发哥的小马哥应该同等值得我们尊重和怀念。 13、《女人花》梅艳芳 女人四十如花，我觉得可以概括她的一生，虽短短四十二载，却是分外灿烂和光辉。 14、《爱的代价》张艾嘉 音乐教父李宗盛每次演唱会必唱的曲目，后来也有梁扳，但我还是选择了张艾嘉，她也出色的完成了对爱情的总结。 15、《月亮代表我的心》邓丽君 音乐与抒情的完美结晶，后来的翻唱者也不计其数。 16、《光辉岁月》BEYOND 当年黄家驹为曼德拉所写，也是BEYOND最广为流传的单曲之一，也是打动无数热血青年的歌曲。 17、《把根留住》童安格 沧桑的岁月，总要给经历的人留下一点纪念，所以我选择了这首单曲。 18、《新长征路上的摇滚》崔健 摇滚教父的代表作之一，他的声音可以振奋一代人的心灵。 19、《孤独的人是可耻的》张楚 都说张楚的歌很像诗人的作品，作为当年魔岩三杰之一，他的地位不能忽视。 20、《天空》王菲 两个天空的版本都很出色，这其中有窦唯的制作，王菲的优美声线和自然表达方式让很多人为之倾倒，她也为后来很多歌手模仿。 21、《黑色梦中》窦唯 这是窦唯的代表作，至今被很多摇滚迷所怀念。 22、《水中花》谭咏麟 同样是金曲，也是百听不厌，校长的唱功非同寻常。 23、《你知道我在等你吗》张洪量 张洪量当年的最畅销的金曲，至今仍被一再在KTV里被温习。 24、《偏偏喜欢你》陈百强 刚才说到百强的情歌，真的是情意绵绵，令听者回味无穷，这首歌可谓情歌中的极品。 25、《坠落天使》郑智化 台湾出了罗大佑，也出了郑智化，他的歌同样直面现实，毫不做任何掩饰，犹如一把利刃直插要害。 26、《尘缘》罗文 好歌当然离不开好的作词，尘缘就做到这一点，全篇里很有古典意味。 27、《鲁冰花》甄妮 现在还有谁记得这首歌，但只要经历那个时代的人都不会忘记，歌里有茶香般的诗境和对亲情的美好表达。 28、《童年》罗大佑 本是校园歌曲的代表，可后人都能从其中听到对不再重来的岁月的感叹，富有童趣的写意令人拍手称快。 29、《上海滩》叶丽仪 当年的连续剧万人空巷，我们在记住许文强和冯程程的同时，也记住了这首大气澎湃的主题歌。 30、《甜蜜蜜》邓丽君 一代天后的金曲之一，与之后来无数次出现在电影中相比，邓丽君的甜蜜嗓音在此达到极致。 31、《在那遥远的地方》王洛宾 西部歌王的最广为人知的作品，现在只会被那代人偶尔在茶话饭余提起，但真的值得一听。 32、《追梦人》凤飞飞 这首歌是罗大佑为纪念三毛而制作，她的传奇一生你可以这首单曲里面了解。 33、《让我欢喜让我忧》周华健 周华健的成名作，在李宗盛的打造下，我们又看到了一位实力派歌手的出世。 34、《朋友别哭》吕方 吕方的唱功其实很棒，这首歌能打动多少人，我无法统计，里面的歌词实属上乘。 35、《真的爱你》BEYOND 黄家驹写给母亲的佳作，至今被公认为初学吉他手的必选曲目，母爱的伟大被他用音乐表达出来，无法不打动听者的心。 36、《我是一只小小鸟》赵传 带着摇滚风格和李宗盛的指导，赵传用一个简单的比喻反映出世人的内心世界。 37、《当爱已成往事》李宗盛、林忆莲 两位主唱现在早已分道扬镳，这首歌似乎暗合他们的爱情之旅，现在听来让人唏嘘不已。 38、《狼》齐秦 一身狼嚎，打动了当年的很多热血青年，偶像的力量是无法衡量的。 39、《水手》郑智化 这首歌的入选得益于它的励志内涵，郑智化的歌虽然没有别人优美动听，但很有深度和力度。 40、《兰花草》刘文正 当年的刘文正被很多少女视为白马王子，他的歌的确有非同一般人的情怀，但诸多的翻唱也害苦了他。 41、《特别的爱给特别的你》伍思凯 这首歌可是伍思凯的巅峰之作，当年的销量极其客观，他也凭此奠定了自己的乐坛地位。 42、《至少还有你》林忆莲 林夕的非凡创造力和天后的优良演绎，让人无法不为之打动。 43、《广岛之恋》张洪量、莫文蔚 张洪量的创作才华早已得到公认，这首歌的点唱率至今仍居前茅，两人的配合也是绝佳的范本。 44、《浪人情歌》伍佰 伍佰的歌中有着不一般的沧桑和对底层人的关注，他的歌虽然粗犷但很实在。 45、《真心英雄》周华健、黄耀明、李宗盛、成龙 四个意气相投的好朋友聚在一起，就创造了这一鼓舞人心的经典歌曲，后来多次出现在体育晚会上。 46、《红豆》王菲 王菲的经典单曲，到现在在KTV点唱率仍居高不下，林夕的词可谓精妙透彻。 47、《敢问路在何方》蒋大为 作为家喻户晓的《西游记》主题曲，我们在怀念流行乐的同时，也需要一首优美的美声歌曲才算全面。 48、《滚滚红尘》陈淑桦 又是罗大佑为电影所创，让人听后无不感叹岁月的无情和对年华逝去的感伤。 49、《青藏高原》李娜 高音歌曲中的翘楚之作，李娜的层次感极强，听她的这首歌只能用震撼来形容。 作为一首武侠连续剧的主题曲，里面的侠义境界很好的表达出来，豪情万丈这在以前的华健作品里很难找到。 54、《风雨无阻》周华健 周华健的演唱实力再次得到展现，歌词工整有诗韵，的确值得珍藏。 55、《睡在我上铺的兄弟》老狼 同样是校园民谣的代表，高晓松和老狼的组合得到了很多人的共鸣。 56、《长城》BEYOND 自古云：不到长城非好汉。黄家驹等四人就去了一次长城，结果就诞生了这一惊世之作，大气激昂的前奏给人留下了深刻的影响。 57、《蜗牛》许茹云、齐秦、熊天平、动力火车 周杰伦也曾唱过，但我还是选择了许版，当年为了赈灾所创，却没想到变成了一首励志经典。 58、《情非得以》庾澄庆 哈林整天给人嘻嘻哈哈的感觉，这首歌却唱出了他对爱情的专一。 59、《我想有一个家》潘美辰 潘美辰当年在春晚曾演唱过，让我们此后永远记住了这个女歌手。 60、《风中有朵雨做的云》孟庭苇 孟庭苇的清新外表和带有忧伤的声线，在当年宛如一朵白玉兰出现在人们的眼前。 61、《倩女幽魂》张国荣 哥哥的称号就是从此片传开，他完美演绎人鬼情的同时，黄霑的作词给了电影最好的画龙点睛作用。 62、《梅花三弄》姜育恒 琼瑶作词，姜育恒的演唱让这一诗般的乐曲流传一时。 63、《赤裸裸》郑钧 我的爱赤裸裸，郑钧在此歌里全面爆发，至今让人难以忘怀。 64、《红日》李克勤 李克勤曾被誉为谭校长的接班人，可见其实力，这首歌目前已公认为经典粤语单曲。 65、《每天爱你多一些》张学友 学友唱歌数十年如一日，他演唱时用情之深，声音细腻之及至，这首歌可谓金曲中的金曲。 66、《李香兰》张学友 学友在粤语歌曲的表现其实更胜一筹，这首歌就是明证。 67、《吻别》张学友 当年有四大国歌之一的美誉，作为学友最经典的金曲值得永远铭记。 68、《选择》林子祥、叶倩文 执字之手，与子偕老，这首歌就是最好的诠释。 69、《庸人自扰》小虎队 当年的小虎队红极一时，但这首歌却被很多人忽视，其实里面的歌词很有寓意，相比较以前的众多畅销曲有了不小的进步。 70、《宝贝，对不起》草蜢 一个优秀的青春组合，当年的声望和小虎队不相上下，他们留给那个时代不可磨灭的回忆。 掀起一阵城市民谣的李春波，用最平实的方式打动当年很多人，可惜除了这首歌和《小芳》，后来就鲜有佳作诞生。 75、《爱如潮水》张信哲 情歌王子的成名作，从此他的情歌雷打不动的延续下去。 76、《下沙》游鸿明 游鸿明也被视为情歌高手，他的风格也永远的被定格。 77、《白桦林》朴树 朴树忧郁的像阵风，但这首歌里他好比一个老战士在诉说一场无情的战争。 78、《一起走过的日子》刘德华
