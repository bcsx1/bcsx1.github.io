面试金典14（Python）—— 链表求和（中等）

链表求和

概述：给定两个用链表表示的整数，每个节点包含一个数位。这些数位是反向存放的，也就是个位排在链表首部。编写函数对这两个整数求和，并用链表形式返回结果。

输入：(7 -> 1 -> 6) + (5 -> 9 -> 2)，即617 + 295
输出：2 -> 1 -> 9，即912

输入：(6 -> 1 -> 7) + (2 -> 9 -> 5)，即617 + 295
输出：9 -> 1 -> 2，即912

方法一：字符串+新链表

思路：该思路非常简单，首先对两个链表单独求和。然后建立一个新的链表，逆序循环，依次存储即可。

# 字符串+新链表
# 该思路非常简单，首先对两个链表单独求和。
# 然后建立一个新的链表，逆序循环，依次存储即可。
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        num_l1, num_l2 = 0, 0
        i, j = 0, 0
        while l1 != None:
            num_l1 += (l1.val) * 10 ** i
            l1 = l1.next
            i += 1
        while l2 != None:
            num_l2 += (l2.val) * 10 ** j
            l2 = l2.next
            j += 1
        ans = num_l1 + num_l2
        new = ListNode(0)
        node = new
        for i in str(ans)[::-1]:
            node.next = ListNode(int(i))
            node = node.next
        return new.next

方法二：列表+新链表

思路：思路和上面基本一致，不同在于对列表的处理。列表可以直接翻转，不必再对位数做乘法。

# 列表+新链表
# 思路和上面基本一致，不同在于对列表的处理。
# 列表可以直接翻转，不必再对位数做乘法。
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        list_l1, list_l2 = [], []
        while l1:
            list_l1.append(str(l1.val))
            l1 = l1.next
        while l2:
            list_l2.append(str(l2.val))
            l2 = l2.next
        ans = str(int(''.join(list_l1[::-1])) + int(''.join(list_l2[::-1])))
        new = ListNode(0)
        node = new
        for i in str(ans)[::-1]:
            node.next = ListNode(int(i))
            node = node.next
        return new.next

方法三：直接相加

思路：建立一个新链表，同时遍历原两个链表，并依次相加，满 10 进 1 。需要注意边界条件，即 l1 或 l2 为空的时候。

# 直接相加
# 建立一个新链表，同时遍历原两个链表，并依次相加，满 10 进 1 。
# 需要注意边界条件，即 l1 或 l2 为空的时候。
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        new = ListNode(0)
        node = new
        res = 0
        while l1 or l2:
            if not l1:
                node.next = l2
                l1 = ListNode(0)
            if not l2:
                node.next = l1
                l2 = ListNode(0)
            res += (l1.val + l2.val)
            node.next = ListNode(res % 10)
            res = res // 10
            node = node.next
            l1 = l1.next
            l2 = l2.next
        if res:
            node.next = ListNode(res)
        return new.next

方法四：直接相加（另外一种写法）

思路：和上面的思路基本一致，不再赘述。

# 直接相加（另外一种写法）
# 和上面的思路基本一致，不再赘述。
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        new = ListNode(0)
        node = new
        res = 0
        while l1 and l2:
            temp = (l1.val + l2.val + res) % 10
            res = (l1.val + l2.val + res) // 10
            node.next = ListNode(temp)
            l1 = l1.next
            l2 = l2.next
            node = node.next
        while l1:
            temp = (l1.val + res) % 10
            res = (l1.val + res) // 10
            node.next = ListNode(temp)
            l1 = l1.next
            node = node.next
        while l2:
            temp = (l2.val + res) % 10

            l2 = l2.next
