1、数字字符串转化为数字


1.1、C语言方法

(1) sscanf

sscanf读取格式化的字符串中的数据

sscanf的原型：

#include <stdio.h>
int sscanf(const char *str, const char *format, ...);

参数列表：

str:待读取的字符串;

format:字符串格式描述;

其后是一序列数目不定的指针参数，存储读取后的数据.

返回值：

函数将返回成功赋值的字段个数

示例：
1.将数字字符串转化为整数

#include<stdio.h>
int main()
{
	int year, month, day;

	int count = sscanf_s("20220208", "%04d%02d%02d", &year, &month, &day);
	printf("count=%d, year=%d, month=%d, day=%d\n", count, year, month, day);

//输出：count=3, year=2022, month=02, day=08

	return 0;
}

2.浮点数转换

#include<stdio.h>
int main()
{
	double a,b;
	int count = sscanf_s("113.123456789 31.123456789", "%lf %lf", &a, &b);
	printf("count=%d, a=%.9lf, b=%lf\n", count, a, b);

//输出：count=2, a=113.123456789, b=31.123457

	return 0;
}

sscanf通过空格来区分所要读取的浮点数。

在VS2019种无法直接使用sscanf，只能使用它的安全版本sscanf_s。

(2) atoi

把字符串转换成int类型的变量

函数原型：

头文件<stdlib.h>

int atoi(const char *nptr);将字符串转化为int

long atol(const char *nptr);将字符串转化为long

long long atoll(const char *nptr);将字符串转化为long long

示例：

#include<stdlib.h>
#include<stdio.h>
int main()
{
	const char* str = "123456";
	int a = atoi(str);
	printf("%d\n", a);//输出123456
	return 0;
}

1.2、C++方法

C++中将字符串转化为数字，主要使用下下列这些库函数，使用这些库函数需要包含相应的头文件<string>。

  1. stoi： string型变量转换为int型变量
  2. stol： string型变量转换为long型变量
  3. stoul：string型变量转换为unsigned long型变量
  4. stoll： string型变量转换为long long型变量(常用)
  5. stoull：string型变量转换为unsigned long long型变量
  6. stof： string型变量转换为float型变量
  7. stod： string型变量转换为double型变量(常用)
  8. stold：string型变量转换为long double型变量

函数原型，参数，返回值等详细介绍请参考https://cplusplus.com/reference/string/stod/

在日常使用中， 最常用的是stoll和stod这两个函数， 因为stoll可以兼容stoi，stol，atoi； 而stod可以兼容stof。

示例：
1. stoll()使用场景：

#include<iostream>
#include<string> 
using namespace std;

int main() {
	string s = "123456";
	int a = stoll(s);
	long b = stoll(s);
	long long c = stoll(s);
	cout << a << endl;//输出123456
    cout << b << endl;//输出123456
	cout << c << endl;//输出123456

	cout << "变量s的大小：" << sizeof(s) << endl;//输出28

	return 0;
}

s的大小依然是28，说明转换原理是：创建了一个值为相应数的int类型的临时对象，然后将临时对象赋值，string类型对象本身不会发生变化，其它的函数同理。

2. stod()使用场景

#include<iostream>
#include<string> 
using namespace std;
int main() {
	string s = "12.34";
	double a = stod(s);
	float b = stod(s);
	cout << a <<endl;//输出12.34
	cout << b <<endl;//输出12.34
	return 0;
}

2、数字转化为数字字符串

2.1、C语言方法

（1）sprintf()函数

主要功能是把格式化的数据写入某个字符串中

函数声明： int sprintf(char *string, char *format [,argument,...]);

参数列表：

  • string-- 这是指向一个字符数组的指针，该数组存储了 C 字符串。

  • format-- 指被写入字符串的文本格式

  • [argument]...表示要被读取入字符数组的数据

示例：

#include <stdlib.h>
#include <stdio.h>
int main(void)
{
    int a = 1234560;
    char str[10] = { 0 };
    sprintf_s(str, "%d", a); //从右至左，把数转换为字符串 
    printf("%s\n", str);
}

 sprintf_s在VS2019中无法直接使用，只能使用它的安全版本。

2.2、C++方法

（1） to_string

将数字转化为对应的字符串

主要原型包括：

string to_string (int val);
string to_string (long val);
string to_string (long long val);
string to_string (unsigned val);
string to_string (unsigned long val);
string to_string (unsigned long long val);
string to_string (float val);
string to_string (double val);
string to_string (long double val);

示例：

#include<iostream>
using namespace std;
#include<string>
int main()
{
	int a = 1234;
	string s = to_string(a);
	cout << s << endl;//输出字符串1234
	return 0;
}

（2）_itoa()

itoa()函数把整数转换成字符串

函数原型：

char* _itoa(int,char*,int radix)；将整型值转换为字符串

char *_ltoa( long value, char *string, int radix );将long值转换为字符串

char *_ultoa( unsigned long value, char *string, int radix );  将无符号长整型值转换为字符串

示例：

#include <stdlib.h>
#include <stdio.h>
int main(void)
{
    int number = -12345;
    char string[32];
    _itoa_s(number, string, 10);
    printf("integer = %d string = %s\n", number, string);
    return 0;

}

在VS2019中无法直接使用_itoa，只能使用它的安全版本_itoa_s。
