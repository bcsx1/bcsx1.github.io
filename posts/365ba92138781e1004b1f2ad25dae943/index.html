如何修改 / 校正Linux系统时间

序言

很多时候我们操作完Linux系统后会直接挂起机器，不是关闭机器，但是这样就会带来一个问题，系统时间会脱离网络实际时间，导致系统时间与时间不一致，下面就是关于修改 / 校正Linux系统时间的操作步骤

操作

  1. 使用date命令查看当前系统时间。如下所示：

    [root@hadoop102 ~]# date
2021年 05月 23日 星期日 11:16:42 CST

    但是目前实际时间是：
    2021年5月25日 20:55:07 星期二

  2. 使用命令rm -f /etc/localtime删除本地时间文件。如下所示：

    [root@hadoop102 ~]# rm -f /etc/localtime

  3. 使用命令cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime复制时区文件到localtime。如下所示：

    [root@hadoop102 ~]# cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

  4. 使用命令“yum install -y ntpdate”安装“ntpdate”。如下所示：

    [root@hadoop102 ~]# yum install -y ntpdate

  5. 使用命令“ntpdate -u ntp.api.bz”将本地时间与网络时间同步，如下所示：

    [root@hadoop102 ~]# ntpdate -u ntp.api.bz
25 May 21:02:50 ntpdate[6867]: step time server 114.118.7.161 offset 207360.947555 sec

  6. 使用命令date查看当前系统时间是否更新。如下所示：

    [root@hadoop102 ~]# date
2021年 05月 25日 星期二 21:04:07 CST

  7. 使用命令“hwclock -w”保存配置，以便重启后时间再次不对。如下所示：

    [root@hadoop102 ~]# hwclock -w
