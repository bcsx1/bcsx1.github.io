/deep/.el-cascader-panel {
  height: 400px !important;

观察发现

整个下拉区域是点击后才自动生成，样式读取时dom还没生成，故无法控制
遂采取原生获取dom改变样式的方式，用visible-change事件监听dom是否已经生成

<el-cascader v-model="value" :options="options" @visible-change="visibleChange"></el-cascader>

visibleChange(){
   const selectDropDownNode = document.getElementsByClassName('el-cascader-panel')
    for (const i = 0; i < selectDropDownNode.length; i++) {
        selectDropDownNode[i].style.maxHeight = '400px'
    }
},
