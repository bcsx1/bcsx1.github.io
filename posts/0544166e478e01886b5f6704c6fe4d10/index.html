windows自建免费无限的开源图片识别公式转换为Latex

一、准备

  1. python3.9.6下载
    在最开始勾选添加环境变量
    https://www.python.org/ftp/python/3.9.6/python-3.9.6-amd64.exe
    验证，右键终端（管理员），输入：python --version
  2. 安装Anaconda
    https://mirrors.bfsu.edu.cn/anaconda/archive/Anaconda3-5.3.1-Windows-x86_64.exe
    运行安装包，一路默认路径就ok
    配置环境变量
    ，主要添加"C:\ProgramData\Anaconda3" “C:\ProgramData\Anaconda3\Scripts”
    验证，右键终端（管理员），输入：conda --version
  3. 下载torch
    https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/torch/
    https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/torchvision/

选torch-1.13.1-cp39-cp39-win_amd64.whl

二、开始安装

1. 安装pytorch

右键终端（管理员）

conda create -n pytorch python=3.9.6
activate base
activate pytorch
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/win-64/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/win-64/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/win-64/
conda config --set show_channel_urls yes
conda config --set ssl_verify false
conda install pytorch torchvision cudatoolkit=10.1

2. 安装torch

终端转到刚刚下载的安装包路径，
输入：pip install torch-1.8.1-cp38-cp38-win_amd64.whl

3. 安装图片识别库

https://github.com/lukas-blecher/LaTeX-OCR/

pip install pix2tex[gui]

要等很久

4.下载模型

继续在刚才的终端输入：pix2tex
