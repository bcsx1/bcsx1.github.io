【2019蓝桥杯国赛B组】排列数

这道题的思路就是从 i i i个数的情况推到 i + 1 i+1 i+1个数的情况，首先设 d p [ i ] [ j ] dp[i][j] dp[i][j]： i i i个数， j j j个折点的方案数

每次插入的数值是 i + 1 i+1 i+1，必定是当前序列中最大的数，它可能使得整个序列中的折点数+0、+1或+2（下文证明）

所以我们要考虑的就是 d p [ i ] [ j ] dp[i][j] dp[i][j]对 d p [ i + 1 ] [ j ] 、 d p [ i + 1 ] [ j + 1 ] 、 d p [ i + 1 ] [ j + 2 ] dp[i+1][j]、dp[i+1][j+1]、dp[i+1][j+2] dp[i+1][j]、dp[i+1][j+1]、dp[i+1][j+2]的影响

下面具体研究状态转移方程的推导：

我们其实可以把整个序列看作被n个折点分成的n+1个单调序列

在一个单调序列中，端点有三种可能：

  • 波峰
  • 波谷
  • 边界

先只考虑将点插入序列中两点之间的情况：

情况1：一个长度为 n 的单调序列（n+1个点，n条边），有n个可以插入的位置，那么其中n-1个位置插入后是能使得折点数+2的，只有靠近波峰的那一侧，折点数不变；

情况2：如果单调序列中波谷那一侧是端点，则还是n-1个位置插入后+2，1个位置插入后折点不变

情况3：如果单调序列中波峰那一侧是端点，则将有n-1个位置插入后+2，1个位置插入后折点数+1

此外当一个单调序列的端点是边界时，还可以把点插入边界外，这样是两种情况

由上面的讨论可以得知：

d p [ i + 1 ] [ j + 2 ] + = d p [ i ] [ j ] ∗ ( i − j − 2 ) dp[i+1][j+2]+=dp[i][j]*(i-j-2) dp[i+1][j+2]+=dp[i][j]∗(i−j−2)

解释：目前总共有 i 个点，也就是i-1 条边；由于有 j 个折点，所以有 j + 1段单调序列，这 j+1 个单调序列上：

  • 假设单调序列长度为k，则必然有k-1个位置是使得折点数+2，即每个单调序列损失了一个折点+2的机会
  • 总共有i-1条边，减去j+1个非+2点，得到 i − 1 − ( j + 1 ) = i − j − 2 i-1-(j+1)=i-j-2 i−1−(j+1)=i−j−2

假设折点数有偶数个，

则要不是 波峰-波谷-波峰-波谷，要不是波谷-波峰-波谷-波峰

也就是在靠近边界的两个单调序列中，有一个满足情况3，则得到1个使折点数+1的方案；另一个满足情况2，此时将折点数插到波谷外，也可以得到使折点数+1的方案。

假设折点数有奇数个

则要不是波峰 - 波谷 -波峰，要不是 波谷 - 波峰 - 波谷

采用上面的推法，可以得知不管怎样都是两个折点数+1的方案
