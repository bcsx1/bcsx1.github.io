剑指offer-面试题04 二维数组中找出某个值

一、题目

在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

示例

[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

给定 target = 5，返回 true

给定 target = 20，返回 false

二、题解

1.暴力解法，利用二重循环，遍历二维数组的每一个数字，判断其与目标值是否相同

（1）时间复杂度：O(N2)，空间复杂度为O(1)，超时

2.根据这个二维数组的特性来解决问题，二维数组每行是递增的，每一列也是递增的，因此可以选一个特殊的起始位置来开始进行判断，这里选择的是最走下角位置的元素

3.分析这种方法的过程

（1）如果目标值大于当前数组的数字，则需要找小点的值，需要往上走，行减小（因为上边的值比下边的小，右边的值比左边的大）

（2）如果目标值小于当前数组的数字，则需要找更大的值，需要往右走，列增加（因为右边的值都比左边的值大，上边的值都比下边的值小）

（3）如果目标值等于当前数组的数字，则直接返回true

4.可以选择使用双指针来遍历数组，一个是行指针，一个是列指针，行指针递减，列指针递增

5.时空复杂度分析

（1）时间复杂度，假设有m行n列，行指针每次都是一行行的移动，列指针每次都是一列的移动，时间复杂度为O(m+n)

（2）空间复杂度为O(1)

三、代码 

bool Find(int target, vector<vector<int> > array) {
        //思路：从二维数组的最左下角开始遍历
        //一：若是目标值大于当前数组值，则要往右走，要找更大的数（因为上面的值都比下面的值小，所以只能往右走）
        //二：若是目标值小于当前数组值，则要往上走，要找更小的数（因为右边的值都比左边的大，所以只能往上走）
        int row=array.size();//代表的是行数
        int col=array[0].size();//代表列数
        int i=row-1;
        int j=0;
        while(i>=0&&j<col){
            if(target>array[i][j]) ++j;
            else if(target<array[i][j]) --i;
            else {
                return true;
            }
        }
        return false;
    }
