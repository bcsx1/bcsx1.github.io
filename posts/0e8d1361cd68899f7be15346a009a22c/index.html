前言

  Multisim是一款功能强大的电路仿真软件，可以用于设计和模拟各种类型的电路，帮助用户快速设计和验证电路。除了上述基本功能外，Multisim还提供了许多高级功能，如自动优化电路、噪声分析、温度分析等等。本文将详细介绍Multisim的基本使用方法，希望能够帮助初学者快速掌握Multisim的基本使用方法。

一、Multisim的用户界面

详细说明见下表：

序号名称功能

二、Multisim基本操作

2.1 创建文件

  打开Multisim软件后，首先要创建一个新的电路。方法是点击菜单栏的“File”->“New”，在弹出的“New Design”窗口中选择“Blank and recent” ——“Blank”，最后点击“Create”。

①工作页面放大——直接滑动鼠标滚轮
②工作页面移动——按下鼠标滚轮后拖动

2.2 放置元器件

  点击Place > Component打开Select a Component对话框。选择对应元器件，具体元器件介绍见第四小节，电路参考见附录

①按Ctrl+R键，可以变换方向
②双击元器件，可以对其参数修改

2.3 连接元件

  添加元件后，需要使用Place——Wire（Ctrl+Shift+W）工具将元件连接起来。
  或者移动光标至元件的引脚时，光标会变成十字型，表示可以开始连线了，在要连线的元件引脚单击，然后移动光标到目标元件的引脚单击，完成连线。

2.4 保存电路

  可以保存电路图和仿真结果。点击“File”->“Save”保存，将电路图文件保存为“***.ms14”。在保存时，可以选择不同的文件格式，如PDF、PNG、JPG等。仿真完成后，点击“File”->“Export Template…”保存仿真结果。

提示：为避免文件意外丢失，可设置自动保存：Options > Global options > Save > Auto-backup。

三、仿真分析

3.1 运行仿真

  点击“Run”按钮，即可开始仿真电路。在仿真结果中，可以查看电路的电压、电流等参数。如果电路中有错误，Multisim会给出相应的提示。

3.2 使用虚拟仪器仿真

  在Multisim中，还可以使用虚拟示波器、虚拟多用表等工具对电路进行分析。

  1. 点击菜单栏中的Simulate——Analyses and simulation，在弹出的对话框中选择Interactive Simulation

  2. 选择Simulate > Instruments > Oscilloscope，添加示波器。
    ①双击示波器，打开其前置面板。
    ②选择Simulation > Run。

3.3 图表分析

3.3.1 用DC Operating Point直流分析

  直流工作点分析用于测量电路的静态工作点

  1. 点击菜单栏中的Options——Sheet properties，在弹出的对话框里选择Sheet visibility选项卡，在Net names栏中选择Show all选项，最后点击OK确定
  2. 选择Simulate > Analyses and Simulation，在弹出对话框中选择DC Operating Point，点击Output选项卡
  3. 在“Variables in circuit”(电路中的变量)列表中列出了所有可选的输出变量。选中用于分析的输出变量，点击Add按钮，即可将其加入“Selected variables for analysis”(选中用于分析的变量)列表中，单击Remove按钮，可以将不需要显示的变量移回“Variables in circuit”栏中。
  4. 点击“run”运行按钮即可开始仿真， Grapher View窗口将弹出，显示仿真结果

3.3.2 用AC Sweep交流分析

  AC Sweep可以用来分析放大器的频率响应

  1. 选择Place——Probe——Voltage放置探针

    注意：当探针放到连线上是，会变成绿色，反之，是灰色。

  2. 选择Simulate > Analyses and Simulation，弹出Analyses and Simulation对话框（也可以单击Simulation工具栏的Interactive），在Active Analysis栏选择AC Sweep

  3. 点击Frequency parameters选项卡设置参数

  • Start frequency (FSTART)：设置交流分析的起始频率
  • Stop frequency (FSTOP)：设置交流分析的终止频率
  • Sweep type：设置扫描类型。有三种选择：Decade（十倍程扫描），Octave（八倍程扫描），Linear（线性扫描）。
  • Number of points (per decade 或 per octave)：设置测试点数目。
  • Vertical scale：设置纵坐标刻度。有四种选择：Linear（线性），Logarithmic(对数)，Decibel（分贝），Octave（八倍）
  1. 然后单击Run打开分析结果的图表，此时弹出Grapher View窗口

四、元器件

4.1 电阻

  在Group选择Basic，Family选择RESISTOR添加电阻

  双击图标打开配置，在Value——Resistance®可修改阻值

4.2 电容

  在Group选择Basic，Family选择CAPACITOR添加电容

  双击图标打开配置，在Value——Capacitance© 填写容值

4.3 开关

  在Group选择Basic，Family选择SWITCH，挑选对应开关

  双击图标打开设置，在Value——Key for toggle设置开关快捷键。或者对图标的开关部分单击可以进行打开闭合操作

4.4 三极管

  在Group选择Transistors，Family选择BJT_PNP，挑选对应PNP型晶体管

  双击图标后点击Value——Edit Model打开打开配置

  修改参数后，点击“Change component”或者“Change all n components”按钮可以将所选中晶体管或者全部晶体管进行批量修改

4.5 电源相关器件

  1. 直流电压源
      在Group选择Sources，Family选择POWER_SOURCES，点击DC_POWER放置

      双击图标，在Value——Voltage(V)后填写新的电压值，最后点击“OK”
  2. 交流电压源
      在Group选择Sources，Family选择SIGNAL_VOLTAGE_SOURCES，点击AC_VOLTAGE放置

      双击图标，在AC_VOLTAGE对话框里点击Value选项卡，在Voltage(Pk)后填写新的电压幅值，在Frequency(F)后填写新的频率，最后点击OK
  3. 添加地
      在Group选择Sources，Family选择POWER_SOURCES，点击GROUND添加

五、虚拟仪器

  介绍三个常用仪器万用表、函数信号发生器、示波器

5.1 万用表

  生活中很常用的工具，可以简单检测电压、电流以及电阻参数

  1. 点击菜单栏中的Simulate——Instruments——Multimeter添加万用表
  2. 双击万用表，点击Set…进行参数设置（基本不需要改动）

  3. 将万用表正负极连接在电路上，就可以查看对应属性

5.2 函数信号发生器

  函数信号发生器可以在电路中产生相应的波形

  1. 点击菜单栏中的Simulate——Instruments——Function generator添加仪器
  2. 双击信号发生器，弹出设置窗口

    ①在Waveforms选择电压输出波形，分别为正弦波、三角波和方波
    ②在Signal options设置Frequency频率、Duty cycle占空比、Amplitude幅值与 Offset输出电压的偏置值
    ③Set rise/Fall Time按钮用来设置方波的上升和下降时间，单位为秒
  3. 连接方式也很简单，通常COM口接地，然后用+、-两个中的一个连在电路另一端即可，正负极不同波形相反

5.3 示波器

  示波器功能与万用表类似，不同的是示波器可以方便显示连续波形

  1. 点击菜单栏中的Simulate——Instruments——Oscilloscope添加示波器

  2. 双击打开面板

    ①当有波形时黑色区域会显示
    ②在Timebase项中Scale，设置X轴方向每格代表的时间。Y/T按钮表示X轴显示时间刻度，Y轴显示电压信号的幅度
    ③Channel通道A、B中， Scale设置Y轴方向每格代表的电压数值，AC按钮只显示信号的交流部分，0按钮代表输入信号与地短接，DC按钮将显示信号的交、直流分量叠加后的结果
    ④T1、T2可以添加光标显示刻度

  3. 示波器连接只需要选择对应通道，-极接地，+极接待测量位置
