一、指针常量与常量指针

（一）指针常量：

辅助记忆，“指针是常量”

int val1=2;
int* const p=&val1;//	定义一个指针常量，

理解一： 指针常量即“指针是常量”，const修饰的是指针p，指针不可改变，如下例：

#include<iostream>
using namespace std;
int main(){
	int val1 = 2;
	int val2 = 3;
	int* const p = &val1;
	p = &val2;
}

vs2017下编译错误：

注: 所谓左值非严格意义上来说可以理解为能取地址的表达式（例如 a+b就是非左值，是临时变量没有地址）

理解二： 指针虽然是常量但指针所指向的内容可修改，如下例：

#include<iostream>
using namespace std;
int main(){
	int val1 = 2;
	int val2 = 3;
	int * const p = &val1;
		cout << *p << "\n";
	*p = val2;
	cout << *p << "\n";
	system("pause");//控制台暂停
}

（二）常量指针 ：

辅助记忆，“常量的指针”

    int val1 = 2;
	const int *p = &val1;//定义一个常量指针

理解一： 常量指针即常量的指针，const修饰的是指针所指向的内存，故指针p本身可以改变，如下例：

#include<iostream>
using namespace std;
int main() {
	int val1 = 2;
	int val2 = 3;
	const int *p = &val1;
	cout << *p << "\n";
	p = &val2;
	cout << *p << "\n";

	system("pause");//控制台停止
}


理解二： const修饰的是指针的内容，所以指针的内容不可改变，如下例：

#include<iostream>
using namespace std;
int main() {
	int val1 = 2;
	int val2 = 3;
	const int *p = &val1;
	cout << *p << "\n";
	*p = val2;//或者写成*p=3;

	system("pause");
}

二、非常量引用与常量引用

（一）引用的基础理论（非常量引用）：

1.引用必须与其所引用的对象进行捆绑，引用可以理解为对象的别名，如下例：

#include<iostream>
using namespace std;
int main() {
 int val1 = 2;
 int &val2 = val1;
 cout << val2 << "\n";//引用类似与设置别名

 system("pause");
}

运行结果：

2.引用可以理解为内存空间（地址）的捆绑，故引用必须初始化；

#include<iostream>
using namespace std;
int main() {
 int val1 = 2;
 int &val2 ;
 system("pause");
}

运行结果：

3.引用不但必须初始化，且必须以左值（可取地址）进行初始化，如下错例：

#include<iostream>
using namespace std;
int main() {
	int val1 = 2;
	int val2 = 3;
	int &val3=val1+val2;//或者是int &val3=3都是错误的
	system("pause");//正确做法是int &val3=val1;
}

运行结果：

4.引用的类型还必须与其所引用的对象一直。如下例子：

#include<iostream>
using namespace std;
int main() {
	int val1 = 2;
	
	double &val3=val1;
	system("pause");
}

（二）常量引用：

辅助记忆，常量的引用，即此引用不可修改
常量引用针对的是，“用的类型还必须与其所引用的对象一直”的扩展（非常量引用中第4点），即常量引用的初始值可以是任意表达式（右值或者临时变量等），但该表达式必须能够转换为引用的类型。
注：const 修饰的常量引用其实是绑定到了临时创建的变量上，而非原来的对象
1.常量上应用（极少使用）

#include<iostream>
using namespace std;
int main() {
	double val1 = 2.9;
	const int &val2=val1;
	cout << val2 << "\n";//支持double 到int 的隐式转换
	
	const int &val3 = 9.6;
	cout << val2 << "\n";//支持非左值（临时变量）
	system("pause");

运行结果：

2.多态中的应用（很重要）

#include<iostream>
using namespace std;

class IDate {
public:
	virtual void  print_date() const = 0;
};
class Date :public IDate {
public:
	int year, mouth, day;
	Date(int y=2018,int m=3,int d=20):year(y),mouth(m),day(d){}
	void print_date() const {
		cout << year << "/" << mouth << "/" << day<<"\n";
	}
};


int main() {
	IDate &id1=*new Date(2018, 9, 30);//注意IDate &id=Date(2018, 9, 30);则出错因为Date()函数返回临时变量
	id1.print_date();

    const IDate &id2 = Date(2018, 9, 30);//const类型的对象只能调用const类型的函数，所以print_date() 必须为const类型
	id2.print_date();

	system("pause");
}

运行结果：

运用到多态中，就可以实现以IDate对date类的封装（因为多态中的操作只能通过指针或者是引用实现）
