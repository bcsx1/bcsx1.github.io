一、数值型数据类型转换

数值型数据类型的转换即为数字之间数据类型的转换，与生活中的数学不同，在Java中没有数学这个统一的概念，在Java的新手学习过程中，我们将把数字进行分割并重组丢弃，最后得到有关数字的概念只有六个数据类型来支持数据的运算。

整型数据类型

        整型有byte、short、int和long这四个数据。

为什么整形要转换数据类型呢?

        因为变量是一个存储空间，他的记录非常有限，我们定义变量的意义在与给它分配一定的空间，空间的大小以数据类型来说明，由小到大依次为byte、short、int、long。

数据类型的转换需要考虑到它们的取值范围和存储空间大小，我在这里再给大家复述一遍。

        所谓的整型就是指没有小数点的数字就是整数的存在但是有取值范围，他们的变量的值都因该由0-9这十个数字和正负符号组成，其中他们的区别在于数值范围，对于新手来说，最值得注意的是byte和short的范围，因为数据的转换有时会因为范围而产生变化

        以下是数据类型转化的基本要诀

    1）小数据类型与大数据类型进行运算的时候， 会默认转换成大数据类型，

        

        long a = 1;
		int b=1;
		a=a+b;
		System.out.println(a);

其输出结果为2数据类型为long型。
    2）大数据类型接收小数据类型的时候，会默认转换成大数据类型，

short a=234;  int b=a;

如图所示，当变量b接收了a以后，b的值就变成了
    3）小数据类型接受大数据类型的时候，需要进行强制类型转换，

		long a = 1;
		int b=(int)a;


    4）数据进行强制类型转换的时可能会造成溢出

		int a=167;
		byte b=(byte)a;
		System.out.println(b);

上述代码的输出结果为 -89，由于byte的取值范围为-128与127之间，其运算的原理为：当数值达到127后运算并不会结束，加1会变为负的符号位，数值变为-128，-128加上余出来的39，最终结果为-89。

非整型数据类型

非整数数据类型有float型和double型，float类型定义的变量必须在数字末尾加上 “ f ” 标签，所以一般在运算中突然出现的不带标签的有小数的数值，通常会被默认为double型。例：

double b=a+3.5;

所以，在类型转换的基本要诀4中需要加上一条（强制类型转换也肯能会造成精度丢失）例如：

		int a=67;
		double b=3.5;
		int c=a+(int)b;
		System.out.println(c);

输出结果为70，正如要诀所预料的一样，小数点后的数据都被当成不在整数取值范围给丢弃了。

二、非数值型数据与数值型数据的关系

由于非数值型数据与数值型数据的不一样，所以一般来说它们之间的不能相互转换数据类型，

当然，有时候也有特殊情况。下面我们来讲解一下新手比较容易接触到的char类型、boolean类型和String类型

1.char字符型

正常情况下，我们不会用到char类型与数值型之间的转换，但是有时候也有要进行的转换的时候，这里我们讲解一下，char类型能进行的转化。

2.boolean布尔型

boolean布尔型定义的变量，它的值只有true和false两种，即为计算机的0和1两种状态。

3.String字符串型

严格意义上来说，字符串型的数据类型不属于八大数据类型，属于引用数据类型，所以它不能与八大数据类型之间进行数据转换，即使它的值可以为数字与数据类型很像，也可以为字符和char类型很像，但是他们之间依旧不能转换。String类型不能转换成任何类型。所以，在使用String类型定义数字的时候，要将其与数值型数据类型区分开来。同样，在进行数值运算的时候，注意不要把String类型牵扯进来。

总结

数据类型转换的几个注意点
    1)小数据类型与大数据类型进行转换的时候，默认转换成大的数据类型
    2)大的数据类型接收小的数据类型的时候，默认转换成大的数据类型
    3)小的数据类型接收大的数据类型的时候，需要强制进行转换
    4)强制类型转换可能会造成溢出的问题，会舍弃精度
