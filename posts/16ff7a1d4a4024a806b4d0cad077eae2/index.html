MySQL8.0.26安装(win7,zip,外部连接)

MySQL8.0.26zip安装(win7)

1.下载zip文件下载连接 密码：lwl0

2.解压zip到C:\Program Files，也可以选择解压到别的地方

3.解压目录下创建my.ini文件


my.ini文件内容：
由于我们使用my.ini自定义的配置文件指定，所以需要创建C:\Program Files\mysql-8.0.26-winx64\MySQL和C:\Program Files\mysql-8.0.26-winx64\MySQL\Data文件夹

[mysqld]
# 设置3306端口
port=3306
# 设置mysql的安装目录
basedir=C:\Program Files\mysql-8.0.26-winx64\MySQL
# 设置mysql数据库的数据的存放目录
datadir=C:\Program Files\mysql-8.0.26-winx64\MySQL\Data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。
max_connect_errors=10
# 服务端使用的字符集默认为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306
default-character-set=utf8mb4

4.安装和初始化

以管理员身份运行cmd,cd到bin目录下,命令：cd C:\Program Files\mysql-8.0.26-winx64\bin，
然后运行mysqld --initialize --console进行初始化，记住最有一行显示的密码,我这里密码是3:?OydxrBr<r

5.安装mysql服务

以管理员身份运行cmd,进入bin目录,运行mysqld --install（如果步骤 4.安装和初始化 的cmd没关掉，可以直接用），再服务中就能看到mysql服务了，双击MySQL服务，将其启动。

6.配置环境变量，这样我们就不需要每次都要去bin目录下进入mysql.

将路径C:\Program Files\mysql-8.0.26-winx64\bin粘贴到path变量的最后，如果path变量最后没有;,则需要加上;再进行粘贴.

7.修改密码

再cmd命令下运行mysql -uroot -p，然后输入步骤4得到的初始化密码，然后再运行ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';将密码改成123456,你可以修改成你自己的密码。可以关掉cmd再新开一个cmd,使用修改后的密码是否能进入，成功的话，mysql就安装成功了。

8.外网访问（额外功能）

8.1修改root用户的方式

  1. 登录mysql
mysql -uroot -p
  1. 使用mysql
use mysql
  1. 修改root用户的host
update user set host='%' where user ='root';
  1. 刷新
FLUSH PRIVILEGES;
  1. 授权
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;

8.2创建新用户的方式

  1. 登录mysql
mysql -uroot -p
  1. 使用mysql
use mysql
  1. 创建用户xpp密码123456
CREATE USER 'xpp'@'%' IDENTIFIED BY '123456';
  1. 刷新
FLUSH PRIVILEGES;
  1. 授权
GRANT ALL PRIVILEGES ON *.* TO 'xpp'@'%' WITH GRANT OPTION;

就可以使用用户名：xpp 密码：123456 进行访问了
