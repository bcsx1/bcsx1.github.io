如何用Eclipse 调试PHP代码

作者:PHPYuan 时间:2018-07-01 23:40:28

Eclipse是很多Java程序员喜爱的一款免费的IDE。PHP也是网络上很多程序员相关段子的最爱。

本文Jerry介绍一种最简单的使用Eclipse调试PHP代码的办法。

1. 百度上搜索服务器WAMPServer，下载安装。

服务器默认的监听端口号是80。我们把它改成其他端口号吧。

看见下面的菜单了么？选择二级菜单最下面的httpd.conf, 这个是服务器的配置文件。

选择之后，会自动打开httpd.conf这个配置文件，它位于您服务器的安装目录下面，后面的路径为：

bin\apache\apache2.4.9\conf\httpd.conf

我把默认端口号改成了8098：

2. 打开您的Eclipse，在设置里，指定php这个执行文件的位置。按照我下面的配置去做：

选择PHP->Debug, PHP Executable设置成上图配置的：

创建一个新的Debug配置，类型选择PHP Web Application，File里选择您想调试的PHP server page，我的例子是/jerry/newfile.php:

3. 第一步的默认端口号不是改成8098了么？在newfile.php里设置好断点，用这个url启动服务器， http://localhost:8098/jerry/newfile.php?XDEBUG_SESSION_START=ECLIPSE_DBGP&KEY=14572383566191
