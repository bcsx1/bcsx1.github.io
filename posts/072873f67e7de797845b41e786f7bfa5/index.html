windows电脑快捷键大全 - 高手总是很酷的

1.    Windows+L键，直接锁屏，这样就不用担心电脑的资料外泄了

    

2.    一般人会先找到“我的电脑”，然后点击打开，而高手总是很酷的， Windows+E键，直接打开电脑的资源管理器

3. 直接按下 Windows+D键，桌面闪现！Win+D:　这是高手最常用的第一快捷组合键。这个快捷键组合可以瞬间回到桌面，其他所有窗口瞬间最小化，无论是聊天的窗口还是游戏的窗口；只要再次按下这个Win+D组合键，刚才的所有窗口都回来了，而且激活的也正是你最小化之前在使用的窗口！

4. 按下 Windows+Tab键，3D效果显示切换窗口

5.Windows自带的录像功能。按下 Windows 键+ R，输入 psr.exe 回车，然后就可以开始记录了。

6. Windows键+ X， 一次性调整亮度、音量大小，打开无线网，电池电量

 

7.Windows + R输入osk，出现炫酷虚拟键盘！

8.试试 Windows 键和 +++++++++，放大镜出现！

9.像很多浏览器都是在同一窗口里打开多个页面。用 Ctrl+Tab，可以在几个页面之间互相切换。用Ctrl+w，可以关闭当前的页面。


10. Windows+ Fn+ Home，打开系统属性。用键盘操作更帅嘛~^^

11.Shift + 任务栏的窗口图标   =  新建一个该窗口；
