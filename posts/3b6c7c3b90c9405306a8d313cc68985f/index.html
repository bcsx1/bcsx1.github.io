一、前言

字典是包含若干“键：值”的无序可改变序列，表示一种映射关系，也称作关联数组，有些类似于c/c++中的结构体数组，具体特点如下：

（1）字典的键通常采用字符串，但也可以用数字、元组等不变的类型。 （2）值可以是任意类型。
（3）它通过键来索引映射的值，而不是通过位置来索引。

二、字符基本操作

1.创建字典

可以用多种方法创建字典，例如：


{} 或者 dict()                    # 创建一个空字典

a = {1:'song',2:"niu",3:"b"}     # 数字当做键

b = {"as":"sa","s":"asd"}        # 字符串作为键

c = {(1,2,3):5,(1,5):99}         # 元组作为键

d = {1:{'a':1,'d':"adf"},2:"sd"} # 嵌套字典

e = dict(name='as',sex='man')    # 赋值法创建字典

f = dict([('a',2),('d',3)])      # 用包含键元组和值元组的列表

g = dict.fromkeys(['a',1])       # 创建无映射值的字典，默认值为None
#         结果：{‘a':None,1:None}

h = dict.fromkeys('ABC',20)# 使用字符串和映射值的字典
#         结果：{'A':20,‘B':20,'C':20}

2.关系判断

关系判断可以判断字典中是否含有这个键，例如：


dic = {'a': 1, 'b': 2}

print('a' in dic)

# Ture

print(1 in dic)

# Flase

3.索引

字典通过键来索引映射的值，例如：


dic = {'a': 1, 'b': 2}

print(dic['a'])  # 打印键’a'的值，结果：1

dic['a'] = 3     # 通过索引修改键‘a'的值

print(dic)       # 打印修改后的值，结果：{'a': 3, 'b': 2}

dic['c'] = 3     # 试图修改不存在的键时，添加一个新的键值对

print(dic)       # 结果：{'a': 3, 'b': 2, 'c': 3}

三、字典常用方法

1.clear()

删除全部字典对象,例如：

dic = {'a': 1, 'b': 2}

dic.clear()

print(dic)  # 结果：{}

2.copy()

复制字典对象，正常来说直接赋值时，会出现以下情况：

dic = {'a': 1, 'b': 2}

dic1 = dic         # 直接赋值时，使用的是同一个对象

print(dic1 is dic) # 判断是否是同一个对象，结果：True

dic1['a'] = 2      # 通过dic1可以改变dic说明这两个指向的是同一个字典

print(dic)         # 结果：{'a': 2, 'b': 2}

但如果我们使用copy()方法，则会：

dic = {'a': 1, 'b': 2}

dic1 = dic.copy()

print(dic1 is dic) # 再判断是否是同一个对象，结果：False

dic1['a'] = 2      # 通过dic1无法改变dic说明这两个是两个独立的个体

print(dic)         # 结果：{'a': 1, 'b': 2}

3.get(key[default])

在编写这个函数时，我们需要先对键值进行判断。如果这个键在字典中存在，我们直接返回对应的值即可。如果不存在，并且没有规定默认值，则会返回空，若规定则返回默认值。例如：

dic = {'a': 1, 'b': 2}

print(dic.get('a'))    # 返回对应值：1

print(dic.get('c'))    # key不存在时返回空None

print(dic.get('c', 5)) # 当key不存在时，并且设置默认值时返回默认值5

4.pop(key[default])

pop方法可以删除键，并返回映射值，若不存在返回default,若未指定default则报错，例如：

dic = {'a': 1, 'b': 2}

print(dic.pop('a'))      # 删除键为'a'的键值对,并返回其值，结果：1

print(dic)        # 结果：{'b': 2}

print(dic.pop('c', 6))   # 结果：6，不设置默认值会报错

5.popitem()

popitem也是删除键值对，和pop不同的的是：它并不能指定位置，只会从最后开始删除，并且返回一个它删除键值对的元组，当目标为空字典时报错。


dic = {'a': 1, 'b': 2,'c':3}

print(dic.popitem())   # 结果：('c', 3)

print(dic)             # 结果：{'a': 1, 'b': 2}

6.setdefault(key[default])

setdefault(key[default])函数的作用是在字典中设置一个默认值，如果字典中已经存在指定的key，则不进行任何操作，如果字典中不存在指定的key，则将其添加到字典中，并设置默认值，不设置默认值时为None。

dic = {'a': 1, 'b': 2}

print(dic.setdefault('a'))   # 返回映射值1

dic.setdefault('c', 3)       # 不存在时，添加新的键值对；

print(dic)                   # 结果：{'a': 1, 'b': 2, 'c': 3}

7.update(other)

uupdate(other)方法用于为字典添加键值对，如果已存在同名的键，则映射值会被覆盖。例如：

dic = {'a': 1, 'b': 2}

dic.update({'a': 0, 'c': 3})   # 同时覆盖’a',添加'b'

print(dic)                     # 结果：{'a': 0, 'b': 2, 'c': 3}

比setdefault便捷的是他可以在一个语句中进行多次操作。

四、字典视图

字典中items(),keys(),values()方法用于返回字典键值对的视图对象，视图对象不是列表，不支持索引。通常将视图对象转化为列表。

1.items()

返回键值对的视图，例如：

dic = {'a': 1, 'b': 2}

print(dic.items())     # 结果：dict_items([('a', 1), ('b', 2)])

dic['a'] = 2           # 修改dic的值

print(dic.items())     # 结果：dict_items([('a', 2), ('b', 2)])

从结果可以看出视图反映了字典的修改内容

2.keys()

返回所有键的视图，例如：

dic = {'a': 1, 'b': 2}

print(dic.keys())        # 结果：dict_keys(['a', 'b'])

for i in dic.keys():     # 迭代打印键

    print(i)             # 结果：a  b

3.values()

返回键的对应的值，例如：

dic = {'a': 1, 'b': 2}

print(dic.values())        # 结果：dict_values([1, 2])

4.键视图的集合操作

键视图支持各种集合运算，键值对视图和值视图不支持集合运算，例如：

dic = {'c':3,'d':4}
kdic = dic.keys()

dic1 = {'d':5,'e':6}
kdic1 = dic1.keys()

print(kdic - kdic1)  # 求差集，结果：{'c'}

print(kdic|kdic1)    # 求并集，结果：{'e', 'd', 'c'}

print(kdic&kdic1)    # 求并集，结果：{'d'}

print(kdic^kdic1)    # 求对称差集，结果：{'e', 'c'}
