def review_to_words(raw_review):
    #这个函数的功能就是将原始的数据经过预处理变成一系列的词。
    #输入是原始的数据（一条电影评论）。
    #输出是一系列的词（经过预处理的评论）
    review_text = BeautifulSoup(raw_review,'lxml').get_text()
    letters_only = re.sub('[^a-zA-Z]',' ',review_text)
    words=letters_only.lower().split()
    stops=set(stopwords.words('english'))
    meaningful_words = [w for w in words if not w in stops]
    return (" ".join(meaningful_words))

现在我们需要对评论数据做一下处理，如何将文本转化为机器学习中的数字表示，一种常见的方法叫做词袋，词汇袋是从所有的文档中学习词汇，然后通过计算每个单词出现的次数来对每个文档建模，例如：
Sentence 1: “The cat sat on the hat”
Sentence 2: “The dog ate the cat and the hat”
从两句话来看，我们的汇总词汇为：
{ the, cat, sat, on, hat, dog, ate, and }
那么句子表示为：Sentence 1: { 2, 1, 1, 1, 1, 0, 0, 0 }，2表示’the’在句1中出现2次，1表示’cat’,’sat’,’on’,’hat’在句1中均出现1次，0表示’dog’,’ate’,’and’在句1中没有出现。
同理Sentence 2 are: { 3, 1, 0, 0, 1, 1, 1, 1}
