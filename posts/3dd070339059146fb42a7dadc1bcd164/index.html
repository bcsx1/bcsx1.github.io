2.1、USB Type-A / B /C接口有什么特点?

  • USB Tyoe-C的特点：
    • 传输速度快，最大数据传输速度达到10Gbpa尺寸下，
    • 接口插座端纤薄，仅为标准USB Type-A接口尺寸的1/3
    • 支持正反插，接口正反面相同，支持从正反两面均可插入的“正反插”功能，
    • 可以实现快速充电和双向充电，配备Type-C接口的标准规则连接线可以通过3A电流，可以自由的改变电力传输的方向，从而实现快速充电和双向充电

	#include<stdio.h>
	#include<math.h>
	float length(floatx1,floatx2,floaty1,floaty2);
	int main()
	{
		float a,b,c,p,S;
		int Xa=1,Ya=1,Xb=4,Yb=7,Xc=-2,Yc=5;
		a=length(Xb,Yb,Xc,Yc);
		b=length(Xa,Ya,Xc,Yc);
		c=length(Xa,Ya,Xb,Yb);
		p=(a+b+c)/2;
		S=sqrt(p*(p-a)*(p-b)*(p-c));
		printf(“三角形的面积是：%f”,S);
		return 0;
	}

10.2.1 给定两个正数 m=630 n=675 利用辗转相除法，求最小公倍数

两个整数最小公倍数 = 两整数的乘积 / 最大公约数
求最大公约数的辗转相除法：
有两个正数m和n

  • n / m 的余数 c
  • 若c = 0，则m为两数的最大公约数
  • 若c != 0 ,则n=m，m=c ，在回去执行上述步骤

求630 和 675 的最大公约数的过程

  • 675 / 630 ,余数是45
  • 630 / 45 ,余数是0
  • 则最大公约数是45
  • 最小公倍数 = 两个整数的乘积 / 最大公约数，
  • 即最小公倍数 = 640 X 675 / 45 = 9450
