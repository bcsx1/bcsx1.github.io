Vscode 常见插件

  1. Chinese (Simplified) Language Pack for Visual Studio Code — 转换中文版本
    注：下载完成后需要关闭重新启动应用，才会转换为中文版

  2. Atuo Rename Tag — 修改 html 标签，自动帮你完成头部和尾部闭合标签的同步修改

  3. beautify — 格式化代码工具，美化javascript，JSON，CSS，Sass，和HTML在Visual Studio代码

  4. Code Spell Checker — 代码拼写检查器

  5. vscode-icons — 显示Visual Studio代码的图标，目前该插件已被vscode内部支持：“文件” -> “首选项” -> “文件图标主题”

  6. guides — 辅助线对齐

  7. Rainbow Brackets — 为圆括号，方括号和大括号提供彩虹色。这对于Lisp或Clojure程序员，当然还有JavaScript和其他程序员特别有用

  8. Path Intellisense — 智能路径提示

  9. HTML Snippets — 超级实用且初级的 H5代码片段以及提示

  10. ESlint — ESlint 接管原生 js 提示，可以自定制提示规则

  11. HTMLHint — html代码检测

  12. filesize — 在底部状态栏显示当前文件大小，点击后还可以看到详细创建、修改时间

  13. vetur — Vue 语法插件， 语法高亮、智能感知、Emmet等

  14. Quokka — Quokka 是一个调试工具插件，能够根据你正在编写的代码提供实时反馈。它易于配置，并能够预览变量的函数和计算值结果。另外，在使用 JSX 或 TypeScript 项目中，它能够开箱即用

  15. Image Preview — 预览图片

  16. Markdown Preview Enhanced — 在 VSCode 里编写 Markdown，支持预览

  17. Prettier - Code formatter — 自动格式化代码

  18. Regex Previewer — 预览正则表达式

  19. CSS Peek — 查看 css 定义

  20. VSC Netease Music — 网易云音乐（程序员版）

  21. daily anime — 追番神器

  22. Browser preview — 在VScode 中打开浏览器（修改后不能实施更新，需要手动刷新）
