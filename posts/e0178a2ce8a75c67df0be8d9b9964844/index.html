安装Anaconda,新建Anaconda虚拟环境，用pycharm配置新建的Anaconda虚拟环境。

一、安装Anaconda

 本人由于某种原因，重装了不下十遍Anaconda。试过多种版本，最终收获了一些经验，在这里分享。

安装好之后又使用命令更新到23.3.1的版本，截至2023年5月1日，Anaconda最新版本为23.3.1


       随着人工智能的发展，Python成为了最受欢迎的编程语言之一。Anaconda是一个Python环境管理器，它提供了一个易于使用的界面来安装、配置和管理Python环境，包括安装Python解释器、第三方库和工具。

本文将向您介绍如何安装Anaconda、新建Anaconda虚拟环境并使用PyCharm配置新建的Anaconda虚拟环境。

二、新建Anaconda虚拟环境

1、安装完成后，我们将创建一个新的Anaconda虚拟环境，在打开Anaconda Prompt终端中输入以下命令来创建一个名为myenv的新环境：(名字可以自己任意命名）

 命令：

conda create --name myenv

还可以创建一个3.7.9的python版本，例如：

conda create --name myenv python=3.7.9

2、运行此命令后，Anaconda会提示您是否继续。输入y并按回车键。Anaconda将自动下载并安装新的Python解释器以及其他必需的库和工具。

3、环境创建完成后，您可以使用以下命令来激活环境：在此环境下，您可以安装所需的Python库和工具。

conda activate myenv


三、用PyCharm配置新建的Anaconda虚拟环境

1、现在我们已经创建了一个新的Anaconda虚拟环境，下面我们将看看如何在PyCharm中配置它：
2、打开PyCharm并创建一个新项目。
3、在项目窗口的左下角，单击设置（设置按钮通常是一个齿轮形状的图标）。
4、在设置窗口中，展开“Project: [your project name]”选项，并单击“Project Interpreter”。
5、在右上角的下拉菜单中，选择“Add...”选项。
6、在“Add Python Interpreter”对话框中，选择“Conda Environment”，然后选择“Existing environment”。
7、在“Interpreter”字段中，输入您的Anaconda虚拟环境的路径。在Windows上，它通常位于“C:\Users[username]\Anaconda3\envs[environment name]”。
8、单击“OK”按钮，并等待一段时间，直到PyCharm成功连接到您的Anaconda虚拟环境。
9、现在，您已经成功配置了PyCharm以使用您的Anaconda虚拟环境。您可以在PyCharm中

10、Python文件，并使用您的Anaconda虚拟环境中的库和工具进行开发。
11、总结
         通过本文，您已经了解了如何安装Anaconda、创建一个新的Anaconda虚拟环境以及如何在PyCharm中配置它。使用Anaconda虚拟环境可以使您更轻松地管理和使用Python库和工具，并且可以使您的项目更加可靠和可移植。希望这篇文章能够帮助您更好地使用Python和Anaconda。
12、注：本文中的示例代码适用于Windows和macOS操作系统。在其他操作系统中，可能会有所不同。
