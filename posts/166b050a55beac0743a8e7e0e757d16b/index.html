FTP服务器及其使用;FTP(file transfer protocol) 文件传送协议。;FTP服务器:;;怎样实现FTP：;在FTP的使用当中，用户经常遇到两个概念："下载"(Download)和"上载"(Upload)。"下载"文件就是从远程主机拷贝文件至自己的计算机上；"上载"文件就是将文件从自己的计算机中拷贝至远程主机上。用Internet语言来说，用户可通过客户机程序向(从)远程主机上载(下载)文件。 ;实现方式：;在地址栏中输入;;双击“WDR”的文件夹;;;;双击桌面上的CUTEFTP PRO V8.0图标;;双击桌面上的CUTEFTP图标;;单击[连接]图标;单击[快速连接]图标;只填入主机地址,不填用户名和密码则进入匿名登录方式(只能下载不能上传文件);另外一种匿名登录方式(只能下载不能上传文件);匿名登录方式下见到的状态(只能下载不能上传文件);匿名登录：能对所有文件夹进行浏览，并能下载所有文件。不能进行写操作。;单击[连接]图标;单击[快速连接]图标;填入主机地址、用户名和密码则进入用户登录方式连接(即能下载又能上传文件);标准用户登录：只能对有权限的文件夹进行浏览，能进行读、写操作。;选择[新建文件夹];新创建的文件夹;上传文件完成;小结：;方法一:利用CuteFTP工具软件进行修改密码：???????用有权用户名和密码进行登陆,然后单击菜单栏中的“命令”项的“自定义命令”中的“定义”。然后在弹出的对话框中单击“添加”按钮,在“标签”处输入命令的名称:PASSWORD, 接着在“命令文本”中输入以下代码:site pswd %[oldpass] %[newpass]。完成后,按“关闭”按钮退出命令编辑窗口。编辑好命令后,我们就可以执行这个命令了。操作步骤如下:单击"命令"下的"自定义命令"中的"PASSWORD"命令,然后在弹出的对话框中分别输入你的旧密码和新密码,这样就可以完成密码的设置.???????;方法二:DOS下进行????? 假设您的FTP地址是：1用户名是：love 原密码是：abcd 新密码是：1234请按如下操作：1.开始——运行——输入 CMD 回车(打开MS-DOS方式)2.输入：ftp 1 回车屏幕提示：Connected to 1.220 welcome joinline sever …User (1:(none)): 3.输入???户名：love　回车屏幕提示：331 User name okay, need password.Password: 4.输入密码：abcd　回车屏幕提示：230 User logged in, proceed.5.输入：quote “site pswd abcd 1234”　回车注：这里　abcd　为原密码　1234 为新密码，不要忘记输引号和空格屏幕提示：230 Password changed okay. 6.输入：bye屏幕提示：221 Goodbye!
