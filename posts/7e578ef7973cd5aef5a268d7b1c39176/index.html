python：变量

变量

  • 变量即是可以变化的量，可以随时进行修改。
  • 程序就是用来处理数据的，而变量就是用来存储数据的。

变量是一个标签，而不是盒子

在python中，变量就是变量，它没有类型，我们所说的类型是指变量所指的内存中对象的类型

>>> a = [1, 2, 3]
>>> b = a
>>> a.append(4)
>>> b
[1, 2, 3, 4]


因为变量只不过是标注，所以无法阻止为对象贴上多个标注。贴的多个标注，就是别名。

理解变量

Python 中的一个变量表示一个实体，其值可以根据需要进行更改。从概念上讲，它是一个存储实际值的内存位置。我们可以通过查询实体从代码中检索值。

但它需要为该内存位置指定一个标签，以便我们可以引用它。在编程术语中，我们称之为变量。以下是关于Python变量的一些关键事实，这些将帮助程序员有效地使用它们。

  • 变量不需要声明。但是，在使用前必须对其进行初始化
  • test = 10 表达将导致以下行为：
    • 创建一个对象来表示值10
    • 如果变量（test）不存在，则会创建它
    • 变量与对象的关联，以便它可以引用值
    • 变量“test”是对值“10”的引用。请参考下图图 1
# 图1
| ~~~~~ | ----- ~~~~~~~~~ -------   ****
( test  ) ----- Reference ------- ** 10 **
| ~~~~~ | ----- ~~~~~~~~~ -------   ****
 Variable ----- ~~~~~~~~~~ -------  Object
  • 每当表达式更改时，Python 都会将一个新对象（一块内存）关联到变量以引用该值。而旧的则被送到垃圾收集器。参考以下代码 2
# 代码 2
>>> test = 10
>>> id(test)
1716585200
>>> test = 11
>>> id(test)
1716585232

  • 此外，为了性能优化，Python 构建了一个缓存，并重用了一些不可变的对象，比如小整数和字符串。

  • 一个对象只是一个存储以下内容的内存区域：

    • 对象的实际值
    • 反映对象类型的类型指示符
    • 确定何时可以回收对象的引用计数器

  • 有类型的是对象，而不是变量。但是，变量可以在需要时保存不同类型的对象。见代码 3

# 代码 3
test = 10
type(test)
# <class 'int'>
test = 'techbeamers'
type(test)
# <class 'str'>
test = {'Python', 'C', 'C++'}
type(test)
# <class 'set'>

命名规则

在Python中，变量命名需要遵循以下这些必须遵守硬性规则和强烈建议遵守的非硬性规则。

硬性规则：

  • 变量名由字母（广义的Unicode字符，不包括特殊字符）、数字和下划线构成，数字不能开头。
  • 大小写敏感（大写的a和小写的A是两个不同的变量）。
  • 不要跟关键字（有特殊含义的单词，后面会讲到）和系统保留字（如函数、模块等的名字）冲突。

PEP 8要求：

  • 用小写字母拼写，多个单词用下划线连接。
  • 受保护的实例属性用单个下划线开头。
  • 私有的实例属性用两个下划线开头。

是否是一个合法的变量名可以用以下方法判断：

'某A'.isidentifier()
# True

下划线

特殊地，_ 也是一个变量名，如果你不定义它，它一直会取上一个执行完的值，它会指向你最后一次执行的表达式。也可用来来表示某个变量是临时的或无关紧要的。Python 会自动将解释器中最后一个表达式的值存储到一个名为 _ 的特定变量中，如果需要，还可以将这些值指定给另一个变量。

a = 8
_ # 8

c = _
c
# 8

比如放在循环中，当你懒得定义变量名时可以用它。可以把它理解成变量的变量名。

# 打印 10 次
>>> for _ in range(10):
...    print('Hello, World.')

可以在拆包中占位，但不使用这个占位上的数据：

>>> car = ('red', 'auto', 12, 3812.4)
>>> color, _, _, mileage = car

>>> color
'red'
>>> mileage
3812.4
>>> _
12

可以认为它是一个一次性的变量，只用一次，仅用于临时数据的传递，以后不会再用。

标识符

Python标识符是用户定义的名称，用于表示变量、函数、类、模块或任何其他对象。如果在 Python 中为一个可编程实体指定了某个名称，那么它在技术上只是一个标识符。在 Python 中，标识符是一个最大和最正规的概念，标识符号包括变量的名称、函数名、类名、方法名、包名等等，它们的命名规则都是一样的。

  • 变量的名称
  • 函数名，常用 func 随便命名
  • 类名
  • 方法名
  • 模块名和包名

规则与原则：

  • 要形成标识符，请使用小写字母（a到z）或大写字母（a到z）的序列。但是，在编写标识符时，也可以将数字（0到9）或下划线（x）混在一起。
  • 标识符名称不能以数字开头。这会导致语法错误。
  • 关键字是保留的，因此不应将其用作标识符。
  • Python标识符的格式中也不能有特殊字符 [‘.’、‘！’、‘@’、‘#’、‘$’、‘%’]，这些符号是禁止的。
  • 虽然官方文档说可以有一个长度不受限制的标识符，但 PEP8 规范要求一行不能超过 79 个字符。

最佳实践：

  • 最好让类名以大写字母开头（如 CamelCase）。所有其他标识符应以小写字母开头。
  • 通过使用下划线 _ 作为其第一个字符来声明私有标识符。
  • 不要在标识符中使用 _ 同时作为前导和尾随字符。因为 Python 内置类型已经使用了这种表示法。
  • 避免使用只有一个字符的名称，取而代之的是，用有意义的名字
  • 可以使用下划线将多个单词组合成一个合理的名称
  • 常量全大写
  • 学习 PEP8 规范

运算符

赋值运算符

等号（=）用来给变量赋值。

  • 等号（=）运算符左边是一个变量名,等号（=）运算符右边是存储在变量中的值

变量不需要事先声明就直接赋值，Python 会自动创建变量，那赋值过程会很简单高效。

# 单个变量赋值
nvshen = 'lily' # 自动创建变量并赋值
nvshen = 'lucy' # 此时变量的值已经发生了变化, 可打印出来看看

counter = 100          # 整型变量
miles   = 1000.0       # 浮点型变量
name    = "runoob"     # 字符串

我们可以用表达式进行赋值：

birth_year = 1996
age = 2020 - birth_year # 此时会将计算结果进行赋值
age = age + 1 # 同上
age += 1 # 上一行的简写，也有 -= 1
this_year = birth_year + age # 变量计算后将结果赋值

批量赋值：

# 同时为多个变量赋值(使用等号连接)
a = b = c = 'hi'

# 多个变量赋值(使用逗号分隔)
lily_age, lucy_age, tom_age = (18, 19, 20) # 可以打印出来看看三个变量的结果
a, b, c = 1, 2, "runoob"
a, b, c = ['hi']*3 # 三个同样的值

运算赋值，可以通过布尔运算或者逻辑运算进行赋值:

# 布尔运算赋值
x = a or b # 哪个为真就就将值赋值到 x
x = a and b # 哪个为假就将值赋到 x

# 条件循环计算赋值
x = a if a > b else b # 如果 a>b 那知 x = a, 否则 x = b
x = [i for i in a] # 将可迭代的 a 生成一个列表赋值给 x

每个变量使用前必须赋值，变量赋值之后该变量才会被创建

class Gizmo:
    def __init__(self):
        print('Gizmo id: %d' % id(self))

print(dir())
"""
['Gizmo', '__annotations__', '__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__']
"""


x = Gizmo()
"""
Gizmo id: 2801912690912, 是创建 Gizmo 实例的副作用。
"""


print(dir())
"""
和上面的dir()对比，可以看到它多了一个x标签
['Gizmo', '__annotations__', '__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__', 'x']
"""

一个变量可以通过赋值指向不同类型的对象。

海象运算符

海象运算符 :=（因为它很像海象的眼睛和长牙）是 python 3.8 开始支持的新方法。它用于在表达式内部赋值避免计算代码重复写几次，例如：

n = '9'
if long:=int(n) < 10:
    print(long+1)
# 10
long
# 9

变量类型

在 Python 里为了应对不同的业务需求，也把数据分为不同的类型。

Python3 中有六个标准的数据类型：

  • Number（数字）
  • String（字符串）
  • List（列表）
  • Tuple（元组）
  • Dictionary（字典）
  • Set（集合）

Python3 的六个标准数据类型中：

  • 不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）；
  • 可变数据（3 个）：List（列表）、（字典）、Set（集合）。
