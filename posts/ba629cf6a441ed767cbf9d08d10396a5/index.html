PR软件“基于文本的编辑”中的填充词检测与批量删除功能

“基于文本的编辑”可让您检测“呃”和“嗯”填充词并批量删除它们，从而使您的转录文本更加准确。就像处理停顿一样，您可以单击填充词并将其从序列转录文本中删除。填充词与语言无关，因此可以使用语音到文本中支持的所有 18 种语言来处理它们。

注意：您可以通过单击文本面板右上角的三个点，然后选择转录文本视图选项，来调整 Premiere Pro 可以检测到的最短停顿长度。

查找并批量删除文本、填充词或停顿，以提高转录文本的准确度。👆

重新设计的 MOGRT 管理功能
动态图形模板 (MOGRT) 为使用 Premiere Pro 开展工作的编辑人员带来 After Effects 动态图形的强大功能，这些图形被打包为模板，同时配有可在 Premiere Pro 中自定义的简单易用的控件。

根据用户的反馈，Adobe官方改进了 MOGRT 的管理界面，使其变得更加直观。更轻松地跨多个位置（包括通过磁盘上的文件夹）查找、查看和组织 MOGRT 集合。

跨多个位置组织您的 MOGRT 集合。👆

现在将 MOGRT 显示为带有复选标记的单独行，而不是在下拉菜单中显示，从而可以更直接地浏览不同位置的集合。

现在有两个视图可用于浏览 MOGRT。浏览器树视图可显示 MOGRT 的位置。下面的 MOGRT 视图展示了选定位置中的所有缩略图模板。您可以通过上下拖动窗口来完全控制每个视图占用的空间，或单击顶部的漏斗按钮以关闭和打开浏览器树视图。

新序列预设

使用适用于 HD、UHD、HDR 和社交媒体项目的新序列预设，快速创建序列。序列预设也经过重新组织和简化，让最常用的选项触手可及。
要查看新预设，请导航到文件 > 新建序列或单击项目面板上的新建项 > 新建序列按钮。现在提供了一个简化的预设列表，其中包括：

HD 1080p

社交媒体 - 4x5、9x16 和 1x1 时间轴

UHD (HDR) - 具有 HDR 色彩空间的 2160p 时间轴

UHD (4K) - 具有 SDR 色彩空间的 2160p 时间轴

旧版 - 以前包含的所有预设

浏览适用于 HD、UHD、HDR 和社交媒体项目的新预设列表。👆

这些预设旨在帮助您快速开始工作，您也可以编辑、删除或创建新的预设以简化自己的工作流程。

提示：借助项目模板，您可以使用现成的序列、素材箱、资源和其他设置创建项目，从而进一步提高效率。

更轻松地访问团队项目版本历史记录和自动保存版本
现在可以通过选择 Premiere Pro 标题栏中的团队项目名称来访问团队项目版本和自动保存历史记录。新的标题栏菜单包含以下选项：

团队项目设置

编辑团队项目名称、描述、协作者和其他设置，如颜色、暂存盘和收录设置。

版本历史记录

打开 Premiere Pro 媒体浏览器，在保存的版本列表中选择您正在处理的团队项目版本。

自动保存历史记录

打开 Premiere Pro 媒体浏览器，选择团队项目的最新自动保存版本。您可以从自动保存版本创建新的团队项目，也可以将任何版本设置为最新版本。

直接从 Premiere Pro 标题栏查看团队项目设置、版本历史记录和自动保存历史记录。👆

修剪和多机位改进
官方听取了有关功能和系列内容工作流程的反馈，相应地逐步对编辑和修剪功能做出改进。

现在，在基于多机位剪辑创建新序列时，新序列将具有可预测性更高的音频映射设置。

一个称为波纹修剪添加编辑内容以使修剪的两侧保持同步的新首选项已添加到修剪首选项，可在移动波纹修剪期间与修剪点重叠的剪辑下找到。

启用后，Premiere Pro 会将编辑内容添加到与修剪重叠的剪辑。然后，这些添加的编辑点将与正在修剪的选定编辑点一起修剪，以防止任何剪辑在编辑内容的两侧出现错位。

启用此首选项可将编辑内容添加到与修剪重叠的剪辑中。👆

浏览云位置
从“导入”模式或“媒体浏览器”直观地导入媒体，就像从本地磁盘导入一样。
存储在链接的 Dropbox 或 OneDrive 文件夹中的文件或媒体的任何更新都将开始反映在媒体浏览器面板中。在“导入”模式下，当前连接到系统的位置会自动列在云下的右列中。

关联的 One Drive 存储位置中的文件在更新时都会同步，并会开始反映在 Premiere Pro 媒体浏览器中。👆

其他更新
新增了对导出 16 位 PNG 文件的支持。

导出模式下的目标分组允许您切换各个部分以隐藏您从不使用的内容，使 UI 的布局更加整洁。

Creative Cloud 同步文件即将停用。自 2024 年 2 月 1 日起，对于和 Creative Cloud 企业版或 Creative Cloud 团队版商业帐户无关联的免费和付费个人用户，Adobe 将停止为他们提供 Creative Cloud 同步文件服务。

Creative Cloud 产品中的同步设置功能即将停用。您可以将自定预设集合从 Premiere Pro 的早期版本迁移到同一台计算机上的其他版本。

更新了系统要求，从 Premiere Pro 24.0 开始，Adobe 视频和音频产品不再支持 Rosetta 仿真（在 Apple M1/M2 上运行的 Intel 版本）。对于 x86 AMD 和 Intel 硬件，Premiere Pro 现在要求 CPU 支持 AVX2。

24.1 更新中修复的问题

音频

已修复：

时间轴中的音频回放可能会短暂出现音量较大的“嗡嗡声”。

重新打开基本声音面板后，清除音频类型按钮不可见。

在自动生成波形期间，未优先处理序列中的剪辑。

重新打开项目后，放置在效果面板中自定义素材箱中的平衡、静音和音量音频效果会消失。

开启本机导入序列时，Premiere Pro 序列中的第三方音频效果无法在 Media Encoder 中渲染。

创建新序列时，会忽略分配给序列的剪辑通道映射。

打开包含未安装的音频增效工具（VST、VST3、AU）的项目时，可能会发生崩溃。

对序列中的音频文件应用重新混合时，修剪编辑视图可能无法可靠打开。

将剪辑的速度/持续时间更改为反向回放或启用保持音频音调后，无法显示渲染进度。

颜色

已修复：

对于某些 Sony 媒体文件，log 视频色彩空间的自动检测可能会失败。

字幕和转录

已修复：

SCC 字幕文件不会在 12 小时标记后导入字幕。

通过回车键编辑字幕段可能会错误地打开序列中的第一个字幕，而不是当前时间指示器（播放指示器）旁边的字幕。

文本面板中的转录文本可能会被替换为错误文本。

导出转录文本时，不会在保存对话框中包含项目的名称。

编辑

已修复：

无法从锁定序列创建自动重构序列。

在自由视图中显示大量多机位剪辑可能会导致应用程序速度变慢。

启用重复帧标记可能会降低时间轴性能。

效果

已修复：

在效果控件面板中选择第三方过滤器进行直接操作时可能会发生崩溃。

导出

已修复：

将导出作业直接发送到 Media Encoder 时，源范围设置不正确。

格式

已修复：

某些 Sony XAVC 媒体的音频和视频可能会有几帧不同步。

注意：在早先版本中生成的代理应使用此版本重新生成，因为在高分辨率和代理媒体之间切换时，您可能会看到同步问题。

在搭载第 14 代 Intel Core 处理器的计算机上处理 H.264 和 HEVC 媒体时，可能会发生崩溃。

启动后，可能会出现消息“lvcod64.dll 中发生低级异常 (GetAVICodecs)”。

宽度为 1440 或 960 的 DNxHR 媒体在播放时仅显示黑色帧，并且会在事件面板中显示帧替换错误。

从 Apple Silicon 计算机导出的 HEVC 或 H.265 文件可以流畅播放，但在拖动时性能较差。

团队项目

已修复：

项目模板中的某些序列具有错误的导出位置。

打开团队项目时，时间轴标记不正确。

将大型项目转换为团队项目时，应用程序会短暂挂起。

在团队项目中执行另存为时，会显示独立项目的另存为对话框。

字幕和图形

已修复：

在 Mac 和 Windows 之间移动项目后，动态图形模板 .aegraphic 文件处于脱机状态。

在对缩放、位置、旋转和锚点等变换进行更改时，Premiere Pro 图形的定界框更新速度很慢。

对于 RTL 文本，使用 Tab 键将现有文本推到右侧。

传输

已修复：

启用传输监视器叠加使得无法在回放期间显示视频。

工作区

已修复：

调整监视器面板大小时，源和节目监视器按钮栏可能会闪烁。
