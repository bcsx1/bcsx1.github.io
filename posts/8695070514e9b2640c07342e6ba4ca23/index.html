旋转X轴标签柱状图

from pyecharts import options as opts
from pyecharts.charts import Bar
# 用链式调用方法
c = (
    Bar()
# 添加X轴标签
    .add_xaxis(
        [
            "名字很长的X轴标签1",
            "名字很长的X轴标签2",
            "名字很长的X轴标签3",
            "名字很长的X轴标签4",
            "名字很长的X轴标签5",
            "名字很长的X轴标签6",
        ]
    )
# 添加Y轴（第一个参数series_name，就是这个柱子的名字，第二个是Y轴数据）
    .add_yaxis("商家A", [10, 20, 30, 40, 50, 40])
    .add_yaxis("商家B", [20, 10, 40, 30, 40, 50])
# 设置全局变量
    .set_global_opts(
        xaxis_opts=opts.AxisOpts(axislabel_opts=opts.LabelOpts(rotate=-15)),
        title_opts=opts.TitleOpts(title="Bar-旋转X轴标签", subtitle="解决标签名字过长的问题"),
    )
    .render("bar_rotate_xaxis_label.html")
)

全局变量参数我说明一下，它有以下几个参数

title_opts:设置标题，它的给值要通过opts.TitleOpts()方法

legend_opts:设置图例

tooltip_opts:设置提示框

toolbox_opts:设置工具箱

brush_opts:设置区域选择组件

xaxis_opts:对x轴设置

yaxis_opts:对Y轴设置

这里面设置的参数非常多，做的图越精准，设置的参数越多，这里知识简单说明一下设置全局变量这个函数有哪些参数，而这些参数本身就是一个函数类型，它里面又有很多参数，具体的到后面作图再解释一下。

xaxis_opts=opts.AxisOpts(axislabel_opts=opts.LabelOpts(rotate=-15)

 解释一下这条代码：设置全局变量对图的X轴标签顺时针旋转15度，rotate为正值时表示逆时针

堆叠柱状图

堆叠柱状图就是将两个柱状堆叠在一起

from pyecharts import options as opts
from pyecharts.charts import Bar
from pyecharts.faker import Faker

c = (
    Bar()
    .add_xaxis(Faker.choose())
    .add_yaxis("商家A", Faker.values(), stack="stack1")
    .add_yaxis("商家B", Faker.values(), stack="stack1")
    .set_series_opts(label_opts=opts.LabelOpts(is_show=False))
    .set_global_opts(title_opts=opts.TitleOpts(title="Bar-堆叠数据（全部）"))
    .render("bar_stack0.html")
)
# Faker是一个生成随机值的辅助库
#调用choose()方法随机获取值作为X轴数据
#调用values()方法随机获取值作为Y轴数据
#用stack参数表示按stack1来堆叠

set_series_opts表示设置系列配置，它有一下参数

这里设置系列配置项表示不在柱状图中表示标签，也就是数字不出现柱状图中。

带有区域缩放的柱状图

 代码：

from pyecharts import options as opts
from pyecharts.charts import Bar
from pyecharts.faker import Faker

c = (
    Bar()
    .add_xaxis(Faker.days_attrs)
    .add_yaxis("商家A", Faker.days_values)
    .set_global_opts(
        title_opts=opts.TitleOpts(title="Bar-DataZoom（slider-水平）"),
        datazoom_opts=opts.DataZoomOpts(),
    )
    .render("bar_datazoom_slider.html")
)

 这里在全局变量设置中多了一个datazoom_opts参数，表示区域缩放配置项，看看图又有什么变化

这里补充一下如果这样设置  datazoom_opts=opts.DataZoomOpts(orient="vertical") ，那么区域选择表示是对Y轴缩放，vertical表示垂直嘛。

如果  datazoom_opts=opts.DataZoomOpts(type_="inside")，那么xy轴都不会出现滑块，我们就可以通过鼠标滚轮来滑动。

直方图 

直方图就简单了，只需要把柱状图之间的间隔调为0就成了直方图

代码

from pyecharts import options as opts
from pyecharts.charts import Bar
from pyecharts.faker import Faker


x = Faker.dogs + Faker.animal
xlen = len(x)
y = []
for idx, item in enumerate(x):
    if idx <= xlen / 2:
        y.append(
            opts.BarItem(
                name=item,
                value=(idx + 1) * 10,
                itemstyle_opts=opts.ItemStyleOpts(color="#749f83"),
            )
        )
    else:
        y.append(
            opts.BarItem(
                name=item,
                value=(xlen + 1 - idx) * 10,
                itemstyle_opts=opts.ItemStyleOpts(color="#d48265"),
            )
        )

c = (
    Bar()
    .add_xaxis(x)
    .add_yaxis("series0", y, category_gap=0, color=Faker.rand_color())
    .set_global_opts(title_opts=opts.TitleOpts(title="Bar-直方图（颜色区分）"))
    .render("bar_histogram_color.html")
)

通过  category_gap这个参数来调整两个柱状图之间的距离

from pyecharts import options as opts
from pyecharts.charts import Bar

c = (
    Bar()

    .add_yaxis("商家A", Faker.values())

    .render("bar_reversal_axis.html")

总结

 全局变量：

1都是参数名，他们的类型都是opts里面的函数，要给他们赋值必须这样调用

比如  title_opts= opts.TitleOpts(xxxx)，前面都要加上opts 

2这些函数都是全局配置项，现在说说这些函数都有哪些参数

Title_opts

 is_show: bool = True,  # 是否显示标题组件。

title: Optional[str] = None,# 主标题文本，支持使用 \n 换行。

 title_link: Optional[str] = None,# 主标题跳转 URL 链接
 title_target: Optional[str] = None, # 主标题跳转链接方式 # 默认值是: blank # 可选参数: 'self', 'blank',其中表示'self' 当前窗口打开; 'blank' 新窗口打开

subtitle: Optional[str] = None,  副标题文本，支持使用 \n 换行

subtitle_link: Optional[str] = None  副标题跳转 URL 链接

subtitle_target: Optional[str] = None  副标题跳转链接方式 # 默认值是: blank # 可选参数: 'self', 'blank' # 'self' 当前窗口打开; 'blank' 新窗口打开

pos_left: Optional[str] = None,    title 组件离容器左侧的距离。 # left 的值可以是像 20 这样的具体像素值，可以是像 '20%' 这样相对于容器高宽的百分比， # 也可以是 'left', 'center', 'right'。 # 如果 left 的值为'left', 'center', 'right'，组件会根据相应的位置自动对齐。

pos_right: Optional[str] = None   和pos_left类似

pos_bottom: Optional[str] = None,和pos_left类似

padding: Union[Sequence, Numeric] = 5,  设置标题内边距，不常用，这里不解释

item_gap: Numeric = 10,   主副标题之间的间距

text_align: str = "auto",   整体（包括 text 和 subtext）的水平对齐。 # 可选值：'auto'、'left'、'right'、'center'。

text_vertical_align: str = "auto",  # 整体（包括 text 和 subtext）的垂直对齐。 # 可选值：'auto'、'left'、'right'、'center'。

is_trigger_event: bool = False,  是否触发事件。

title_textstyle_opts: Union[TextStyleOpts, dict, None] = None,  主标题字体样式配置项，参考 `series_options.TextStyleOpts`

subtitle_textstyle_opts: Union[TextStyleOpts, dict, None] = None, 副标题字体样式配置项，参考 `series_options.TextStyleOpts`
