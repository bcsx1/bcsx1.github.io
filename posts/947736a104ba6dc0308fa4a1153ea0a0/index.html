Linux安装配置Redis

1、下载Redis

(1) 官网下载：Download | Redis

（2）百度网盘： 链接：https://pan.baidu.com/s/1Zu22HfBwDd4bEpEkPuvuSw 
提取码：1234

2、 Redis 是基于C语言编写的，因此首先需要安装Redis所需的gcc依赖：

 yum install -y gcc tcl                        安装gcc依赖

yum install -y gcc tcl

3、上传并解压

（1）把我们下载的Redis上传到Linux的任意目录，例如我把文件放在 /usr/local目录

（2）解压 tar -zxvf redis-6.2.7.tar.gz 

 

 4、编译安装

（1）进入Redis目录    cd redis-6.2.7/

（2）运行编译命令

        make && make install                make  编译                  make install  安装

cd redis-6.2.7/
make && make install

 

 （3）默认安装目录实在  /usr/local/bin 目录下

cd /usr/local/bin/
ll

 5、启动Redis的两种方式

（1）默认启动，可以在任意目录执行 redis-server 命令运行Redis

redis-server

 

 注意: 这种方式叫做前台启动，如果当前页面关掉，那么Redis也会被停止。

（2）要让Redis后台启动，则必须修改Redis配置文件 就在我们解压的目录里面：

        /usr/local/redis-6.2.7/redis.conf 

        注意：修改之前我们需要做好备份操作

         cp /usr/local/redis-6.2.7/redis.conf /usr/local/redis-6.2.7/redis.conf.back

cp /usr/local/redis-6.2.7/redis.conf /usr/local/redis-6.2.7/redis.conf.back

 

# 监听的地址，默认是127.0.0.1 会导致只能在本地访问，修改为0.0.0.0 则任意ip都可以访问，生成环境下不要设置为0.0.0.0

bind 0.0.0.0

# 守护进程，修改为yes后便可以后台运行

daemonize yes

#  密码，设置后访问Redis必须输入密码，其中这个默认时注释的

requirepass 123321

-----------------------------其他常用的配置

# 监听的端口
port 6379


# 工作目录，默认是当前目录，也就是运行redis-server时的命令，日志、持久化等文件会保存在这个目录
dir .


# 数据库数量，设置为1，代表只使用1个库，默认有16个库，编号0~15
databases 1


# 设置redis能够使用的最大内存
maxmemory 512mb


# 日志文件，默认为空，不记录日志，可以指定日志文件名
logfile "redis.log"

 vim redis.conf

启动服务

        a、进入安装页面  cd  /usr/local/redis-6.2.7

        b、redis-server redis.conf

cd /usr/local/redis-6.2.7
redis-server redis.conf

 已经后台运行了，查看有没有后台运行

ps -ef | grep redis

ps -ef | grep redis

 6、停止Redis

使用 kill -9 25294  其中 25294 是当前Redis的进程号，每次运行可能不一样。

kill -9 25294

 7、对于上面两种方式启动都很麻烦，可以选择使用开机自启

（1）新建一个系统服务文件

 cd /etc/systemd/system/redis.service 里面的东西可以复制粘贴 需要注意自己安装的目录

[Unit]
Description=redis-server
After=network.target

[Service]
Type=forking
ExecStart=/usr/local/bin/redis-server /usr/local/redis-6.2.7/redis.conf
PrivateTmp=true

[Install]
WantedBy=multi-user.target

 （2）重启服务 systemctl daemon-reload

systemctl daemon-reload

 （3）现在，我们可以用下面这组命令来操作redis了

        # 启动
        systemctl start redis


        # 停止
        systemctl stop redis


        # 重启
        systemctl restart redis


        # 查看状态
        systemctl status redis

（4） 执行下面的命令，可以让redis开机自启

systemctl enable redis

 8、安装完成
