pip 使用机巧

使用国内镜像源下载
pip install -i https://pypi.douban.com/sample/ 库名

windows下
https://www.lfd.uci.edu/~gohlke/pythonlibs/下寻找对应的.whl文件
pip install .whl文件

-i: 指定库的安装源
-U:升级 原来已经安装的包，不带U不会装新版本，带上U才会更新到最新版本

一、Linux版本：

linux的文件存放在：~/.pip/pip.conf
具体命令：
（1）cd
（2）mkdir .pip
（3）cd .pip
（4）vim pip.conf

pip.conf中的内容：

[global]
trusted-host=pypi.doubanio.com              #豆瓣源，可以换成其他的源
index-url=https://pypi.doubanio.com/simple/ #添加豆瓣源为可信主机，要不然可能报错

二、windows版本：

在用户文件夹下创建pip目录，并在pip目录下创建pip.ini文件（%HOME%\pip\pip.ini）
pip.ini中的内容：

[global]
trusted-host=pypi.doubanio.com              #豆瓣源，可以换成其他的源
index-url=https://pypi.doubanio.com/simple/ #添加豆瓣源为可信主机，要不然可能报错

三、国内可用的pip源:

https://pypi.doubanio.com/ 豆瓣
http://mirrors.aliyun.com/pypi/ 阿里
http://pypi.mirrors.ustc.edu.cn/ 中国科学技术大学
