Ps：图像处理器

Ps菜单：文件/脚本/图像处理器

Scripts/Image Processor

尽管在 Photoshop 中有多种方式来存储和导出图像，但图像处理器 Image Processor脚本有着更强大的功能。

提示：

可在 Adobe Bridge 中打开 Photoshop 的图像处理器。

Br菜单：工具/Photoshop/图像处理器

Photoshop/Image Processor

◆  ◆  ◆

图像处理器的主要用途

1、可将一个或多个图像文件（包括 Raw 格式文档）输出为 JPEG、PSD 或 TIFF 格式，或者同时输出其中两种或全部三种格式。

JPEG 文件常用于 Web 发布。

TIFF 文件常用于打印输出。

PSD 文件常用于备份或进一步编辑。

2、输出时可调整图像大小，使其转换为指定的像素大小。

3、输出时可指定要运行的 Ps 动作。

比如，对一组图像进行统一的调色处理，或者进行基于 AI 技术的磨皮祛痘处理。

4、在输出的图像中还可嵌入 ICC 配置文件和版权元数据。

命令选项说明

①选择要处理的图像

①Select the images to process

仅处理来自 Bridge 的文件

Process files from Bridge Only

使用 Adobe Bridge 打开“图像处理器”时才显示此选项。

仅处理在 Bridge 中选中的文件。

使用打开的图像

Use Open Image

处理当前在 Photoshop 中已打开的文档。

选择文件夹

Select Folder

指定要处理的图像所在的文件夹。

包含所有子文件夹

Include All sub-folders

若勾选，子文件夹里的所有文件也将被处理。

打开第一个要应用设置的图像

Open first image to apply settings

当选择多个 Raw 格式文件时，可打开第一个文件时自动启动“Camera Raw”。

说明：

1、在 Camera Raw 中设置完成后需点击“打开”按钮。

2、在 Camera Raw 中的设置仅记录在第一个 Raw 格式文件的 XMP 中，对其它选中的文档只会在转换时应用设置，不会记录在 XMP 中。

②选择位置以存储处理的图像

②Select location to save processed images

在相同位置存储

Save in Same Location

将处理后的图像保存在原文件夹下面的对应名称（JPEG、PSD、TIFF）的子文件夹中。

选择文件夹

Select Folder

将处理后的图像保存在指定文件夹下面的对应名称（JPEG、PSD、TIFF）的子文件夹。

保持文件夹结构

Keep folder structure

若不勾选（默认），则不同类型的输出文件保存独立的子文件夹中（JPEG、PSD、TIFF）。

若勾选，则所有的输出文件保存在指定的文件夹中。

③文件类型

③File Type

存储为 JPEG

Save as JPEG

存储为 JPEG 格式文件。

品质

Quality

设置 JPEG 文件的图像品质，范围从 1 ~ 12。

将配置文件转换为 sRGB 

Convert Profile to sRGB

将颜色配置文件转换为 sRGB。

如果要将配置文件与图像一起存储，请勾选下方的“包含 ICC 配置文件”选项。

存储为 PSD

Save as PSD

存储为 Photoshop 格式文件。

最大兼容

Maximize Compatibility

若勾选（默认），将在目标文件内存储分层图像的复合版本，以兼容无法读取分层图像的应用程序。

存储为 TIFF

Save as TIFF

存储为 TIFF 格式文件。

LZW 压缩

LZW Compression

使用 LZW 压缩算法存储 TIFF 文件。

调整大小以适合

Resize to Fit

每种要输出的文件类型都可以重新设置宽度 W 像素值和高度 H 像素值。

即便设置了调整大小，输出图像仍将保持其原始比例。

若新的宽度和高度的比例与原始比例不匹配时，将以长边为依据来等比例调整图像大小，从而保证输出图像不变形。

④首选项

④Preference

运行动作

Run Action

选择要运行哪个动作组中的哪个动作。

要运行的动作（组）必须在“动作”面板中。

版权信息

Copyright Info

包括在输出文件的 IPTC 版权元数据中输入的任何文本。

此处输入的文本将覆盖原始文件中的版权元数据。

包括 ICC 配置文件

Include ICC Profile

在存储的文件中嵌入颜色配置文件。

请参阅：

《数字媒体技术基础之：ICC 配置文件》

存储/载入

Save/Load

可以存储 Save“图像处理器”对话框中的当前设置（.xml 文件）。

下次要使用同样的设置时可载入 Load。
