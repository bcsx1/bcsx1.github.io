扩展运算符是JavaScript ES6版本中提供的新功能之一。

使用场景

迭代对象或数组

扩展运算符...用于展开可迭代对象或数组。例如，

const arrValue = ['My', 'name', 'is', 'Jack'];

console.log(arrValue);   // ["My", "name", "is", "Jack"]
console.log(...arrValue); // My name is Jack

在这个例子中，代码：

console.log(...arrValue)

等同于：

console.log('My', 'name', 'is', 'Jack');

使用扩展运算符复制数组

你也可以使用扩展语法...将项目复制到一个单独的数组中。例如，

const arr1 = ['one', 'two'];
const arr2 = [...arr1, 'three', 'four', 'five'];

console.log(arr2); 
//  输出:
//  ["one", "two", "three", "four", "five"]

使用扩展运算符克隆数组

在JavaScript中，对象是通过引用而不是值进行赋值的。例如，

let arr1 = [ 1, 2, 3];
let arr2 = arr1;

console.log(arr1); // [1, 2, 3]
console.log(arr2); // [1, 2, 3]

// 将一个项目添加到数组中
arr1.push(4);

console.log(arr1); // [1, 2, 3, 4]
console.log(arr2); // [1, 2, 3, 4]

在这里，变量arr1和arr2都指向同一个数组。因此，对一个变量的更改会影响到另一个变量。

然而，如果你想要复制数组而不是引用同一个数组，你可以使用扩展运算符。这样，一个数组的更改不会反映在另一个数组上。例如，

let arr1 = [ 1, 2, 3];

// 使用扩展语法进行复制
let arr2 = [...arr1];

console.log(arr1); // [1, 2, 3]
console.log(arr2); // [1, 2, 3]

// 将一个项目添加到数组中
arr1.push(4);

console.log(arr1); // [1, 2, 3, 4]
console.log(arr2); // [1, 2, 3]

与对象一起使用的扩展运算符

你也可以在对象字面量中使用扩展运算符。例如，

const obj1 = { x : 1, y : 2 };
const obj2 = { z : 3 };

// 将obj1和obj2的成员添加到obj3中
const obj3 = {...obj1, ...obj2};

console.log(obj3); // {x: 1, y: 2, z: 3}

在这里，使用扩展运算符将obj1和obj2的属性添加到obj3中。

Rest参数

当扩展运算符作为参数使用时，它被称为rest参数。

你也可以使用rest参数在函数调用中接受多个参数。例如，

let func = function(...args) {
    console.log(args);
}

func(3); // [3]
func(4, 5, 6); // [4, 5, 6]

在这里，

  • 当一个参数传递给func()函数时，rest参数只接受一个参数。
  • 当传递三个参数时，rest参数接受所有三个参数。

注意：使用rest参数将参数作为数组元素传递。

你也可以使用扩展运算符将多个参数传递给函数。例如，

function sum(x, y ,z) {
    console.log(x + y + z);
}

const num1 = [1, 3, 4, 5];

sum(...num1); // 8

如果使用扩展运算符传递多个参数，函数将接受所需的参数并忽略其余部分。

注意：扩展运算符是在ES6中引入的。某些浏览器可能不支持使用扩展语法。请访问JavaScript Spread Operator support了解更多信息。
