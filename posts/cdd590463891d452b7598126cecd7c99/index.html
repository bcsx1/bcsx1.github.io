1.数组和指针

关系及基本使用

数组是一个数据的集合，它在内存中的体现是一个连续的内存区域。如果当前声明的int类型数组，数组类型为3，这样我可以用一个int类型的指针，指向一个首地址，这样就可以通过指针操作数组了

#include<stdio.h>


void main(){
	int arr[] = { 100, 200, 300 };
	for (int i = 0; i < 3; i++)
	{
		printf("数组%d\n", arr[i]);
	}
	printf("======================\n");
	int* p = &arr;
	*p = 400;
	for (int i = 0; i < 3; i++)
	{
		printf("数组%d\n", arr[i]);
	}
	system("pause");
}

上述代码中，通过指针方式操控数组，把数组第一个元素，由100转换为400

通过指针进行运算

int* p = &arr;这是我们刚才设置的变量，当这个变量指向数组时，实际上指向的是数组的首地址，即数组的第一个元素，那我们能不能让这个指针指向第二个元素呢？

答案是肯定的，只需要*p的基础上面加1，就可以指向第二个元素

#include<stdio.h>


void main(){
	int arr[] = { 100, 200, 300 };
	for (int i = 0; i < 3; i++)
	{
		printf("数组%d\n", arr[i]);
	}
	printf("======================\n");
	int* p = &arr;
	*(p+1) = 400;
	for (int i = 0; i < 3; i++)
	{
		printf("数组%d\n", arr[i]);
	}
	system("pause");
}

这段代码与上面代码相比，几乎完全一样，只修改了*p被赋值的一条语句，运行结果，是数组第二个元素值被修改，*(p+1) = 400;被称为指针运算

大家想一下，我们+1的操作，它是针对当前类型进行+1，一个int类型，它是4个字节，那么+1，它会在内存中走4个字节的步长。那假设，如果是byte值类型，它会根据当前的值类型，走相应的步长，只+1个字节，这是根据指针类型所得到的。

可以这样取理解，假设有一个4G内存条，然后将内存条每一个区域进行编码，编码的内存就是当前的指针，假设一个int类型的数组，它的长度是3，在一块内存中连续性的区域，我们将数组的首地址取到，赋值给了一个指针，那么这个指针，就有了操作数组中每一个元素的能力。

2. 数组指针（也称行指针）

定义

int (*p)[n]

优先级高，说明p是一个指针，指向一个整型的一维数组，这个一维数组长度为n，也可以说是p的步长。也就是说执行p+1时，p要跨过n个整型数据的长度

int a[3][4];
int (*p)[4];//该语句是定义一个数组指针，指向含有四个元素的一维数组
p=a;//将该二维数组的首地址赋给p，也就是a[0]或者&a[0][0]
p++;//该语句执行过后，也就是p=p+1;跨过行a[0][]指向a[1][]

3.指针数组

定义

指针数组，顾名思义，这个数组中每一个元素，都是一个指针

假设有一块内存区，这块内存区有一给数组，这个数组每一个元素都存放在一个地址，这个地址分布指向的是一个变量，这样，我们就把这样一个数组称为指针数组。

代码实现：

#include<stdio.h>

void main(){
	int arr[] = { 100, 200, 300 };
	int *p[3];//指针数组
	for (int i = 0; i < 3; i++)
	{
		p[i] = &arr[i];
	}

	for (int i = 0; i < 3; i++)
	{
		printf("数组%d\n", *p[i]);
	}
	system("pause");
}

指针优先级

在指针数组里面，int *p[3];这种表达式是有优先级的

定义：

() > []> *

我们回过头来，看刚才的声明int *p[3];它是先结合后面的*p[3]，它代表一个数组，数组种每一个元素的类型，代表一个指针，所以它是一个指针的数组，每一个元素存放的是指针变量。再看前面的int，它是说明，这个指针数组种存放的每一个指针变量，所指向的值都是int类型。

我们不能给这个指针数组赋值int型变量，只能够赋值取地址，将
地址赋值给每一个元素。·取的时候，也要遵从存的规则，要加上*，表示指向地址所在的变量，这就是指针数组

4.指针数组和数组指针的区别

指针数组是代表一个数组，每一个元素存放的是一个地址；数组指针不是一个数组，它只是一个变量，这个变量，放在这一个指针，但是它的步长发生了变化，它是指向二维数组的首地址，如果对当前指针进行+1操作，它会指向第二行的首地址。

虽然在写法上上相差不大，只是加了一个()而已，但是在表示形式上，确实完全不一样。
