超多微软 API 服务价值等您 Pick

1、Azure API for FHIR

微软通过云计算平台 Azure 提供了世界上第一个基于云的 FHIR 服务。基于 Azure 的FHIR 服务器可作为开源产品使用，而为 FHIR 设计的 API 则可作为完全托管的“平台即服务”来提供。借助托管的 FHIR 服务，使用健康数据的任何人可在数分钟内在云中进行预配，同时在兼容的云环境中安全管理受保护健康数据并实现控制审核和跟踪。

2、Azure 文本翻译 API

Azure 认知服务中的文本翻译 API采用神经网络机器翻译，能够大幅度提升翻译质量和速度。开发人员通过将文本翻译 API 集成到应用程序中，可以轻松实现超过 60 种语言的文本互译，为企业开辟出更多的业务场景，例如网站本地化、电子商务、客户支持、消息处理、内部沟通等等，赋能企业开拓业务，提升效率。

3、Windows Vision Skills

对于开发者而言，Windows Vision Skills框架极大地降低了计算机视觉技术的应用门槛。应用开发人员可以使用WinRT APIs来轻松地在Windows应用(.NET、Win32和UWP)上集成已搭建好的视觉技术，例如物体检测、人体骨架检测等，而无需了解技术内部复杂的算法和设计，缩短开发周期，提升开发效率。另外，计算机视觉开发者可以借助Windows设备上的硬件加速框架来将其方案打包成一个视觉技术包，而无需担心底层的设计。
