一、线性表定义和特点

定义：由n(n>=0)个数据特性相同的元素构成的有限序列
特点：

1存在唯一的一个被称为“第一个”的数据元素
2存在唯一的一个被称为“最后一个”的数据元素
3除第一个外，结构中的每一个数据元素均只有一个前驱
4除最后一个外，结构中的每一个数据元素均只有一个后驱

存储方式：1、顺序存储：顺序表 2、链式存储：链表
注：线性表是一个相当灵活的数据结构，其长度可根据需要增长或缩短。

二、顺序存储

1.定义

定义：线性表的顺序表示指的是用一组地址连续的存储单元依次存储线性表的数据元素，这种表示也称作线性表的顺序存储结构或顺序映像。通常称这种存储结构的线性表为顺序表

2.完整代码及注解

class SQList:
    def __init__(self, maxsize):
        self.curLen = 0  # 顺序表当前长度
        self.maxSize = maxsize  # 顺序表最大长度
        self.listItem = [None] * self.maxSize  # 顺序表存储空间

    def clear(self):
        """将线性表制成空表"""
        self.curLen = 0

    def is_empty(self):
        """判断线性表是否为空表"""
        return self.curLen == 0

    def length(self):
        """返回线性表长度"""
        return self.curLen

    def get(self, i):
        """读取并返回线性表中位序为i的元素"""
        if i < 0 or i > self.curLen - 1:
            raise Exception("第i个元素不存在")
        return self.listItem[i]

    def insert(self, i, x):
        """在顺序表位序为i的位置插入x"""
        if self.curLen == self.maxSize:
            raise Exception("顺序表满")
        if i < 0 or i > self.curLen:
            raise Exception("插入位置非法")
        for j in range(self.curLen, i - 1, -1):
            self.listItem[j] = self.listItem[j - 1]
        self.listItem[i] = x
        self.curLen += 1

    def remove(self, i):
        """删除序数为i的元素"""
        if i < 0 or i > self.curLen - 1:
            raise Exception("删除位置非法")
        for j in range(i, self.curLen):
            self.listItem[j] = self.listItem[j + 1]
        self.curLen -= 1

    def index_of(self, x):
        """返回元素x首次出现的序号"""
        for i in range(self.curLen):
            if self.listItem[i] == x:
                return i

    def display(self):
        """输出线性表各个元素的数值"""
        for i in range(self.curLen):
            print(self.listItem[i], end=" ")

三、链式存储

1、定义

用一组任意的存储单元存储线性表的数据元素（这组存储单元可以是连续的，也可以是不连续的）。其中任意一个数据元素的存储位置称为一个结点，它包含两个域，分别为数据域（存储数据元素信息的域）和指针域（存储直接后继存储位置的域）。

2、分类

（根据链表节点所含指针个数，指针指向和指针连接方式）：
1、单链表：每个节点中只包含一个指针域
2、循环链表：尾结点（表中最后一个结点）指针域指向头节点，使整个链表形成一个环
3、双向链表：结点中含有两个指针域，一个指向直接后继，一个指向直接前驱

5、十字链表：

3、相关概念

1、头指针：指向链表中第一个结点的指针，通常为链表的表名。
2、头结点：指针域指向首元结点，数据域可为空，也可为相关附加信息（线性表长度等）
3、首元结点：链表中存储第一个数据元素a1的结点
4、尾结点：链表中存储最后一个数据元素an的结点

class Node(object):
    def __init__(self, data, next):
        self.data = data
        self.next = next
class LinkList():
    def __int__(self):
        self.head=Node(None,None)  #构造函数初始化头节点
    def creat(self,l,order):
        '''创建一个链表'''
        if order:
            self.create_tail(l)
        else:
            self.create_head(l)
    def create_tail(self,l):
        '''创建尾节点'''
        for item in l:
            self.insert(self.length(),item)
    def create_head(self,l):
        '''创建头节点'''
        for item in l:
            self.insert(0,item)
    def clear(self):
        '''将线性表置成空表'''
        self.head.data=None
        self.head.next=None
    def isEmpty(self):
        '''判断链表是否为空'''
        return self.head.next==None
    def length(self):
        '''返回链表长度'''
        p=self.head.next
        length=0
        while p is not None:
            p=p.next
            length+=1
        return length
    def get(self,i):
        '''读取并返回线性表中的第i个元素'''
        p=self.head.next
        j=0
        #从首节点开始向后查找，直到p指向第i个节点或者p为None
        while j<i and p is not None:
            p=p.next
            j+=1
        if j>i or p is None:   #i不合法时抛出异常
            raise Exception(f"第{i}个数据元素不存在")
        return p.data
    def insert(self,i,x):
        '''(带头结点)插入x作为第i个元素'''
        p=self.head
        j=-1
        while p is not None and j<i-1:
            p=p.next
            j+=1
        if j>i-1 or p is None:
            raise Exception("插入位置不合法")
        s=Node(x,p.next)
        p.next=s
    def remove(self,i):
        '''删除第i个元素'''
        p=self.head
        j=-1
        while p is not None and j<i-1:
            p=p.next
            j+=1
        if j>i-1 or p.next is None:
            raise Exception("删除位置不合法")
        p.next=p.next.next
    def indexOf(self,x):
        '''返回x首次出现的位序号'''
        p=self.head.next
        j=0
        while p is not None and p.data!=x:
            p=p.next
            j+=1
        if p is not None:
            return j
        else:
            return -1
    def display(self):
        '''输出线性表中各个元素的值'''
        p=self.head.next
        while p is not None:
            print(p.data,end=' ')
            p=p.next

四、顺序表与链表的比较

顺序表链表
