1.5 markdown设置页内跳转

需求分析：有时候写了比较长的文档，需要引用前面写的内容进行说明，这时候如果能够点击一个本地跳转链接，直接跳转到前面指定位置就好的。

首先，设置跳转位置标签，格式如下，其中"label1"为标签，可以任意取名，后面通过该标签进行引用；"实验结果表1"为需要引用的内容。

<a id="label1">实验结果表1</a>

然后进行引用：其中"label1"为前面设置的标签，[]中"实验结果表1"为本地跳转链接的名字即点击这几个字会发生跳转。

二 插入图片

        注：我的markdown文档和图片在同一个目录下

2.1 html风格代码：

（1）插入图片

```

<img src="图片路径">

# 注： 图片路径可以是相对路径、绝对路径和网络路径

```

<img src="picture1.jpeg">

（2）设置图片比例/高度/宽度

```

# 修改width：

<img src="picture1.jpeg" width="200px" />

# 直接设置宽度和高度

<img src="http:..." width = "100" height = "100" />

# 或者修改height：

<img src="picture1.jpeg" style="height:200px" />

# 直接设置宽度和高度比例

<img src="http:..." width = 30% height = 30% />

# 也可以直接按比例缩放：

<img src="picture1.jpeg" style="zoom:70%" />

```

<img src="picture1.jpeg" style="zoom:20%" />

（3）设置图片对齐

```

<div align=center><img src="路径" /></div>

# 注：外层加个div块级标签可以解决图片与紧接着的文字可能在一行的情况

# 注：right left center

```

<div align=center><img src="picture1.jpeg" width="400px" /></div>

2.2 另一种风格代码

```

![图片说明文字](图片地址)

# 注：图片路径可以是相对路径、绝对路径和网络路径

```

![图片说明文字](picture1.jpeg)

三、vscode插件

3.1 Markdown Preview Enhanced

        安装后，在markdown文档界面点击鼠标右键会多出来一个"Markdown Preview Enhanced: Open Preview to the Side"或"打开侧边预览"

        效果见前面图片。

        注：有时候右键会没有"打开侧边预览"的快捷方式，重新安装一次就好，唉，好像不是很稳定

3.2 Markdown All in One

        该插件用于在编写文档时自动帮我们填充内容，比如支持使用快捷键调整格式，在使用列表时会自动帮我们补全下一项的值

3.3 Markdown TOC

        该插件用于生成目录，比如给标题自动添加序号

3.4 Markdown PDF

        用于转换为pdf，markdown文档界面点击鼠标右键会多出来一个“Markdown PDF: export(pdf)”的选项
