  • 定点数补码加法的计算准则是：
    [ x ] 补 + [ y ] 补 = [ x + y ] 补 m o d 2 n + 1 [x]_补+[y]_补=[x+y]_补mod2^{n+1} [x]补​+[y]补​=[x+y]补​mod2n+1,n为字长。

  • 定点数减法计算准则：
    [ x − y ] 补 = [ x ] 补 + [ − y ] 补 [x-y]_补=[x]_补+[-y]_补 [x−y]补​=[x]补​+[−y]补​
    [ − y ] 补 [-y]_补 [−y]补​可直接根据定义计算，也可以根据y的补码保留最右边的一个1和之后的数，1左边的数全部取反。

  • 数据溢出检测：
    运算结果超出了数据的表示范围时就表示溢出了。
    溢出只发生在同符号数相加时。
    检测溢出：
    1、检测结果和操作数的符号位，当运算结果和操作数的符号位不同时就表示溢出了。

    2、对最高数据位进位和符号位进位进行检测。
    以C0表示符号位进位，C1表示最高数据位进位，C0与C1异或结果为1时就发生了溢出。
    3、用变形补码：即用两位符号位，初始时两位符号位相同，当两位符号位的异或结果为1时就发生了溢出。
    还有软件检测测溢出。
    无符号整数检测溢出：无符号数加法可以用ALU的进位位检测，无符号数减法根据ALU进位取反检测。

原码乘法
  • 逻辑左移：各位向左移，空位补零；
  • 算术左移：与逻辑左移相同，左移一位扩大2倍；
  • 逻辑右移：向右移动，高位补零；
  • 算术右移：高位用符号位填补。右移一位缩小2倍。
    二进制原码乘法：（与慕课的不太一样）
    • 采用累加的方法；
    • 首先设置部分积为0,
    • 根据乘数的最后一位是0还是1，最后一位是0，被乘数左移一位，是1乘积加上被乘数写进乘积寄存器，被乘数左移一位，乘数右移一位。
    • 符号位由乘数和被乘数的符号位异或决定。
补码乘法

补码的乘法与原码的乘法不同，补码的乘法符号位要参与运算。
运算规则：
设被乘数为 [ X ] 补 = X 0 X 1 X 2 X 3 . . . X n [X]_补=X_0X_1X_2X_3...X_n [X]补​=X0​X1​X2​X3​...Xn​,乘数为： [ Y ] 补 = Y 0 Y 1 Y 2 Y 3 . . . Y n [Y]_补=Y_0Y_1Y_2Y_3...Y_n [Y]补​=Y0​Y1​Y2​Y3​...Yn​
则 [ X Y ] 补 = [ X ] 补 ∗ ( 0. Y 1 Y 2 . . . . Y n ) − Y 0 ∗ [ X ] 补 [XY]_补=[X]_补*(0.Y_1Y_2....Y_n)-Y_0*[X]_补 [XY]补​=[X]补​∗(0.Y1​Y2​....Yn​)−Y0​∗[X]补​
简化得： [ X Y ] 补 = [ X ] 补 ∗ 求 和 ( Y i + 1 − Y i ) ∗ 2 − i [XY]_补=[X]_补*求和(Y_{i+1}-Y_i)*2^{-i} [XY]补​=[X]补​∗求和(Yi+1​−Yi​)∗2−i
具体规则为：
+ 如果 Y n + 1 = Y n Y_{n+1}=Y_n Yn+1​=Yn​部分积加0，算术右移一位；
+ 如果 Y n + 1 Y n = 10 Y_{n+1}Y_n=10 Yn+1​Yn​=10部分积加 [ X ] 补 [X]_补 [X]补​，算术右移一位；
+ 如果 Y n + 1 Y n = 01 Y_{n+1}Y_n=01 Yn+1​Yn​=01部分积加 [ − X ] 补 [-X]_补 [−X]补​，算术右移一位。
+ 重复进行n+1次，最后一步不右移。

由于Y中出现了n+1，所以需要在乘数寄存器中补一位，该位为0，不改变Y的值。

预算时，部分积和乘数寄存器都右移，部分积右移后的超出位保存在乘数的高位。

定点数除法

  • 定点数除法手工计算
    看除数和被除数是否够减，构建商上1，不够减商上0，除数右移一位再比较余数和除数，重复多次操作。

  • 恢复余数计算

利用减法，判断余数的正负，如果余数为负，商上0，恢复余数并左移一位，如果余数为正，商上1，余数左移一位，然后余数减除数再进行同样的操作，直到结果符合要求的位数时结束。

这种计算的弊病就是不知道要计算几次才能结束。

  • 不恢复余数计算
    恢复余数就是余数加上除数再左移，实际上，余数先左移一位再加除数和恢复余数再左移的结果相同，因此不恢复余数就根据这个方法进行计算。

    • 首先，开始时余数就是被除数，余数减除数，判断余数的正负，如果为负商上0，余数左移一位加上除数；如果余数为正，商上1，余数左移一位。之后再减除数进行重复操作，过程中商也在左移。

浮点数加减运算

  • 规格化
    浮点数的小数点是不定的，运算时就存在麻烦，因此需要规格化。
    一般浮点数的规格化方法为：
    00.1…或者11.0…,双符号位

    • 规格化方法
      00.0…和11.1…将尾数左移若干位，同时指数减相应的数。
      01.x…和10.x…这种表示位数相加结果大于1，需要一次右移操作，同时指数加1.

  • 运算步骤

    • 对阶：求出两个数字的阶差，将指数小的变成指数大的，同时位数需要右移阶差位。
    • 尾数加或减：用对阶后的位数相加或相减。
    • 结果规格化：尾数相加后根据规格化的方法进行规格化。
    • 舍入：右移规格化时，末位的1可能丢失，为提高精度可以采用舍入的方法，具体为：0舍1入，就是右移出的是1就将最低位置为1；恒置1：只要1被移掉，最低位就置为1。
    • 溢出处理：阶码上溢，阶码的符号位为01；阶码下溢，阶码的符号位是10
