安装scikit-learn，win7 64位 （装了一下午，终于搞定了）解决ImportError: DLL load failed

         安装sklearn之前，我们需要先安装numpy，scipy函数库。

         说说我的经历，我是从网上下的numpy-MKL-1.8.0.win-amd64-py2.7.exe,scipy_0.14.0.win_amd64_py2.7.exe,scikit-learn-0.14.1.win-amd64-py2.7.exe,完全装好之后，但是运行不了。

         当要运行如下语句时：

          from sklearn import metrics，会报如下错误。

      安装sklearn报错：ImportError: DLL load failed
              因此，非常不建议，下载exe文件来装。我就改用pip来装，最后装成功。分成以下几个步骤：
          1、先安装pip，可见以下博客：http://blog.csdn.net/licong_carp/article/details/43487101。
//-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

          最近在研究scikit-learn的开源包，总是遇到scikit-learn包找不到的问题，按照以下步骤问题就解决了，具体步骤如下：

         1、  安装python，一般为2.7，网址：https://www.python.org/downloads/

          2、  Python安装distribute：打开windows的cmd，切换到distribute-0.6.36目录. 输入：python setup.py install。网                     址：https://pypi.python.org/pypi/distribute

         3、  安装pip：下载get-pip.py.放到python安装的文件夹里面。运行：python get-pip.py 。网址：https://bootstrap.pypa.io/get-pip.py，将内容复制保存为get-pip.py即可

         4、  当利用sklearn进行机器学习的研究的时候，当遇到当我们遇到类似“ImportError：cannotimport name **”这样的问题时，我们运行pip install -U scikit-learn，这样问题就解决了。

         5、  我们可以将pip所在的路径添加到系统环境中，这样每次打开cmd时，就不要再改变路径，直接敲pip即可。

Reference：http://blog.csdn.net/weiyidemaomao/article/details/8779728

//------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

        总结：主要就两点(只要做第3，第5点即可)，

                   1、一个下载get-pip.py,放到python的安装文件夹中，然后执行python get-pip.py。就可以安装成pip,在python的安装目录里面的script文件夹就会看到pip

                   2、系统变量设置。我python的安装路径如图，只需把D:\lujiandong\Python27 和D:\lujiandong\Python27\Scripts加入到环境变量中即可。
                

         做完以上两步，就把pip装好了，接下来要用pip来装，numpy，scipy，scikit learn。这几个包不要到网上去搜exe文件，而是去次网址去搜， http://www.lfd.uci.edu/~gohlke/pythonlibs/#scikit-learn。该网址上的文件是whl文件，该安装文件可用pip安装。
因为我装的python的版本是2.7，所有numpy选择，numpy-1.9.2+mkl-cp27-none-win_amd64.whl(64位，cp27猜测是对应python2.7)。numpy的版本和python的版本一定要对应，不然安装不了。

 

同理，选择scikit_learn-0.16.1-cp27-none-win_amd64.whl.

               scipy-0.15.1-cp27-none-win_amd64.whl


好了现在下午这几个文件，该如何用pip安装呢。

第一步：先在window的cmd上，执行pip install wheel ，安装wheel.

接着，使用cd命令，引导当前目录到下载的whl文件的目录，然后pip install 包名字.whl。即可安装。

比如安装numpy,就是 pip install numpy-1.9.2+mkl-cp27-none-win_amd64.whl

这样就基本安装完毕，亲测可行
