python面向对象入门

什么是类

类是描述一类有相同行为和属性的对象的集合，类定义了每个对象具有的共同属性和方法，对象是类的实例化。在python中使用关键字Class可以定义一个类 :

class Cat():
	year = 4
	weight = 10
	def eat(self):
		print("I'm eating")
	def run(self):
		print("I'm running")

通过实例化生成一个猫对象，具有猫类中共有的属性和方法，猫的属性和方法可以通过python引用的标准语法访问 : obj.name

cat = Cat()
#访问属性
print(cat.year)
#使用方法
cat.eat()

构造方法

python中有一类特殊的方法，开头和结尾都有双下划线，比如要介绍的__init__()方法就是这类方法的一种。当类实例化时会自动调用__init__()方法，对实例初始化。

class Cat():
	def __init__(self,year,weight):
		self.year = year
		self.weight = weight
	def eat(self):
		print("I'm eating")

这样我们就可以自己定义一个有自己间年龄和体重的猫。

cat = Cat(4,10)
cat.year
cat.eat()

输出4和I'm eating，
关键字self指向实例化的内存地址，即指向实例本身，则self.name可以理解为通过对象名访问方法和属性，与obj.name起到的作用一样。上述代码中的self.weight = weight可以理解为创建一个实例的weight属性并把它赋值为传进来的参数。

继承

为了简化代码，避免代码冗长，同时避免重复造轮子，我们可以定义一个抽象类，再让子类继承父类的属性和方法。

class Animal():
	def __init__(self,year,weight):
		self.weight = weight
		self.year = year
	def eat(self):
		print("I'm eating")
	def run(self):
		print("I'm running")

有了这个抽象类之后，我们可以让猫类或者狗类继承这个类，因为他们有着相同的属性和方法。

class Cat(Animal):
	def __init__(self,year,weight):
		super().__init__(year,weight)

代码可以等价写为:

class Cat():
	def __init__(self,year,weight):
		self.year = year
		self.weight = weight
	def eat(self):
		print("I'm eating")
	def run(self):
		print("I'm running")

继承的同时，也可以让子类定义自己的属性和方法 :

class Cat(Animal):
	def __init__(self,year,weight,color):
		super().__init__(year,weight)
		self.color = color
	def call(self):
		print("喵喵喵")

我们还可以在继承
注1:
父类构造函数有三种等价写法，它们除了写法不同，产生的效果没有任何差别。super().__init__(*kargs)=super(Animal,self).__init__(*kargs)=Animal.__init__(*kargs)
注2:
如果没有使用super().__init__(*kargs)初始化父类属性，子类将会失去父类的初始化属性，但仍能继承父类的方法; super()将会调用父类，再调用父类的__init__()方法，self已经在super中传递，在__init__()会隐式传递。

方法重写

当继承自父类的属性和方法并不能满足要求的时候，就需要用到方法重写。这里我们重写一只不爱跑步的猫，当实列化一只不爱跑步的猫时，调用run()函数会打印I don't like running。这是因为当实例对象寻找属性或者方法时，会优先从当前类中查找，如果没有找到再去父类查找。

class Cat(Animal):
	def __init__(self,year,weight):
		super().__init__(year,weight)
	def run(self):
		print("I don't like runnig")

类属性、类方法和静态方法
  1. 类属性是所有属于这个类的对象所共有的，在内存中只有一份副本。类属性可以通过实例对象和类来访问:
class Cat():
	color = "blue"
	def __init__(self,year,weight):
		self.year = year
		self.weight = weight

即可以通过class.name的方式访问，也可以实例化对象后通过对象访问

print(Cat.color)
cat = Cat(10,4)
print(cat.color)
#打印结果都为blue

但是注意是不能通过实例对象来修改类属性的，因为修改通过实例修改类属性影响的只是实例的属性，类属性并没有发生改变:

cat.color = 'yellow'
print(cat.color)
print(Cat.color)

控制台输出结果发现类属性没有变，说明通过实例修改属性只能影响实例。

  1. 类方法
    类方法是类所拥有的方法，类方法用装饰符@classmethod标识，以此区别于其它方法。对于类方法，第一个参数必需为类本身，习惯上用cls为参数，类方法能通过实例方法和类方法访问。
class Cat():
	name = " "
	def __init__(self,name):
		self.name = name
	@classmethod
	def call(cls):
		print(f"{cls.name}喵喵喵")

Cat.call()
cat = Cat("Tom")

两者都输出 “喵喵喵”，但是可以注意到我们实例化了一只叫Tom的猫，代码并没有打印出我们想要的结果 : Tom喵喵喵。这再一次说明了类属性并不会因为实例对象而改变。
从传参的形式也可以看出来，cls传递的是类对象，通过cls引用的必定是类属性或者类方法；实例对象传递的是self，为实例本身，可以引用实例属性或者类属性。

  1. 静态方法
    静态方法不需要传递参数，使用标识符@staticmethod可以标识一个静态方法。
class Cat():
	@staticmethod
	def call():
		print("喵喵喵")
