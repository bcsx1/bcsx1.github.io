01. i1Profiler概述

X-Rite 的 i1 软件解决方案非常适合寻求最大程度控制色彩的成像专业人士。为所有设备（包括扫描仪、显示器、投影仪、RGB 和 CMYK+ 打印机）创建自定义配置文件可确保一致、可预测和可重复的结果，同时节省宝贵的时间和金钱。i1Profiler 软件具有基本模式和高级模式，可实现专业校准和配置设备，以及显示器和打印机的质量保证控制。

i1Profiler 软件提供校准自动化与全方位配置选项的完美平衡，以实现显示器和打印机的创造性色彩控制。

02. i1Profiler特征

打印机

  • 支持最准确的打印机配置文件的高补丁计数图表 - 非常适合美术打印和打样
  • 通过使用线性化功能，随着时间和运行之间保持颜色正确
  • 通过闪光灯和环境光测量捕捉真实色彩，以实现完美的色彩匹配
  • 利用荧光增白剂补偿 (OBC) 功能

监视器

  • 提供无限的伽马、白点和亮度设置
  • 使用参考配置文件进行工作组匹配
  • 优化灰平衡以获得更中性和更清晰的灰色
  • 利用色度适应公式实现不同显示器更紧密的视觉匹配

独特的能力

  • 创建最高质量的专业色彩配置文件
  • 补偿纸张中的荧光增白剂
  • 使用高级控件控制黑色分离
  • 使用白点编辑微调配置文件

04. i1Profiler官方安装说明

在 Windows 上安装 i1Profiler 软件

在安装 i1Profiler 软件套件之前，请查看所需的系统环境。要从 DVD 或下载安装 i1Profiler 软件套件，请按照以下步骤操作。

注意：Windows XP、Vista 和 7 上的软件安装过程几乎相同。但是，在 Windows Vista 或 7 系统上安装之前，我们建议您执行以下操作：

  • 在系统的控制面板“安全”或“Windows Defender”中禁用 Windows Defender。
  • 在 Vista 的控制面板“用户帐户”中分别禁用用户帐户控制 (UAC)…
  • 在 Windows 7 控制面板“用户帐户”中将用户帐户控制 (UAC) 设置为最低级别。

从 DVD 安装

  • 断开测量设备和加密狗。
  • 将 i1Profiler 软件 DVD 插入 DVD 驱动器，安装程序将自动启动。
  • 按照屏幕上的说明进行操作。
  • 选择安装程序的语言（注意：i1Profiler 软件套件将以 Windows 操作系统设置的语言安装。如果 i1Profiler 不支持该语言，软件将以英语安装）
  • 阅读并接受软件许可协议。
  • 仔细阅读发行说明（系统要求、已知问题）。
  • 选择软件安装的目的地
  • 选择您要安装的附加软件（ColorChecker Passport、PANTONE Color Manager）。
  • 选择程序应出现的开始菜单文件夹。
  • 开始安装（根据选择的要安装的软件组件，ColorChecker Passport 和 PANTONE Color Manager 的子安装程序将自动进行）
  • 安装完成后，重新启动计算机
  • 连接您的测量仪器和/或加密狗

从下载安装

  • 可在此处下载最新的 i1Profiler 软件套件
  • 断开测量设备和加密狗。
  • 双击下载的 i1 Applications Setup.exe 文件。
  • 按照屏幕上的说明进行操作。
  • 选择安装程序的语言（注意：i1Profiler 软件套件将以 Windows 操作系统设置的语言安装。如果 i1Profiler 不支持该语言，软件将以英语安装）
  • 阅读并接受软件许可协议。
  • 仔细阅读发行说明（系统要求、已知问题）。
  • 选择软件安装的目的地
  • 选择您要安装的附加软件（ColorChecker Passport、PANTONE Color Manager）。
  • 选择程序应出现的开始菜单文件夹。
  • 开始安装（根据选择的要安装的软件组件，ColorChecker Passport 和 PANTONE Color Manager 的子安装程序将自动进行）
  • 安装完成后，重新启动计算机
  • 连接您的测量仪器和/或加密狗

已知问题
在 macOS 12 (Monterey) 系统上，以 TIFF 格式保存的测试图表，其上面的文本遭损坏。以 PDF 格式保存测试图表，以获得正确的文本渲染。
Apple 新款 M1 Mac 最多支持两台显示器。
适用于光源箱手动透射扫描的测量算法没有错误检查选项。进入下一个工作流程步骤之前，请在屏幕上视觉验证所有测量数据。
环境光监测时，i1Profiler 托盘应用程序（仅限 Mac）有内存泄漏。定期重启托盘应用程序以释放系统内存。
在色域视图窗口内滚动时，有刷新问题。如果屏幕无法正确刷新，请单击应用程序窗口中任意位置，色域视图将更新。
管理和优化有大量色块集（>3000）的配置文件需要大量 RAM。如果色彩管理失败，请减少测试图表中的色块数量。
多台显示器处于镜像模式时，显示器校准无效（该问题仅影响 Windows® 安装程序）。
i1Profiler 不支持校准通过 AirPlay 连接的显示器。
如果在创建显示器配置文件之后，加载过程中遇到问题，或系统无法加载显示器 LUT，请在测量页面关闭自动显示器控制 (ADC) 再重试。
i1Profiler使用OpenGL显示配置文件的3D色域。如果在色域预览中遇到问题，请确保视频卡驱动程序最新。
i1Profiler 可以导入其他应用程序中的 CGATS 测量文件。然而，色彩引擎已经优化，可以配合 i1Profiler 内生成的色块集使用。在 i1Profiler 工作流程内完整生成的配置文件质量将优于旧色表与测量中生成的配置文件质量。至于 CMYK+N 配置文件，一些旧版色块集甚至可能无法成功创建配置文件。强烈建议用户在 i1Profiler 内为 CMYK+N 色彩管理新建图表。
如果在创建显示器配置文件时使用环境光智能控制，预计在显示器 QA 中会获得较高 ΔE 值。此功能根据测量的环境条件优化配置文件的视觉外观，而非最小 ΔE 值。
i1ProfilerTray 应用程序在启动时查找连接的显示器。如果断开或连接显示器，i1ProfilerTray 将在重新启动、用户退出或系统重启之后才能看到更改。在 Windows 环境下，i1ProfilerTray 可从程序\启动菜单重新启动，而在 Mac 环境下，衬盘位于与 i1Profiler 应用程序相同的文件夹。
如果计算机进入睡眠或休眠模式，此应用程序可能会丢失与测量设备的连接。如果出现上述情况，请断开并重新连接设备，以便恢复连接。
安装使用上一版本爱色丽设备服务的旧版应用程序，可能导致 i1Profiler 无法连接设备或无法启动。如果出现这种情况，请重新安装 i1Profiler 以便恢复最新设备服务。
如果在连接到测量设备时遇到任何问题，请断开并重新连接设备，以便恢复连接。
连接 USB 电缆时，请确保 i1iSis 电源按钮已开启。如果在电源按钮关闭后连接，然后再开启电源，将不会显示 i1iSis。
i1iO 设备不支持单行色表。
