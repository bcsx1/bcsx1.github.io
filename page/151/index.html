python缩进规则

在 Python 中，缩进是语法的一部分，它用于标识代码块的开始和结束。Python 采用的是强制缩进规则，即每个代码块必须有相同的缩进量。 在 Python 中，常用的缩进量是4个空格。一般来说，每个缩进级别对应4个空格，而不是制表符（tab）或混合空格和制表符。这是因为不同的文本编辑器和操作系统对制表符的显示方式不同，可能会造成代码在不同环境下的显示不一致。 下面是一个简单的例子，演示了缩进的用法： if x > 0: print("x is positive") else: print("x is non-positive") 在上面的代码中，if和else后面的代码块都有相同的缩进，它们属于同一个代码块。如果你使用了错误的缩进，Python 就会抛出 IndentationError 异常。例如，下面的代码就会产生一个缩进错误： if x > 0: print("x is positive") else: # 错误的缩进 print("x is non-positive") 注意到 else 的缩进是 2 个空格而不是 4 个空格，所以这个代码就会抛出缩进错误异常。因此，在 Python 中编写代码时，要特别注意缩进规则，否则就会引发错误。
